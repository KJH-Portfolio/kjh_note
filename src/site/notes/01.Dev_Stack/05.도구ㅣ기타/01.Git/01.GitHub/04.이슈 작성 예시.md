### ✨ 기능 요약
카카오 맵 API를 연동하여 상점 위치를 지도 위에 마커로 표시합니다.

### 📖 상세 설명
- 사용자가 근처의 제로웨이스트 샵을 쉽게 찾기 위해 지도가 필요합니다.
- 주소 텍스트만 보여주는 것보다 시각적인 정보를 제공하여 편의성을 개선합니다.

### 🛠 구현 아이디어 (선택)
- Kakao Map API 활용
- DB의 `store_address` 데이터를 위경도 값으로 변환하여 마커 생성
- 마커 클릭 시 해당 상점의 간단한 정보(이름, 평점) 팝업 노출

### 🖥 적용 범위
- [x] Backend (Spring / API)
- [x] Frontend (JSP / JS)
- [ ] Database (Mybatis / Table)
- [x] External API

### 📎 참고 자료 (선택)
- 카카오 맵 API 가이드 문서 (링크)
