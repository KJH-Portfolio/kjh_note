# 📘 JDBC SELECT 실행부 정리

## Summary 📝

> [!summary] Summary 📝
> 
> - `executeQuery` 메서드로 SELECT 실행
> - 결과는 `ResultSet` 객체로 반환
> - `ResultSet.next()`로 행 이동 후 `getXxx("컬럼명")`으로 데이터 추출

---

## 목적 🎯

- **데이터 조회 목적:** DB 테이블에서 원하는 데이터를 검색하여 애플리케이션에서 활용
- **VO 매핑 목적:** 조회된 데이터를 VO 객체에 담아 컬렉션으로 관리

---

## 핵심 개념 💡

### ✅ SELECT 실행부 주요 메서드

| 메서드                       | 역할            | 반환값/동작                 |
| ------------------------- | ------------- | ---------------------- |
| `executeQuery(sql)`       | SELECT 구문 실행  | `ResultSet` 반환         |
| `ResultSet.next()`        | 커서를 다음 행으로 이동 | 행 존재 시 true, 없으면 false |
| `ResultSet.getXxx("컬럼명")` | 현재 행의 컬럼 값 추출 | 컬럼 타입에 맞는 값 반환         |

---

## 특징 ✨

- **반환 타입:** `executeQuery`는 `ResultSet` 반환
- **커서 기반 순회:** `ResultSet`은 처음에 첫 행 이전에 위치
	- `next()` 호출 시 다음 행으로 이동하고 이동했으면 true 반환
- **컬럼 접근:** 컬럼명 기반 접근 권장 (`getString("USER_ID")`)
- **VO 매핑:** 반복문 안에서 VO 객체 생성 후 리스트에 추가

---

## 팁과 주의점 ⚠️

- **컬럼명 정확성:** 반드시 실제 테이블 컬럼명과 일치해야 함
- **자료형 일치:** `getInt`, `getString`, `getDate` 등 컬럼 타입에 맞는 메서드 사용
- **널 처리:** `getString` 등은 널 반환 가능 → NPE 주의
- **자원 반납:** `ResultSet`은 사용 후 반드시 `close`

---

## 예시 📌

### ✅ SELECT

```java
// 준비
Connection conn = null;
PreparedStatement pstmt = null;
ResultSet rset = null;
Scanner sc = new Scanner(System.in);
// 책 목록 담을 리스트 준비
ArrayList<Book> list = new ArrayList<>();

// 사용자 입력 받기
System.out.print("찾고싶은 책 검색 : ");
String keyword = sc.nextLine();

// SQL 구문
String sql = "SELECT * FROM MEMBER WHERE BOOK_TITLE LIKE ?";

try {
    // Connection 객체 생성
    conn = DriverManager.getConnection(
            "jdbc:oracle:thin:@localhost:1521:xe",
            "JDBC", "JDBC");

    // PreparedStatement 객체 생성
    pstmt = conn.prepareStatement(sql);

    // 바인드 변수 채우기
    pstmt.setString(1, "%" + keyword + "%");

    // 구문 실행
    rset = pstmt.executeQuery();

    // 반환받은 ResultSet을 ArrayList에 담기
    while (rset.next()) {
        Book b = new Book(
            rset.getString("BOOK_TITLE"),
            rset.getString("AUTHOR"),
            rset.getInt("BOOK_PRICE")
        );
        list.add(b);
    }

} catch (SQLException e) {
    e.printStackTrace();
} finally {
    // 자원 반납
    try {
        if (rset != null) rset.close();
        if (pstmt != null) pstmt.close();
        if (conn != null) conn.close();
    } catch (SQLException e) {
        e.printStackTrace();
    }
}

```
