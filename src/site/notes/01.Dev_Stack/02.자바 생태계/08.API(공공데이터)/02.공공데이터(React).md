## React 기반 클라이언트 사이드 요청 (Axios)
- **Axios 라이브러리 기반 비동기 통신**
    - 브라우저 환경에서 외부 API 서버로 직접 HTTP GET/POST 요청 수행.
    - `useEffect` 훅을 활용한 컴포넌트 마운트 시점의 데이터 페칭(Fetching).
- **실시간 데이터 바인딩**
    - 서버(Spring Boot)를 거치지 않고 직접 응답을 받아 React 상태(State)에 즉시 반영.
    - 사용자 경험(UX) 향상을 위한 로딩 처리 및 에러 핸들링 용이.
- **주의사항**
    - API 서비스 키 노출 위험에 따른 `.env` 환경 변수 관리 필수.
    - 브라우저 정책에 따른 CORS(Cross-Origin Resource Sharing) 설정 확인 필요.
```javascript
import axios from 'axios';
import { useState, useEffect } from 'react';

const WeatherComponent = () => {
  const [forecasts, setForecasts] = useState([]);
  
  const fetchWeather = async () => {
    try {
      // // Axios를 활용한 공공데이터 API 직접 호출
      const response = await axios.get('https://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getVilageFcst', {
        params: {
          serviceKey: 'API_KEY_HERE', // // 환경 변수 권장
          pageNo: '1',
          numOfRows: '1000',
          dataType: 'JSON',
          base_date: '20240101', // // 날짜 로직 필요
          base_time: '0500',
          nx: '55',
          ny: '127'
        }
      });
      // // 응답 구조에 따른 데이터 추출 및 상태 업데이트
      const items = response.data.response.body.items.item;
      setForecasts(items);
    } catch (error) {
      // // 요청 실패 시 에러 로깅
      console.error("데이터 수신 실패:", error);
    }
  };

  useEffect(() => {
    fetchWeather();
  }, []);
};
```
