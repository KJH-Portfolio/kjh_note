# JSTL (Standard Tag Library)

> [!summary] 요약
> JSTL의 3대 핵심 라이브러리인 **Core(제어), Formatting(포맷), Functions(문자열 처리)**의 **태그 문법과 속성**을 총정리한 노트입니다.

## 1\. Core Library (`prefix="c"`)

변수 저장/삭제,제어문,토큰분리,url이동..등을 담당하는 핵심 라이브러리입니다.

### 1.1. 태그 및 속성 구조

| 기능         | 태그 및 주요 속성 (Syntax)                                                                   |
| :--------- | :------------------------------------------------------------------------------------ |
| **변수 저장**  | `<c:set var="변수명" value="값" scope="범위" target="객체" property="필드명" />`                 |
| **변수 삭제**  | `<c:remove var="변수명" scope="범위" />`                                                   |
| **단순 조건**  | `<c:if test="${조건식}" var="결과변수" scope="범위"> ... </c:if>`                              |
| **다중 조건**  | `<c:choose> ... </c:choose>` (속성 없음, 내부 태그 감싸기용)                                      |
| **조건 분기**  | `<c:when test="${조건식}"> ... </c:when>`                                                |
| **그 외**    | `<c:otherwise> ... </c:otherwise>` (속성 없음, `else` 역할)                                 |
| **반복문**    | `<c:forEach items="${리스트}" var="변수명" varStatus="상태변수" begin="시작" end="끝" step="증가" >` |
| **토큰 분리**  | `<c:forTokens items="문자열" delims="구분자" var="변수명" >`                                   |
| **URL 이동** | `<c:redirect url="이동할주소"> <c:param name="키" value="값"/> </c:redirect>`                |
| **URL 생성** | `<c:url value="경로" var="변수명" scope="범위"><c:param name="키" value="값"/></c:url>`        |


### 1.2. 속성 상세 설명

  * **`<c:set>`**
      * `var`: 값을 저장할 변수명 (필수).
      * `value`: 저장할 값 (필수).
      * `scope`: 변수 생존 범위 (`page`[기본], `request`, `session`, `application`).
      * `target`/`property`: 자바빈(JavaBean)이나 Map의 값을 수정할 때 사용.
  * **`<c:if>`**
      * `test`: `true`/`false`를 반환하는 조건식 (필수).
  * **`<c:forEach>`**
      * `items`: 반복할 데이터 객체(List, Array 등).
      * `var`: 현재 아이템을 담을 변수명.
      * `varStatus`: 반복 상태 객체 (index, count, first, last, current  등 확인 가능).

## 2\. Formatting Library (`prefix="fmt"`)

숫자, 날짜 형식 지정 및 다국어 처리를 담당하는 라이브러리입니다.

### 2.1. 태그 및 속성 구조

| 기능          | 태그 및 주요 속성 (Syntax)                                                                          |
| :---------- | :------------------------------------------------------------------------------------------- |
| **숫자 포맷**   | `<fmt:formatNumber var="변수명" value="값" type="유형" pattern="패턴" currencySymbol="기호"/>`         |
| **날짜 포맷**   | `<fmt:formatDate value="${Date객체}" type="유형" pattern="패턴" dateStyle="스타일" timeStyle="스타일"/>` |
| **파싱(역변환)** | `<fmt:parseDate var="변수명" value="문자열" pattern="패턴"/>`                                        |
| **로케일 설정**  | `<fmt:setLocale value="언어코드" />`                                                             |

### 2.2. 속성 상세 설명

  * **`<fmt:formatNumber>`**
      * `value`: 포맷팅할 숫자 (필수).
      * `pattern`: 사용자 지정 형식 (예: `#,###.00`).
      * `type`: `number`(기본), `currency`(통화), `percent`(퍼센트).
  * **`<fmt:formatDate>`**
      * `value`: `java.util.Date` 객체 (필수).
      * `pattern`: 날짜 형식 (예: `yyyy-MM-dd HH:mm`).

-----

## 3\. Functions Library (`prefix="fn"`)

**태그가 아닌 EL 함수 형태**(`${fn:함수()}`)로 사용되며, 주로 문자열 처리에 사용됩니다.

### 3.1. 함수 및 인자 구조

| 기능          | 함수 문법 (Syntax)                                     |
| :---------- | :------------------------------------------------- |
| **길이 확인**   | `${fn:length(객체)}`                                 |
| **문자열 자르기** | `${fn:substring(문자열, 시작idx, 끝idx)}`                |
| **문자열 치환**  | `${fn:replace(문자열, "기존문자", "새문자")}`                |
| **위치 찾기**   | `${fn:indexOf(문자열, "찾을문자")}`                       |
| **포함 여부**   | `${fn:contains(문자열, "찾을문자")}`                      |
| **대소문자**    | `${fn:toUpperCase(문자열)}`, `${fn:toLowerCase(문자열)}` |
| **공백 제거**   | `${fn:trim(문자열)}`                                  |
| **문자열 분리**  | `${fn:split(문자열, "구분자")}`                          |

### 3.2. 인자 상세 설명

  * **`length(obj)`**
      * `obj`: 문자열인 경우 글자 수, `Collection`(List 등)인 경우 아이템 개수를 반환.
  * **`substring(str, begin, end)`**
      * `begin`: 시작 인덱스 (0부터 시작, 포함됨).
      * `end`: 끝 인덱스 (포함되지 않음).
  * **`contains(str, sub)`**
      * 대소문자를 구분함. 구분하지 않으려면 `containsIgnoreCase` 사용.
  * **`split(str, delims)`**
      * 결과를 문자열 배열(`String[]`)로 반환하므로 주로 `<c:forEach>`와 함께 사용됨.

## 4\. 종합 예시 (Core + Fmt + Fn)

> [\!EXAMPLE] 게시판 목록 출력 예제
> 리스트를 반복(`c`)하며, 날짜를 포맷팅(`fmt`)하고, 제목이 너무 길면 잘라서(`fn`) 보여줍니다.

```ejs
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<%@ taglib prefix="fmt" uri="http://java.sun.com/jsp/jstl/fmt" %>
<%@ taglib prefix="fn" uri="http://java.sun.com/jsp/jstl/functions" %>

<table border="1">
    <tr>
        <th>번호</th>
        <th>제목 (요약)</th>
        <th>작성일</th>
        <th>첨부파일</th>
    </tr>
    
    <c:forEach items="${boardList}" var="board" varStatus="st">
        <tr>
            <td>${st.count}</td>
            
            <td>
                <c:choose>
                    <c:when test="${fn:length(board.title) > 10}">
                        ${fn:substring(board.title, 0, 10)}...
                    </c:when>
                    <c:otherwise>
                        ${board.title}
                    </c:otherwise>
                </c:choose>
            </td>
            
            <td>
                <fmt:formatDate value="${board.regDate}" pattern="yyyy.MM.dd" />
            </td>
            
            <td>
                <c:if test="${fn:contains(board.fileName, '.pdf')}">
                    [PDF 문서]
                </c:if>
            </td>
        </tr>
    </c:forEach>
</table>
```
