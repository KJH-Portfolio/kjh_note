# EL (Expression Language) 기본 및 연산자
> [!summary] 요약
> \*\*EL(표현 언어)\*\*은 JSP 2.0부터 도입된 데이터 출력 기술로, 기존의 `<%= %>` 표현식을 대체합니다. **`${}`** 문법을 사용하여 Scope에 저장된 데이터나 파라미터를 자바 코드 없이 간결하게 출력하며, 강력한 **Empty 연산자**를 지원합니다.
## 1.EL의 기본 문법 및 특징
  * **구문**: `${표현식}` 사용.
  * **Null 처리**: 값이 `null`이거나 존재하지 않아도 **에러(NullPointerException)를 발생시키지 않고 빈 문자열("")을 출력**하여 안전함.
  * **자동 형변환**: 문자열로 된 숫자도 자동으로 수치 연산이 가능함.
  * **접근 방법**: **Getter 메서드**를 호출하지 않고 멤버 변수명으로 접근 (예: `${vo.name}`은 내부적으로 `vo.getName()` 호출).
- ! EL표기법은 js에서 단순한 값으로 출력되기에(변수명으로 착각할 수 있음) 문자열이라면 "${userId}"처럼 작성해야함

## 2. Scope 검색 순서 (우선순위)

별도의 Scope 지정 없이 `${key}`만 입력했을 때, **좁은 영역부터 넓은 영역 순서**로 데이터를 자동 검색함.

1. `pageScope`
2. `requestScope` (가장 빈번하게 사용)
3. `sessionScope`
4. `applicationScope`

## 3. EL 연산자

자바 문법보다 직관적인 **키워드 연산자**를 추가로 지원.

| 분류        | 기호            | 키워드     | 설명                                 |
| :-------- | :------------ | :------ | :--------------------------------- |
| **산술**    | `+`, `-`, `*` |         | 숫자 연산에 사용                          |
|           | `/`           | `div`   | 나눗셈                                |
|           | `%`           | `mod`   | 나머지 (Modulo)                       |
| **비교**    | `==`          | `eq`    | 같다 (Equal)                         |
|           | `!=`          | `ne`    | 다르다 (Not Equal)                    |
|           | `<`           | `lt`    | 작다 (Less Than)                     |
|           | `>`           | `gt`    | 크다 (Greater Than)                  |
|           | `<=`          | `le`    | 작거나 같다 (Less or Equal)             |
|           | `>=`          | `ge`    | 크거나 같다 (Greater or Equal)          |
| **논리**    | `&&`          | `and`   | 논리 곱                               |
|           | `ㅣㅣ`          | `or`    | 논리 합                               |
|           | `!`           | `not`   | 논리 부정                              |
| **Empty** |               | `empty` | **핵심**.<br>null이거나 비어있으면 `true` 반환 |

### 3.1. Empty 연산자 상세

`${empty 변수명}` 형태로 사용하며, 다음의 경우 `true`를 반환.

  * 값이 `null` 일 때.
  * 빈 문자열 (`""`) 일 때.
  * 크기가 0인 **컬렉션 객체** (List, Map, Array) 일 때.

## 삼항 연산자_js
- 자바스크립트와 문법은 동일하지만 연산 결과가 마지막으로 연산한 피 연산자 값으로 남는다
```js
//결과값에 따라 A 또는 B를 반환
const result = 조건식? A : B;

//논리연산자를 이용한 조건문
result && A : result가 참이면(값이 있다면) A반환
result || B : result가 거짓이면(값이 없다면) A 반환
```

### 💡 개념 예시

> [!EXAMPLE] EL의 직관성 및 안전성
>
>   * **자바 코드**: `if(list != null && list.size() > 0)` (복잡하고 Null 체크 필수)
>   * **EL/JSTL**: `${not empty list}` (직관적이며, `list`가 null이어도 안전하게 동작)

### 💻 코드 예시

```jsp
이름 파라미터 출력: ${param.name} 

산술 연산: ${10 + 20}
삼항 연산: ${age ge 20 ? '성인' : '청소년'}

객체 접근 (Getter 대신): ${student.name}
리스트 인덱스 접근: ${subjectList[0]}

<c:if test="${empty boardList}">
    등록된 게시물이 없습니다.
</c:if>
```
