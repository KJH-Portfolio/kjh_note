# Action Tag (액션 태그 - jsp:)

> [!summary] 요약
> **액션 태그(`jsp:`)**는 XML 기술을 이용해 JSP 페이지 간의 **흐름 이동(Forward)**과 **동적 포함(Include)**을 자바 코드 없이 태그만으로 제어합니다. 디자인과 로직의 분리를 용이하게 합니다.

## 1. 흐름 제어 및 포함 비교

| 액션          | 태그                          | 동작 방식                                      | 특징                                                     |
| :---------- | :-------------------------- | :----------------------------------------- | :----------------------------------------------------- |
| **Forward** | `<jsp:forward page="경로" />` | 현재 페이지 작업을 **중단**하고 제어권을 대상 페이지로 완전히 넘김.   | **Request/Response 객체 유지**하며 데이터 공유 가능. **URL 변경 없음**. |
| **Include** | `<jsp:include page="경로" />` | 대상 페이지를 **실행**하고, **결과(HTML)**만 현재 위치에 포함. | 레이아웃 모듈화에 사용. 지역 변수는 공유되지 않음.                          |
## 2. 파라미터 전달
  * **태그**: `<jsp:param name="키" value="값" />`
  * **용도**: `forward`나 `include` 태그 내부에서 사용하여, 이동 또는 포함되는 페이지에 **추가 데이터**를 전달.
-----
### 💡 개념 예시
> [!EXAMPLE] Forward vs Include
>
>   * **Forward**: **전화 돌려주기**. "잠시만요, 담당자에게 연결해 드리겠습니다." (내 역할 종료, URL 유지)
>   * **Include**: **자료 인용**. "A팀 보고서를 참고해서 이 페이지에 붙여넣겠습니다." (내 페이지에 다른 결과 삽입)
### 💻 코드 예시
```ejs
<jsp:include page="/layout/header.jsp">
    <jsp:param name="title" value="메인 화면" />
</jsp:include>

<hr>

<c:if test="${empty sessionScope.user}">
    <jsp:forward page="loginError.jsp">
        <jsp:param name="msg" value="Access Denied" />
    </jsp:forward>
</c:if>
```
