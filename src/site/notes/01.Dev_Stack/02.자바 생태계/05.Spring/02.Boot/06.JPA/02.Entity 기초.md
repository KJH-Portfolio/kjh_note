# JPA 엔티티(Entity) 핵심 매핑 문법
> [!summary] 핵심 요약
> 
> 자바 객체와 관계형 데이터베이스 테이블 간의 매핑을 위한 핵심 어노테이션 및 제약 사항 정리.
> 
> 엔티티 생명 주기 관리를 위한 기본 생성자 준수 및 불변성 유지 원칙.
> 
> 기본키 생성 전략 및 필드 수준의 세부 매핑 설정을 통한 데이터 무결성 확보.
## 기본 엔티티 선언 가이드
- **@Entity**
    - JPA가 관리할 클래스임을 명시하는 필수 어노테이션.
    - 기본 생성자(No-argument Constructor) 필수 (public 또는 protected 권장).
    - 클래스 및 필드에 `final` 키워드 사용 금지.
- **@Table**
    - 엔티티와 매핑할 데이터베이스 테이블 이름 지정.
    - 생략 시 엔티티 클래스 이름을 테이블 이름으로 자동 매핑.
    - `uniqueConstraints`를 통한 복합 유니크 제약 조건 설정 가능.
## 기본키(PK) 매핑 및 생성 전략
- **@Id**
    - 테이블의 기본키(Primary Key)와 매핑될 필드 지정.
- **@GeneratedValue**
    - 기본키의 자동 생성 전략(Strategy) 결정.
    - **IDENTITY**: 기본키 생성을 데이터베이스에 위임 (MySQL, PostgreSQL 등).
    - **SEQUENCE**: 데이터베이스 시퀀스 오브젝트 사용 (Oracle, PostgreSQL 등).
    - **TABLE**: 키 생성 전용 테이블을 생성하여 시퀀스 흉내.
    - **AUTO**: 데이터베이스 방언에 따라 상기 전략 중 하나를 자동 선택.
```java
@Entity
@Table(name = "members")
public class Member {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    // ...
}
```
## 필드 및 컬럼 매핑 상세
- **@Column**
    - 객체 필드와 테이블 컬럼의 세부 매핑 설정.
    - `name`: DB 컬럼명 지정.
    - `nullable`: NULL 허용 여부 설정 (기본값 true).
    - `unique`: 단일 컬럼 유니크 제약 조건.
    - `length`: 문자열 길이 제한 (String 타입 전용).
- **@Enumerated**
    - 자바 Enum 타입을 매핑할 때 사용.
    - **EnumType.STRING**: Enum 이름을 DB에 저장 (권장).
    - **EnumType.ORDINAL**: Enum 순서를 DB에 저장 (비권장, 순서 변경 시 치명적 오류 발생).
- **@Lob**
    - 대용량 데이터(BLOB, CLOB) 매핑 시 사용.
- **@Transient**
    - 특정 필드를 데이터베이스 매핑에서 제외할 때 사용 (메모리 내 계산용).
```java
@Entity
public class Member {

    // 1. PK 설정 + 자동 생성 + 컬럼명 변경
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    @Column(name = "member_id") 
    private Long id;

    // 2. 일반 설정 + 제약 조건 3개 (이름, 필수, 길이)
    @Column(name = "user_name", nullable = false, length = 20)
    private String username;

    // 3. Enum 타입 매핑 + 필수 조건 추가 (가장 많이 쓰는 조합!)
    @Enumerated(EnumType.STRING)
    @Column(name = "user_role", nullable = false)
    private RoleType roleType;

    // 4. 대용량 데이터 + 컬럼명 명시 (함께 사용 가능!)
    @Lob
    @Column(name = "user_description")
    private String description;

    // 5. DB 저장 안 함 + 코드에서만 계산용으로 사용
    @Transient
    private String tempToken;
}
```
## 날짜 및 시간 매핑
- **TemporalType**
    - 과거 `java.util.Date`, `Calendar` 매핑 시 사용 (`DATE`, `TIME`, `TIMESTAMP`).
    - 최신 자바 버전(8 이상)의 `LocalDate`, `LocalDateTime` 사용 시 어노테이션 생략 가능.
