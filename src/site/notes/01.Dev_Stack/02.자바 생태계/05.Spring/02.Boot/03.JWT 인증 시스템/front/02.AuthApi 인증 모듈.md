# authApi.js 기반 회원 및 인증 API 통신 명세
> [!summary]
> 
> Axios 인스턴스를 활용하여 회원 가입, 로그인, 아이디 중복 체크, 정보 수정 및 탈퇴 등 **백엔드 인증 엔드포인트와의 통신**을 전담하는 API 모듈임.
---
## 인증 및 계정 관리 API
- **로그인 (login)**
    - 사용자 자격 증명(`credentials`)을 POST 방식으로 전달하여 인증 수행.
    - 서버로부터 토큰 및 사용자 정보 데이터를 반환받음.
- **로그아웃 (logout)**
    - 서버 측 세션 종료 또는 토큰 무효화를 위한 POST 요청 수행.
- **회원가입 (register)**
    - 신규 회원 정보(`submitData`)를 `/member/register` 경로로 전송.
    - 성공 시 201 상태 코드와 완료 메시지, 실패 시 500 상태 코드 응답 수신.
```javascript
// 로그인 API: 사용자 계정 정보 전송 및 응답 데이터 반환
login : async (credentials) => {
    const response = await api.post('/member/login', credentials);
    return response.data;
},
// 로그아웃 API: 서버 세션 종료 요청
logout : async () => {
    const response = await api.post('/member/logout');
    return response.data;
},
// 회원가입 API: 201(성공) 또는 500(실패) 상태값 반환 처리
register : async (submitData) => {
    const response = await api.post('/member/register', submitData);
    return response.data;
}
```

## 회원 정보 관리 및 검증 API
- **아이디 중복 체크 (checkIdDuplicate)**
    - 경로 변수(`PathVariable`) 형식을 사용하여 아이디 사용 가능 여부 조회.
    - 사용 가능 시 'NNNNY', 불가능 시 'NNNNN' 문자열 반환.
- **회원정보 수정 (updateMember)**
    - PUT 매핑을 사용하여 수정한 `formData`를 서버에 반영.
- **회원 탈퇴 (deleteMember)**
    - DELETE 매핑을 사용하며 보안을 위해 `userId`와 `userPwd`를 쿼리 파라미터(`params`)로 전달.
```javascript
// 아이디 중복 체크: 경로 변수(userId)를 통한 가용성 확인
checkIdDuplicate : async (userId) => {
    const response = await api.get(`/member/checkId/${userId}`);
    return response.data;
},
// 회원 정보 수정: PUT 메서드를 활용한 데이터 업데이트
updateMember : async (formData) => {
    const response = await api.put('/member/update', formData);
    return response.data;
},
// 회원 탈퇴: 복수 데이터(ID, PWD)를 params에 담아 DELETE 요청 전송
deleteMember : async (user) => {
    const response = await api.delete('/member/delete', {
        params : {
            userId : user.userId,
            userPwd : user.userPwd
        }
    });
    return response.data;
}
```
