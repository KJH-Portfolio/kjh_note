# React 라우트 접근 제어 시스템 (PrivateRoute, PublicRoute)
> [!summary]
> 
> `AuthContext`의 인증 상태를 참조하여 로그인 여부에 따라 특정 페이지의 접근 권한을 동적으로 제한하고 리다이렉션을 처리하는 가드(Guard) 컴포넌트 명세임.
---
## PrivateRoute (인증 회원 전용 라우트)
- **핵심 기능 및 목적**
    - **로그인 사용자 보호**: 인증된 회원만 특정 컴포넌트(children)에 접근하도록 제한함.
    - **위치 정보 보존**: `useLocation`을 활용하여 미인증 사용자가 강제 이동되기 전의 경로를 `state`에 담아 전달함.
- **상태별 렌더링 로직**
    - **로딩 중 (`isLoading`)**: 인증 확인 절차가 완료되기 전까지 로딩 UI 반환.
    - **비인증 상태 (`!isAuthenticated`)**: 경고 알림(`alert`) 출력 후 메인 페이지(`/`)로 강제 리다이렉트 처리.
    - **인증 완료**: 접근 허용 및 자식 컴포넌트(`children`) 렌더링.
```javascript
// 인증된 사용자만 접근 가능한 보호 라우트 관리
import { Navigate, useLocation } from "react-router-dom";
import { useAuth } from "../context/AuthContext";

// 인증 사용자 전용 가드 컴포넌트 (회원 전용 페이지)
export function PrivateRoute({children}) {
    // AuthContext에서 인증 상태 및 로딩 여부 추출
    const {isAuthenticated, isLoading} = useAuth();
    // 로그인 후 복귀를 위해 현재 브라우저 위치 정보 획득
    const location = useLocation();

    // 서버 또는 로컬스토리지에서 인증 정보를 확인 중인 경우
    if(isLoading){
        return <div>로딩 중....</div>;
    }

    // 미인증 사용자가 접근 시도 시 처리 로직
    if(!isAuthenticated){
        // 사용자 알림 메시지 표시
        alert('로그인 후 사용해주세요');
        // 메인 페이지로 이동하며, 원래 방문하려던 위치(from)를 상태로 전달
        return <Navigate to="/" state={{from : location}} replace/>
    }

    // 인증된 사용자인 경우 요청한 자식 컴포넌트 표시
    return children;
}
```

## PublicRoute (비인증 회원 전용 라우트)
- **핵심 기능 및 목적**
    - **비인증 접근 보장**: 로그인, 회원가입 등 로그인한 회원이 다시 접근할 필요가 없는 페이지 관리.
    - **중복 로그인 방지**: 이미 로그인된 사용자가 해당 경로 진입 시 메인 페이지로 자동 추방.
- **상태별 렌더링 로직**
    - **인증 상태 확인**: `isAuthenticated`가 `true`일 경우 접근을 차단하고 리다이렉트 실행.
```javascript
// 비인증 사용자만 접근 가능한 라우트 가드 (로그인/회원가입 등)
export function PublicRoute({children}){
    // 인증 상태 및 로딩 플래그 추출
    const {isAuthenticated, isLoading} = useAuth();
    
    // 인증 정보 동기화 중 로딩 UI 처리
    if(isLoading){
        return <div>로딩 중..</div>;
    }

    // 이미 로그인된 상태에서 접근 시 메인으로 리다이렉트
    if(isAuthenticated) {
        return <Navigate to="/" replace/>
    }

    // 로그인하지 않은 사용자인 경우 정상 렌더링
    return children;
}

export default PrivateRoute;
```
