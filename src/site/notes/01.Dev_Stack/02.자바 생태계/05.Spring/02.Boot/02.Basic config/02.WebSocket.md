# Spring WebSocket 설정 (WebSocketConfig)

> [!summary] 요약
> Spring Framework의 WebSocket 기능 활성화 및 각 기능별 핸들러(Basic, Group, Member,...)의 Bean 등록. URL 경로와 핸들러 매핑 및 `HttpSessionHandshakeInterceptor`를 통한 HTTP 세션 데이터 연동 설정.

## 기본 설정 및 활성화

> [!info] 설정 어노테이션 및 인터페이스
> * **@Configuration**: Spring 설정 클래스 명시.
> * **@EnableWebSocket**: WebSocket 요청 처리 활성화.
> * **WebSocketConfigurer**: WebSocket 핸들러 레지스트리 설정을 위한 인터페이스 구현 (`implements`).
> 
> 
```java
@Configuration
@EnableWebSocket
public class WebSocketConfig implements WebSocketConfigurer{...}
```
## WebSocket 핸들러 Bean 등록

> [!abstract] 핸들러 객체 관리
> 각 WebSocket 통신을 담당할 서버 구현체를 Spring Bean으로 등록하여 컨테이너 관리 위임.

```java
    // 각 웹소켓 서버를 Bean 등록하기
    @Bean
    public WebSocketHandler basicServer() {
        return new BasicServer();
    }
```
## 핸들러 매핑 및 인터셉터 설정

> [!example] `registerWebSocketHandlers` 메소드
> 클라이언트 접속 URL과 핸들러 연결 및 핸드쉐이크 과정의 인터셉터 설정.

* **URL 매핑**: `addHandler(핸들러, "주소")` 형식을 통해 요청 경로 지정.
* `/basic`, `/group`, `/member`등  경로 설정.


* **Interceptor 추가**: `addInterceptors(new HttpSessionHandshakeInterceptor())` 적용.
* **목적**: HTTP 세션(로그인 정보 등)을 WebSocket 세션(`WebSocketSession`)으로 전달.
* **적용 대상**: Member, Chat, Private 서버 (사용자 식별이 필요한 서비스).

```java
@Configuration //설정 파일임을 명시
@EnableWebSocket //Spring Boot : 웹소켓 기능 활성화
//웹소켓 커스터마이징 인터페이스
public class WebSocketConfig implements WebSocketConfigurer {

    // 각 웹소켓 서버를 Bean 등록하기
    @Bean
    public WebSocketHandler basicServer() {
        return new BasicServer(); //BasicServer.java 객체
    }

    @Bean
    public WebSocketHandler groupServer() {
        return new GroupServer();
    }
    
    // ... (나머지 Bean 등록 생략) ...

	//핸들러 등록
	@Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry registry) {

        // 기본 핸들러 등록
        // "/basic"으로 접속하면 basicServer()가 통신을 담당하도록 함
        registry.addHandler(basicServer(), "/basic");
        registry.addHandler(groupServer(), "/group");
		
		//인터셉터가 포함된 핸들러 등록
		//HTTP세션에 데이터를 웹소켓 세션(Attributes)에 복사하는 인터셉터
		//아이디나 유저 정보를 유지할 수 있다
        registry.addHandler(memberServer(), "/member")
                .addInterceptors(new HttpSessionHandshakeInterceptor());
        registry.addHandler(privateServer(), "/private")
                .addInterceptors(new HttpSessionHandshakeInterceptor());
        // ...(기타 생략)
    }
}
```
