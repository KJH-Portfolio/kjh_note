# Spring Security 암호화 빈 설정 명세

> [!NOTE] Title
> Contents
> [!summary]
> **BCryptPasswordEncoder** 빈 등록을 통해 애플리케이션 전역에서 사용할 비밀번호 암호화 객체 구성 요약.
## 주요 어노테이션 문법 (Core Annotations)
> [!info] 클래스 및 메서드 단위 설정 규칙
* **@Configuration** : 해당 클래스를 스프링 부트의 **설정 정보(Configuration)** 클래스로 지정.
* **@Bean** : 메서드에서 반환하는 객체를 스프링 컨테이너가 관리하는 **빈(Bean)**으로 등록.
* **역할** : XML 설정 파일 없이 자바 코드를 통한 의존성 주입(DI) 환경 구축.
---
## 암호화 객체 및 구현 (Encryption Object)
> [!tip] BCrypt 알고리즘 기반 보안 설정
* **BCryptPasswordEncoder** : 스프링 시큐리티에서 제공하는 **강력한 단방향 해시** 함수 구현체.
* **객체 생성 및 반환** : `new` 키워드를 통한 인스턴스화 및 스프링 빈 등록 수행.
* **활용 목적** : 회원가입 시 비밀번호 암호화 및 로그인 시 비밀번호 일치 여부 대조.
```java
// SecurityConfig.java 핵심 구조
@Configuration
public class SecurityConfig {

    @Bean
    public BCryptPasswordEncoder bcrypt() {
    
	    //BCrypt 객체 반환
        return new BCryptPasswordEncoder();
    }
}
```
