# Spring AOP 기반 Aspect 구성 및 실무 적용 지침

>[!summary]
Spring Framework에서 **AspectJ** 어노테이션을 활용한 관점 지향 프로그래밍(AOP) 구현 방법론. **Advice**의 시점별 활용, **Pointcut** 표현식의 세부 문법, **JoinPoint**를 통한 실행 시점 데이터 제어 및 실무적 응용(로그, 성능 측정, 결과 가공)을 포함한 명사형 요약.

---

## 주요 구성 어노테이션 및 환경 설정

> [!info] **Aspect 선언 필수 구성 요소**
> * **@Aspect** : 해당 클래스의 Aspect 지정.
> * **@Component** : Spring Bean 컨테이너 등록.
> * **@Slf4j** : Logback 기반 로그 기록 지원.(선택)

---

## Advice 시점 및 정의

* **@Before** : 대상 메서드 실행 전 시점 간섭.
* **@After** : 대상 메서드 실행 후 시점 간섭.
* **@Around** : 대상 메서드 실행 전후 통합 제어 및 실행 여부 결정.
* **@AfterReturning** : 메서드 정상 종료 후 결과값 반환 시점 간섭.
* **@AfterThrowing** : 메서드 실행 중 예외 발생 시점 간섭.

---

## Pointcut 표현식 및 대상 지정자

### 지시자 종류 (Designators)

* **target** : 특정 인터페이스 및 자식 클래스의 메서드 대상.
* **within** : 특정 패키지 또는 클래스 내부의 모든 메서드 대상.
* **execution** : 반환형, 패키지, 메서드명, 매개변수를 조합한 정교한 표현식 지정.
```java
//기본형태 예시
@Before("target(com.kh.spring.board.model.dao.BoardDao)")
public void before(JoinPoint joinpoint){...}
```

### execution 문법 구조

* **구조** : `execution([접근제한] 반환형 풀클래스명.메서드명(매개변수))`
* **와일드카드 (*)** : 모든 값 허용.
* **매개변수 표현** :
* `(*)` : 매개변수 1개 존재.
* `(..)` : 매개변수 개수 무관 (0개 이상).
* `[접근제한]`은 대체적으로 public

```java
// 특정 패키지 내 모든 클래스 메서드 대상
@Around("within(com.kh.spring.board.controller.*)")

// 특정 패키지 내 모든 클래스 및 메서드 대상 (매개변수 무관)
@Around("execution(* com.kh.spring.board.model.dao.*.*(..))")

// 특정 클래스의 특정 메서드 대상
@Around("execution(* com.kh.spring.board.model.dao.BoardDao.boardList(..))")

// 특정 반환형(int)을 가진 모든 메서드 대상
@Around("execution(int com.kh.spring.board.model.dao.BoardDao.*(..))")
```

## JoinPoint 객체 활용 및 메서드 제어

### JoinPoint (Before/After 활용)

* **getTarget().getClass()** : 실행 대상 클래스 정보 획득.
* **getSignature().getName()** : 실행 메서드 명칭 획득.
* **getArgs()** : 메서드 전달 인자(Arguments) 배열 반환.
### ProceedingJoinPoint (Around 활용)
> 위 JoinPoint 관련 기능도 사용 가능

* **proceed()** : 대상 메서드 본래 기능 수행.
* **proceed(args)** : 변경된 매개변수를 적용한 본래 기능 수행.
```java
@Around("target(...)")
public Object around(ProceedingJoinPoint joinPoint) throws Throwable{
		
		//실행 전 영역
	Object obj = joinPoint.proceed();
		//실행 후 영역
		
	return obj;
}

```

## 실무 기반 AOP 적용 예시

### 1. 로그 기록 및 전달값 확인

* **로그 출력 구성** : JoinPoint를 통한 메서드명 및 매개변수 배열의 문자열화 출력.

```java
public void before(JoinPoint joinPoint) {
    log.debug("실행 클래스 : {}", joinPoint.getTarget().getClass());
    log.debug("메서드명 : {}", joinPoint.getSignature().getName());
    log.debug("전달값 : {}", Arrays.toString(joinPoint.getArgs()));
}
```

### 2. 실행 시간 측정 (성능 분석)

* **측정 로직** : `System.currentTimeMillis()`를 이용한 실행 전후 시각 차이 산출.

```java
public Object timeCheck(ProceedingJoinPoint joinPoint) throws Throwable {
    long startTime = System.currentTimeMillis();
    Object obj = joinPoint.proceed();
    long endTime = System.currentTimeMillis();
    
    log.debug("현재 수행 대상 : {}", joinPoint.getSignature().getName());
    log.debug("수행시간 : {} 초", (endTime - startTime) / 1000);
    return obj;
}
```

### 3. 전달 매개변수 강제 조작

* **조작 로직** : 인자 배열의 특정 인덱스 객체 추출 및 필드 값 수정 후 재삽입.

```java
public Object memberAop(ProceedingJoinPoint joinPoint) throws Throwable {
    Object[] args = joinPoint.getArgs();
    Member m = (Member) args[0];
    m.setUserId("admin"); // 아이디 고정
    args[0] = m; 

    return joinPoint.proceed(args);
}
```

### 4. 반환값(Result) 가공 및 초기화

* **가공 로직** : `instanceof` 활용 타입 확인 후 특정 조건(ArrayList 등) 충족 시 빈 객체 반환.

```java
public Object emptyListAop(ProceedingJoinPoint joinPoint) throws Throwable {
    Object obj = joinPoint.proceed();
    if (obj instanceof ArrayList) {
        log.debug("리스트 결과값 초기화 처리");
        obj = new ArrayList<>();
    }
    return obj;
}
```

## 상세 구현 예시 : 성능 모니터링 및 데이터 보안 검증 Aspect

> [!example] **Service 계층 통합 제어 시나리오**
> * 서비스 메서드 실행 시간 측정.
> * 특정 반환 타입에 대한 데이터 마스킹 및 무결성 검증.
> 
> 

```java
@Aspect
@Component
@Slf4j
public class DetailAdvancedAspect {

    // Service 패키지 내 모든 public 메서드 대상
    @Around("execution(public * com.kh.spring..*Service.*(..))")
    public Object processServiceMonitoring(ProceedingJoinPoint joinPoint) throws Throwable {
        String targetMethod = joinPoint.getSignature().toShortString();
        Object[] args = joinPoint.getArgs();
        
        log.info("[START] 메서드: {} | 인자: {}", targetMethod, Arrays.toString(args));
        long start = System.currentTimeMillis();
        
        Object result;
        try {
            // 원본 메서드 실행
            result = joinPoint.proceed();
        } catch (Exception e) {
            log.error("[ERROR] {} 발생: {}", targetMethod, e.getMessage());
            throw e;
        } finally {
            long end = System.currentTimeMillis();
            log.info("[END] {} | 소요시간: {}ms", targetMethod, (end - start));
        }

        // 반환 데이터가 특정 VO인 경우 보안 처리 예시
        if (result instanceof Member) {
            Member m = (Member) result;
            m.setUserPwd("****"); // 비밀번호 마스킹 처리
            return m;
        }

        return result;
    }
}

```
