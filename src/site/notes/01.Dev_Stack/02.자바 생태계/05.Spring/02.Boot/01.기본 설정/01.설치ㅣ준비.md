# Spring Boot 개발 환경 구축 및 Lombok 설정 명세
> [!summary]
> **Amazon Corretto 17** 및 **STS 4(5.0.1)** 기반의 개발 환경 조성, **Maven** 프로젝트 초기화, **Eclipse Marketplace**를 통한 웹 도구 확장, 그리고 **Lombok** 라이브러리의 IDE 수동 연동 절차 요약.
## JDK 및 IDE 설치 환경 (JDK & STS Setup)
> [!info] 자바 개발 키트 및 통합 개발 환경 구성
* **Amazon Corretto 17** : Windows x64 JDK MSI 버전 설치 및 운용.
* **환경 변수 설정** : 시스템 변수 내 **JAVA_HOME** 등록 및 사용자 변수 **Path** 수정을 통한 자바 버전 관리 체계 구축.
* **STS(Spring Tool Suite) 4** : `spring.io/tools` 내 **5.0.1 - WINDOWS X86_64** 버전 선택 및 설치 완료.
---
## 프로젝트 초기화 및 의존성 구성 (Project Initialization)
> [!tip] Spring Starter Project 생성 명세
* **Service URL** : `https://start.spring.io` 기반 프로젝트 생성.
* **Project Type** : **Maven** 빌드 도구 선택.
* **Spring Boot Version** : **5.0.1** 최신 안정 버전 기준.
* **핵심 의존성(Dependencies)** :
* **Spring Boot DevTools** : 실시간 코드 변경 반영 등 개발 편의 도구.
* **Lombok** : Getter, Setter 등 반복 코드 자동 생성 라이브러리.
* **Spring Web** : RESTful 및 Spring MVC 웹 애플리케이션 구축 핵심 모듈.
* **Spring Security** : 사용자 로그인 처리, 권한 기반의 리소스 접근 제어, 그리고 해킹 공격(Session Fixation, Clickjacking 등)에 대한 방어 기능을 제공하는 통합 보안 솔루션
* **실행 방식** : 메인 메서드 직접 실행 혹은 **Run As > Spring Boot App** 선택을 통한 내장 톰캣 구동.

> [!example] War 방식 및 라이브러리 경로 설정

## Lombok 설치 및 IDE 연동 (Lombok Installation)
> [!important] 롬복 라이브러리 수동 적용 및 설정 파일 수정
* **Maven 설정** : **Preferences > Maven > User Settings** 내 로컬 저장소(ex: `C:\Users\user1\.m2\repository`) 경로 유효성 검증.
* **Lombok 실행 파일** : `repository\org\projectlombok` 경로 이동 후 해당 **jar** 파일 직접 실행 및 설치.
* **IDE 설정 파일(ini) 수정** : STS 설치 폴더 내 **sts.ini** 혹은 **sts5** 설정 파일 하단에 자바 에이전트 경로 명시.
* **최종 검증** : IDE 재시작 후 롬복 어노테이션(`@Data`, `@Getter` 등) 정상 동작 및 컴파일 여부 확인.
```
// sts.ini (혹은 관련 설정 파일) 수정 내용 예시
-vmargs
-javaagent:C:\실제 겅로\repository\org\projectlombok\lombok\1.18.30\lombok-1.18.30.jar

```
