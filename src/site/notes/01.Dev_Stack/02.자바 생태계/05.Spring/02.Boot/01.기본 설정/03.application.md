# Spring Boot application.properties 환경 설정 및 코드 명세

> [!summary]
> 서버 기동 환경, **JSP ViewResolver**, **Multipart** 파일 업로드, **Oracle DB** 및 **MyBatis** 연동, **JWT 보안 인증**, 그리고 패키지별 **로그 레벨** 제어를 위한 속성 정의 및 원본 코드 기술 요약.

## 서버 및 뷰 엔진 설정 (Server & View Config)

> [!info] 구동 환경 및 웹 자원 경로 정의

* **spring.application.name** : 프로젝트 식별을 위한 고유 애플리케이션 명칭.
* **server.port** : 웹 서버 구동 포트 번호 지정 (충돌 방지를 위한 사용자 정의).
* **server.servlet.context-path** : 애플리케이션 진입 시 공통으로 사용되는 루트 경로 설정.

```properties
spring.application.name=springbootJSP

#서버 포트번호 변경(자유)
server.port=8798

#ContextRoot 설정
server.servlet.context-path=/spring

```

> [!tldr]- War 방식일 때만 추가
> 설정 영역(war 방식에서만 필요함)
> JSP를 사용하기 위한 prefix, suffix 설정(viewResolver)
> spring.mvc.view.prefix=/WEB-INF/views/
> spring.mvc.view.suffix=.jsp

## 파일 업로드 및 데이터베이스 설정 (Multipart & DB Config)

> [!tip] 외부 자원 처리 및 데이터 소스 연결 핵심

* **Multipart 설정** : 단일 파일 및 전체 요청 용량 제한을 통한 서버 자원 보호.
* **DataSource 설정** : Oracle 데이터베이스 접속을 위한 드라이버, URL, 계정 정보 정의.
* **불필요 항목** : `driver-class-name`은 최신 라이브러리에서 자동 감지 가능하여 생략 가능.

```properties
#파일 업로드를 위한 MultipartResolver 설정
spring.servlet.multipart.enabled=true
spring.servlet.multipart.max-file-size=10MB
spring.servlet.multipart.max-request-size=10MB

#마이바티스 설정
#데이터베이스 정보 설정
spring.datasource.driver-class-name=oracle.jdbc.OracleDriver
spring.datasource.url=jdbc:oracle:thin:@localhost:1521:xe
spring.datasource.username=SPRING
spring.datasource.password=SPRING
```

## MyBatis 및 로깅 최적화 설정 (MyBatis & Logging)

> [!example] SQL 매핑 및 시스템 로그 기록 전략

* **MyBatis 최적화** : Null 처리 방식, 매퍼 위치, 카멜 표기법 변환 등 레거시 대비 간소화된 설정.
* **Logging 레벨** : 루트 및 패키지 단위의 로그 출력 세분화를 통한 디버깅 효율 증대.
* **정리 대상** : 주석 처리된 `logging.pattern` 및 `logging.file`은 가독성을 위해 삭제 혹은 유지 선택.

```properties
#mybaits-config에 작성한 setting들을 이곳에 작성
mybatis.configuration.jdbc-type-for-null=OTHER
#mappper 파일 경로
mybatis.mapper-locations=classpath:mappers/*.xml
#카멜표기법 처리
mybatis.configuration.map-underscore-to-camel-case=true
#별칭 설정(하위 패키지 설정)
mybatis.type-aliases-package=com.kh.spring

#log4j 설정
#로그 레벨 설정(기본 레벨)
logging.level.root=INFO

#패키지별 설정 또는 클래스 별 설정
logging.level.com.kh.spring.aop=DEBUG
logging.level.com.kh.spring.board=INFO

```

## JWT 보안 인증 설정 (JWT Security Config)

> [!quote] 토큰 기반 인증 및 보안 명세

* **jwt.secret** : 토큰 서명에 사용할 비밀 키 (최소 32바이트 권장).
* **jwt.expiration** : JWT 토큰 만료 시간 (밀리초 단위 설정).

```properties
#JWT 관련 설정
#JWT 토큰 서명에 사용할 비밀 키(최소32바이트)
jwt.secret=mysecretkeyTestHOHHOHOHO123qwqwdoiqjd2das0kdasd
#JWT 토큰 만료 시간 ( 밀리초 단위 설정 ) 
jwt.expiration=864000000

```
