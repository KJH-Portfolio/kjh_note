> Message 송수신을 위한 VO 객체 기본 필드
```java
package com.kh.spring.websocket.model.vo;
import ...
@AllArgsConstructor
@NoArgsConstructor
@Builder
@Data
public class MessageVO {
	private int type;//1 이면 일반채팅 / 2면 귓속말 
	private String myId;
	private String otherId;
	private String time;
	private String message;
}
```
