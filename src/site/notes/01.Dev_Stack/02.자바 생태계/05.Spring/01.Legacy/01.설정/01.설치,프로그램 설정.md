## 2\. ⚙️ Maven 설정 (settings.xml)
**1) Repository 폴더 생성**

**2) 설정 파일 수정**

```

```

  * 메뉴: `Window` \> `Preferences` \> `Maven` \> `User Settings`
  * **User Settings** 항목에 `apache-maven\src\conf\settings.xml` 파일 경로 등록
## 3. 📂 Spring Legacy Project 생성
### 📝 초기 세팅 (Metadata 오류 해결)
프로젝트 생성 시 템플릿 로딩이 안 될 경우 수행하는 작업입니다.
1. **초기화 시도:** IDE 상단 `File` \> `New` \> `Spring Legacy Project` 1회 클릭 (폴더 생성을 위해)
2. **파일 덮어쓰기 (Devtools 내 원본 파일 이용):**
      * `SpringWorkSpace\.metadata\.sts` 경로에 `content` 폴더 붙여넣기
      * `SpringWorkSpace\.metadata\.plugins\org.springsource.ide.eclipse.commons.content.core` 경로에 `https-content` 붙여넣기
3. **프로젝트 생성 재시도:** 다시 `New` \> `Spring Legacy Project` \> **Spring MVC Project** 선택
4. 기본 패키지 구조(Top-level package) 작성 (예: `com.company.app`)
## 4. 🏗️ 프로젝트 환경 설정 (Project Facets & Web)
[[3. Resource/03.Dev_Stack/02.자바 생태계/01.Java/01.설정,기능/01.환경 설정|01.환경 설정]] 해당 설정 완료
**1) Project Facets 설정**
  * 프로젝트 우클릭 \> `Properties` \> `Project Facets`
	  * **Dynamic Web Module**: `4.0` 으로 변경
	  * **Java**: `11` 로 변경
  * 프로젝트 Properties > Deployment Assembly에 Maven Dependencies가 있는지 확인
	  * 없다면 Add -> java build Path Entries -> next -> MavenDependencies
**2) 라이브러리 추가 (JSTL)**
  * 경로: `src/main/webapp/WEB-INF/lib` 폴더 생성
  * 해당 폴더에 `jstl` 관련 jar 파일 넣기
**3) 기타**
  * **Spring Explorer**: 메뉴 `Window` \> `Show View` \> `Spring Explorer` 활성화
  * 전체 환경설정에서 **java ee~** : Maven Archiver generates files under the build directory 체크 해제
## 5. 🌶️ Lombok 설치 (CMD)
`pom.xml` 설정 후에도 Lombok이 적용되지 않을 경우 수동 설치를 진행합니다.
1. **CMD(명령 프롬프트)** 실행
2. Lombok jar 파일이 있는 경로로 이동 (`cd` 명령어 사용)
3. 설치 명령어 실행 (`java -jar`)
```
:: 1. 해당 경로로 이동 (사용자 환경에 맞게 수정)
cd C:\Users\user1\Desktop\DevTools\apache-maven-3.8.9\apache-maven\src\repository\org\projectlombok\lombok\1.18.30

:: 2. 롬복 설치 실행
java -jar lombok-1.18.30.jar
```
4. **설치 창(GUI)이 뜨면:**
      * **Specify location...** 클릭
      * STS 경로 추가: `DevTools\sts-bundle\sts-3.9.18.RELEASE.exe`
      * Eclipse 경로 추가: `DevTools\eclipse-jee-2025-09-R-win32-x86_64\eclipse\eclipse.exe`
      * **Install / Update** 클릭 후 종료 및 IDE 재시작
