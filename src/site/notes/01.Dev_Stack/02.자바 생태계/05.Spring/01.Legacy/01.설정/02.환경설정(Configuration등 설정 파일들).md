# 환경 설정 (Configuration)

> [!summary] 요약
>
>   * **root-context.xml**: DB 연결, 트랜잭션 등 비즈니스 로직 관련 설정 (가장 먼저 로드됨).
>   * **mybatis-config.xml**: MyBatis 전역 설정 (Null 처리, 별칭 등).
>   * **Mapper.xml**: 실제 SQL 쿼리를 작성하는 파일.

## root-context.xml (비즈니스 설정)

서버 구동 시 `web.xml`을 거쳐 가장 먼저 읽히며, DB 연결 및 모듈을 세팅합니다.

### 주요 Bean 등록 코드

```xml
<bean class="org.apache.commons.dbcp2.BasicDataSource" id="dataSource">
    // JDBC 드라이버 클래스
    <property name="driverClassName" value="oracle.jdbc.OracleDriver"/>
    // DB URL (localhost, 포트, SID)
    <property name="url" value="jdbc:oracle:thin:@localhost:1521:xe" />
    // DB 계정명
    <property name="username" value="SPRING"/>
    // DB 비밀번호
    <property name="password" value="SPRING"/>
</bean>

<bean class="org.mybatis.spring.SqlSessionFactoryBean" id="sqlSessionFactory">
    // 위에서 만든 dataSource 연결
    <property name="dataSource" ref="dataSource"/>
    // MyBatis 설정 파일 위치 지정
    <property name="configLocation" value="classpath:mybatis-config.xml"/>
    // SQL문이 작성된 Mapper 파일 위치 지정 (*로 전체 지정)
    <property name="mapperLocations" value="classpath:mappers/*-mapper.xml"/>
</bean>

<bean class="org.mybatis.spring.SqlSessionTemplate" id="sqlSession">
    // 생성자를 통해 Factory 주입
    <constructor-arg ref="sqlSessionFactory"/>
</bean>
```

## mybatis-config.xml 및 Mapper

### 1. mybatis-config.xml

```xml
<configuration>
    <settings>
        // 데이터가 NULL일 경우 에러 방지 및 NULL로 처리
        <setting name="jdbcTypeForNull" value="NULL"/>
    </settings>
</configuration>
```

### 2. member-mapper.xml

```xml
// namespace: 외부에서 이 매퍼를 호출할 때 사용하는 별칭
<mapper namespace="memberMapper">
	//resultmap작성
	<resultMap type="Member" id="memberResultMap">
		  <result column="USER_ID" property="userId"/>
	</resultMap>
    // 이곳에 쿼리 태그 작성 (select, insert 등)
</mapper>
```

## 3\. 설정 파일별 역할 비교
1. web.xml (배포 서술자)
	- 서버 구동 시 가장 먼저 읽힘
	- 한글 인코딩 필터 등록
	- DispatcherServlet(메인 컨트롤러) 등록

1. servlet-context.xml (웹 설정)
	- ViewResolver (JSP 경로 앞/뒤 붙이기)
	- Interceptor (요청 가로채기)
	- 정적 리소스 (resources 폴더: js, css, img) 처리

1. root-context.xml (비즈니스 설정)
	- DB 접속 정보 (DataSource)
	- 트랜잭션 매니저
	- Service Bean 등록 등
