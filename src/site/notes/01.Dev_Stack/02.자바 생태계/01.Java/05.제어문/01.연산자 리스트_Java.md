# Java 연산자 (Operators) 핵심 정리
> [!summary] 핵심 요약
> 
> - Java 프로그래밍에 사용되는 단항, 이항, 삼항, 대입 연산자 및 접근 연산자의 종류와 동작 원리
>     
> - 목적: 조건 처리, 데이터 조작 및 효율적인 제어 흐름 구현을 위한 연산자 활용 능력 확보
>     
> - 핵심: 문자열의 논리적 동등성 비교 시 ==반드시 `equals()` 사용== (참조형의 `==` 사용 금지)
>     
## 연산자 종류 및 기호

|**구분**|**연산자 종류**|**기호 및 문법**|
|---|---|---|
|접근 및 그룹화|괄호, 배열 접근, 멤버 접근 등|`()`, `[]`, `.`, `{}`|
|단항 연산자|부호, 논리 반전, 형변환, 증감|`+`, `-`, `!`, `(자료형)`, `++`, `--`, `~`|
|이항 연산자|산술 연산|`*`, `/`, `%`, `+`, `-`|
||비트 이동 (Shift)|`<<`, `>>`, `>>>`|
||크기 및 동등 비교|`>`, `<`, `>=`, `<=`, `==`, `!=`|
||비트 논리 연산|`&`, `^`, `\|`|
||일반 논리 연산|`&&`, `\|`|
|삼항 연산자|조건 연산|`(조건식) ? 값1 : 값2`|
|대입 연산자|기본 및 복합 대입|`=`, `+=`, `-=`, `*=`, `/=`, `%=`, `<<=`, `>>=`, `>>>=`, `&=`, `^=`, `\|=`|
|나열 연산자|다중 선언 및 나열|`,`|

## 핵심 특징 및 동작 원리
- **접근 연산자 (`.`)**
    - 객체의 내부 필드 및 메서드 접근 수단
- **동등 비교 (`==`)**
    - 기본형(Primitive Type): 실제 데이터의 '값(Value)' 비교
    - 참조형(Reference Type): 메모리상의 ==물리적 '주소(Address)' 비교== (자주 헷갈리는 것)
- **문자열(String) 비교**
    - 주소 비교 방지를 위한 ==`equals()` 메서드 사용 필수== (금지·경고)
- **단락 평가 (Short-Circuit Evaluation)**
    - 논리 연산(`&&`, `||`) 시 좌측 항의 평가 결과만으로 전체 결과가 확정될 경우 ==우측 항 연산 생략== (핵심 개념)
    - Java 특성: 논리 결과값(`true`/`false`) 도출 시 즉시 평가 종료
    - JS 특성 비교: 단락 평가 원리는 동일하나, 평가 종료 시 논리값이 아닌 피연산자의 원본 값 자체 반환
## 주요 주의사항 및 예외 처리
- **의도치 않은 주소 비교 위험**
    - 참조형 변수에 대한 `==` 연산자 사용 시 논리적 동등성 검증 실패 우려
- **단락 평가의 부작용**
    - 우측 항에 증감 연산자나 메서드 호출 배치 시 ==실행 누락(Skip) 현상 발생 주의== (경고)
- **증감 연산자의 실행 순서**
    - 전위 연산(`++x`): 변수 참조 전 즉시 증감 수행
    - 후위 연산(`x++`): 현재 값을 먼저 반환한 후 증감 수행 (실행 순서 혼동 주의)
- **객체 배열의 복사 한계**
    - 단순 대입 연산자(`=`) 사용 시 값의 복사가 아닌 주소만 공유되는 ==얕은 복사(Shallow Copy)== 발생 (전제 조건)
## 연산자 활용 예시
- **문자열 비교 (동일성과 동등성)**
```java
String a = "hello";
String b = new String("hello");

System.out.println(a == b);        // false (메모리 주소 비교)
System.out.println(a.equals(b));   // true (논리적 값 비교)
```
- **전위/후위 증감 연산자**
```java
int x = 3;
int y = x++ + ++x; 
// 평가 순서:
// 1. x++ -> 3 평가 후 x=4로 증가
// 2. ++x -> x=4에서 5로 증가 후 5 평가
// 3. y = 3 + 5 = 8 도출
System.out.println("x=" + x + ", y=" + y); // 출력 결과: x=5, y=8
```
- **논리 연산자와 단락 평가(Short-Circuit)**
```java
int i = 0;
// 좌측 항(1 == 0)이 false이므로 && 우측 항(2 > i++)은 실행되지 않음
if (1 == 0 && 2 > i++) {
    // 실행 블록 진입 불가
}
System.out.println(i); // 출력 결과: 0 (증가 연산 미수행)
```
- **삼항 연산자를 활용한 조건 분기**
```java
int score = 75;
String result = (score >= 60) ? "합격" : "불합격";
System.out.println(result); // 출력 결과: 합격
```
