# Java 제어문 - 분기문 (Branch Statements)
> [!summary] 핵심 요약
> 
> - 루프의 실행 흐름을 강제로 제어하거나 메서드를 종료하는 분기문(`continue`, `break`, `return`, 라벨)의 구조
>     
> - 목적: 특정 조건에서의 불필요한 반복 건너뛰기, 반복문 조기 종료, 메서드 결괏값 반환 등 세밀한 실행 흐름 제어
>     
> - 핵심: 다중 중첩 루프 탈출 시 라벨 제어보다는 ==메서드 분리를 통한 `return` 사용 권장==
>     
## 분기문 종류 및 핵심 스펙

| **제어문**    | **제어 대상** | **핵심 동작 및 특징**                                        |
| ---------- | --------- | ----------------------------------------------------- |
| `continue` | 반복 스킵     | 현재 반복 회차의 남은 코드를 생략하고 ==다음 반복의 (조건식/증감식)으로 즉시 이동==    |
| `break`    | 루프/블록 종료  | 조건 검사와 무관하게 가장 가까운 단일 반복문 또는 `switch` 블록 완전 탈출        |
| `return`   | 메서드 종료    | 현재 실행 중인 ==메서드 자체를 즉각 종료==하고 호출자(Caller)에게 제어권 및 값 반환 |
| 라벨(Label)  | 외부 루프 제어  | 식별자가 지정된 특정 외부 반복문의 흐름(종료 또는 스킵) 직접 제어                |

## 주요 특징 및 실무 주의점
- **스코프(Scope) 및 제어 범위 차이**
    - `continue` 및 `break`: 호출 위치에서 가장 인접한 단일 반복문에만 제한적 적용
    - `return`: 루프 중첩 깊이와 무관하게 즉각적인 메서드 전체 종료 보장
- **`continue` 구문의 동작 흐름**
    - `for` 문 내부: `continue` 발생 시 ==증감식으로 이동==
    - `while` 문 내부: `continue` 발생 시 ==조건식으로 이동== (증감식 누락으로 인한 무한 루프 발생 주의)
- **라벨(Label) 사용의 부작용**
    - 임의의 위치로 제어권을 넘기는 특성상 실행 흐름 추적의 어려움 발생
    - 가독성 저하 및 유지보수성 하락의 우려가 있으므로 필수적인 상황 외 사용 지양
- **Early Return (조기 반환) 패턴**
    - 예외 조건이나 부정 조건을 먼저 검사하여 `return`으로 메서드를 즉시 종료하는 기법
    - 불필요한 `if-else` 중첩 방지 및 코드 가독성 향상 효과
## 분기문 활용 예시
> [!example] `continue` (특정 회차 건너뛰기)
```java
for (int i = 1; i <= 5; i++) {
    if (i == 3) continue; // i가 3일 때 하위 코드(출력)를 스킵하고 증감식(i++)으로 이동
    System.out.print(i + " ");
}
// 최종 출력: 1 2 4 5
```
> [!example] `break` (반복문 조기 종료)
```java
for (int i = 0; i < 5; i++) {
    if (i == 3) break; // i가 3이 되는 순간 반복문을 완전히 탈출
    System.out.print(i + " ");
}
// 최종 출력: 0 1 2
```
> [!example] `return` (메서드 종료 및 값 반환)
```java
public int calculateTarget(int value) {
    if (value < 0) {
        return 0; // 조건 부합 시 메서드 즉시 종료 (Early Return)
    }
    // 복잡한 연산 로직 생략
    return value * 100; // 최종 결괏값 반환
}
```
> [!example] 라벨 문법 (중첩 반복문의 외부 루프 제어)
```java
outerLoop: // 외부 루프에 'outerLoop'라는 라벨 지정
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (j == 1) {
            break outerLoop; // 가장 가까운 루프(j)가 아닌, 외부 루프(i) 전체를 종료
        }
        System.out.println("i=" + i + ", j=" + j);
    }
}
// 최종 출력: i=0, j=0 (j가 1이 되는 순간 외부 루프까지 완전 종료됨)
```
