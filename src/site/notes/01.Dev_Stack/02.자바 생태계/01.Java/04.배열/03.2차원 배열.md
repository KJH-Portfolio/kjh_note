# 📘 Java 2차원 배열 (2D Array)
> [!summary] 핵심 요약
> 
> - 1차원 배열의 참조(주소)를 요소로 가지는 다차원 배열 구조
>     
> - 목적: 행(Row)과 열(Column)로 구성된 매트릭스(Matrix) 형태의 데이터 집합 효율적 관리
>     
> - 핵심: `arr.length`(행)와 `arr[i].length`(열)의 명확한 구분 및 ==중첩 반복문(Nested Loop)==을 통한 순회 제어
>     
## 2차원 배열의 핵심 개념 및 구조
- **다차원 메모리 참조**
    - 실제 데이터가 아닌 1차원 배열들의 ==메모리 주소의 집합==을 관리하는 최상위 배열

|**구분**|**주요 특징**|
|---|---|
|**정의**|1차원 배열을 요소로 가지는 최상위 1차원 배열의 확장 형태|
|**구조**|행(Row) 역할을 하는 메인 배열과 열(Column) 역할을 하는 서브 배열의 계층적 결합|
|**형태 분류**|행과 열 크기가 동일한 정방형(Rectangular) 배열과 행마다 열 크기가 다른 가변(Jagged) 배열|
|**활용 분야**|관계형 데이터베이스 테이블 구조 매핑, 행렬 수학 연산, 2D 게임 그리드 좌표 시스템|

```plaintext
2D Jagged Array Memory Structure (int[][] arr = new int[3][])
 ├─ 1. Stack Area
 │   └─ arr (Heap 영역의 2차원 메인 배열 시작 주소 보관)
 └─ 2. Heap Area
     ├─ [메인 배열] arr (행 관리)
     │   ├─ index 0: 서브 배열 1의 주소 (ex: 0x100) -> new int[2]
     │   ├─ index 1: 서브 배열 2의 주소 (ex: 0x200) -> new int[4]
     │   └─ index 2: 서브 배열 3의 주소 (ex: 0x300) -> new int[3]
     │
     └─ [서브 배열] (열 관리 및 실제 데이터 보관)
         ├─ [0x100 위치] int, int
         ├─ [0x200 위치] int, int, int, int
         └─ [0x300 위치] int, int, int
```
## 배열 길이(Length) 및 인덱스 제어
- **행과 열의 독립적 길이 속성**
    - `arr.length`: 2차원 배열의 ==행(Row)의 전체 개수==
    - `arr[i].length`: `i`번째 행이 가리키는 내부 1차원 배열의 ==열(Column)의 개수==
- **인덱스 접근의 한계점**
    - 모든 차원의 인덱스는 ==0부터 시작== (접근 범위 초과 시 `ArrayIndexOutOfBoundsException` 발생)
## 주요 배열 생성 및 할당 패턴
- **가변(Jagged) 배열 구조화**
    - 행 크기 우선 지정 후, ==각 행별 열 크기 명시적 할당 필수== (미할당 인덱스 접근 시 `NullPointerException` 발생)
```java
    int[][] jagged = new int[3][]; // 행 크기만 지정, 각 행의 열은 null 상태
    jagged[0] = new int[2];        // 0번 행: 2개의 열 공간 생성
    jagged[1] = new int[4];        // 1번 행: 4개의 열 공간 생성
    jagged[2] = new int[3];        // 2번 행: 3개의 열 공간 생성
```
- **향상된 for문(Enhanced for-loop) 기반 데이터 순회**
    - 인덱스 직접 제어 없이 배열 내 모든 요소의 ==안전한 읽기 전용(Read-Only) 순회== 수행
    - `for (타입 변수 : 배열)` 구조의 계층적 중첩 활용
```java
    int[][] arr = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    
    // 전체 요소 출력 예시
    for (int[] row : arr) {         // 1단계: 메인 배열에서 1차원 배열(행) 추출
        for (int val : row) {       // 2단계: 서브 배열에서 개별 요소(열) 추출
            System.out.print(val + " ");
        }
        System.out.println();
    }
```
- **Stream API 기반 다차원 데이터 제어**
    - `Arrays.stream()`과 `flatMap`을 활용한 ==2차원 배열의 1차원 평탄화(Flatten)== 및 일괄 데이터 처리
```java
    int[][] numbers = {
        {1, 2, 3},
        {4, 5, 6}
    };
    
    // 2차원 배열 내 모든 요소의 합계 산출
    int totalSum = Arrays.stream(numbers)
                         .flatMapToInt(Arrays::stream) // 1차원 IntStream으로 변환
                         .sum();
```
- **2차원 객체 배열 결합**
    - 객체 배열 참조와 다차원 구조의 혼합 활용
    - 다차원 데이터 순회 및 처리 시 ==해당 인덱스의 객체 생성(new) 여부 검증== 최우선 수행
