# 📘 Java 배열 복사 (Array Copy)
> [!summary] 핵심 요약
> 
> - 자바 배열의 얕은 복사(Shallow Copy)와 깊은 복사(Deep Copy) 메커니즘
>     
> - 목적: 참조 공유 문제 방지 및 독립적인 배열 생성을 통한 데이터 안정성 확보
>     
> - 핵심: 원시 타입 배열과 객체/다차원 배열의 복사 방식 차이 인지 및 ==완전한 깊은 복사 로직 구현==
>     
## 메모리 복사 방식의 이해
- **배열 복사 유형 비교**

|**구분**|**메모리 할당**|**데이터 독립성**|**특징**|
|---|---|---|---|
|**얕은 복사 (Shallow Copy)**|동일 주소 참조|종속적 (동기화)|단순 대입(`=`)을 통한 ==참조값(메모리 주소)만 복사==|
|**깊은 복사 (Deep Copy)**|신규 메모리 할당|완전 독립|값 자체를 새로운 메모리 공간에 복사|

## 깊은 복사 지원 API 및 방법론
- **주요 복사 메서드 스펙 및 특징**

| **메서드**                               | **용도 및 특징**           | **성능 및 권장 상황**                               |
| ------------------------------------- | --------------------- | -------------------------------------------- |
| `clone()`                             | 가장 간편한 형태의 복사 지원      | 1차원 원시 타입 배열의 전체 복사에 적합                      |
| `System.arraycopy()`<br>`(매개변수 많음)`   | 원본과 대상의 세밀한 위치 지정 복사  | OS 레벨 네이티브 구현으로 ==대용량 복사 시 최고 성능==           |
| `Arrays.copyOf()`<br>`(원본 배열, 복사 배열)` | 배열의 처음부터 지정된 길이만큼 복사  | 내부적으로 `System.arraycopy()` 활용, 배열 크기 조절 시 유용 |
| `Arrays.copyOfRange()`                | 지정된 시작점부터 끝점까지의 부분 복사 | 특정 범위 데이터 추출 목적에 부합                          |

## 다차원 및 객체 배열 복사 시 주의사항
- **참조 타입(Reference Type) 복사의 한계**
    - 객체 배열 또는 2차원 이상의 다차원 배열에 `clone()` 적용 시 ==최상위 배열 껍데기만 깊은 복사== 수행
    - 내부 요소(객체 또는 하위 배열)들은 여전히 동일한 주소를 가리키는 얕은 복사 상태 유지
    - 완전한 독립성을 위해서는 ==반복문 또는 Stream을 활용한 내부 요소 개별 복사== 필수
```plaintext
Object / 2D Array Copy Process
 ├─ 1. Basic API Copy (clone, Arrays.copyOf)
 │   ├─ 최상위 배열: 신규 메모리 할당 [깊은 복사 성공]
 │   └─ 내부 요소(객체/배열): 동일 참조값 유지 [얕은 복사 상태] -> 사이드 이펙트 발생 위험
 │
 └─ 2. Full Deep Copy (Custom Implementation)
     ├─ 원본 배열 순회 (for loop / Stream API)
     └─ 각 요소별 `new Object()` 생성 또는 `.clone()` 호출 [완전한 깊은 복사 달성]
```
## 코드 구현 예시
- **1차원 원시 타입 배열의 복사**
```java
int[] original = {1, 2, 3};

// 1. 얕은 복사 (참조값 공유)
int[] shallowCopy = original;

// 2. 깊은 복사 (clone 활용)
int[] deepCopyClone = original.clone();

// 3. 깊은 복사 (System.arraycopy 활용)
int[] deepCopySys = new int[3];
System.arraycopy(original, 0, deepCopySys, 0, original.length);
//(원본 배열,복사 시작 인덱스,복사배열,복사될 시작 인덱스,몇개)
```
- **객체 배열의 완전한 깊은 복사**
```java
class Student {
    String name;
    Student(String name){ this.name = name;}
}

Student[] original = { new Student("홍길동"), new Student("김철수") };

// [권장] 완전한 깊은 복사 (새 객체 개별 생성)
Student[] deepCopy = Arrays.stream(original)
                           .map(s -> new Student(s.name)) // 각 요소마다 새 객체 생성 (핵심!)
                           .toArray(Student[]::new);      // 결과를 다시 배열로 변환
```
- **2차원 배열의 완전한 깊은 복사 (for문,Stream)**
```java
int[][] original2D = {{1, 2}, {3, 4}};

// 1. for문을 활용한 2차원 배열 깊은 복사
int[][] deepCopyFor = new int[original2D.length][];
for (int i = 0; i < original2D.length; i++) {
    // 각 행(1차원 배열)마다 개별적으로 clone() 호출
    deepCopyFor[i] = original2D[i].clone(); 
}

// 2. Stream API를 활용한 2차원 배열 깊은 복사 (Java 8 이상)
int[][] deepCopyStream = Arrays.stream(original2D)
                               .map(int[]::clone)
                               .toArray(int[][]::new);
```
