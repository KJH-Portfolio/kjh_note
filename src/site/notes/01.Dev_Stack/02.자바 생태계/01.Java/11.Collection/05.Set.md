# 📘 Set 계열 정리

## Summary 📝

> [!summary] Summary 📝
> 
> - `Set`은 **순서가 없는 데이터 집합**으로, **중복을 허용하지 않음**
> - 인덱스 기반 접근 불가 → 반복문이나 Iterator로만 접근 가능
> - 대표 구현체: `HashSet`, `LinkedHashSet`, `TreeSet`

---

## 목적 🎯

- 중복 없는 데이터 집합 관리
- 순서와 상관없이 **고유한 값**만 저장
- 검색, 포함 여부 확인에 유용

---

## 핵심 개념 💡

### ✅ Set 특징

|구분|설명|
|---|---|
|순서|없음 (단, `LinkedHashSet`은 입력 순서 유지, `TreeSet`은 정렬)|
|인덱스|없음|
|중복|불가|
|구현체|`HashSet`, `LinkedHashSet`, `TreeSet`|

### ✅ 주요 메서드

```java
set.add("A");           // boolean
                       // 요소 추가 (중복된 경우 false 반환)

set.remove("A");        // boolean
                       // 특정 요소 삭제

set.contains("A");      // boolean
                       // 특정 요소 포함 여부 확인

set.size();             // int
                       // Set에 포함된 요소 개수 반환

set.clear();            // void
                       // 모든 요소 삭제

set.iterator();         // Iterator<E>
                       // Set의 반복자 반환
```

---

## 특징 ✨

- 순서 없음 → 인덱스로 접근 불가
- 중복 불가 → 고유한 값만 저장
- `TreeSet`은 자동 정렬 지원

---

## 팁과 주의점 ⚠️

- 중복 데이터 삽입 시 `add()`는 `false` 반환
- 순차 접근 시 반드시 **Iterator** 또는 **for-each** 사용
- 정렬이 필요하다면 `TreeSet` 사용
