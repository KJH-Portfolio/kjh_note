# 📘 Properties 클래스 정리

## Summary 📝

> [!summary] Summary 📝
> 
> - **Properties**는 `Map` 계열 컬렉션으로 key와 value를 **String 타입**으로 제한
> - 주로 **프로퍼티 파일(.properties)**을 읽고 쓰는 데 사용
> - JDBC, Spring Boot 등에서 **설정 파일 관리** 용도로 활용
> - 파일 입출력 지원 메서드 제공 (`store`, `load`, `storeToXML`, `loadFromXML`)

---

## 목적 🎯

- 파일 입출력을 편리하게 할 수 있는 Map 계열 중 한가지지

---

## 핵심 개념 💡

### ✅ Properties 특징

| 구분            | 내용                                      |
| ------------- | --------------------------------------- |
| 상속 구조         | `Hashtable<Object,Object>`를 상속받은 Map 계열 |
| key/value 타입  | **String만 허용**                          |
| 주요 사용처        | `.properties` 파일 읽기/쓰기, 설정 관리           |
| properties 파일 | `key=value` 형태의 텍스트 파일                  |
| 문자셋           | ISO 8859-1 기본, 한글은 Unicode 변환 저장        |

---

## 특징 ✨

### ✅ 프로퍼티 파일(.properties)

- 옵션 정보, DB 연결 정보, 다국어 메시지 등을 기록하는 텍스트 파일
- `key=value` 형식으로 작성
- 예시:
    
    ```
    db.user=root
    db.password=1234
    app.language=ko
    ```
    

### ✅ Properties 주요 메서드
- get,set
- store(ToXML),load(FromXML)
```java
properties.getProperty("key")
properties.setProperty("key", "value");    // Object: 이전 값 반환
properties.store(out_S, "설명");
properties.store(writer, "설명");
properties.storeToXML(out_S, "설명");
properties.load(in_S);
properties.load(reader);
properties.loadFromXML(in_S);
```
---

## 팁과 주의점 ⚠️

- 동일 key 저장 시 **덮어쓰기** 발생
- XML파일에 특수 문자 사용하는 법
	- 1. <![CDATA[ ... ]]>에서는 특수 문자를 문자열로 인식하며 escape하지 않아도 됨
	- 2. 지정된 명령어 사용
		- &lt; : less than (<)
		- &gt; : greater than (>)
		- &amp; : ampersand (&)
		- &apos; : apostrophe (')
		- &quot; : quotation mark (")

## 예시 📌

### ✅ 파일 저장 및 불러오기

```java
// 파일 저장
try (FileOutputStream fos = new FileOutputStream("config.properties")) {
    prop.store(fos, "App Configurations");
}

// 파일 읽기
Properties loadProp = new Properties();
try (FileInputStream fis = new FileInputStream("config.properties")) {
    loadProp.load(fis);
}
System.out.println(loadProp.getProperty("Key1"));
```
