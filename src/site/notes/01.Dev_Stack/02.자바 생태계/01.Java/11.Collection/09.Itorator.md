# 📘 Iterator 계열 - Iterator, ListIterator

## 목적 🎯

- 컬렉션 요소를 반복적으로 접근하기 위한 인터페이스 제공

## 핵심 개념 💡

| 인터페이스        | 특징                                       |
| ------------ | ---------------------------------------- |
| Iterator     | 단방향 탐색, `hasNext()`, `next()` 제공         |
| ListIterator | 양방향 탐색, `hasPrevious()`, `previous()` 추가 |


## 특징 ✨

- List와 Set에서 사용 가능
- Map은 `keySet()` 또는 `entrySet()`으로 Set 객체로 변환 후 사용

## 팁과 주의점 ⚠️

- `Iterator` 사용 시 `remove()`로 안전하게 삭제 가능
- `ListIterator`는 `List`에서만 사용 가능

## 예시 📌

```java
// Iterator
Iterator<String> it = set.iterator();
while (it.hasNext()) {
    System.out.println(it.next());
}

// ListIterator
ListIterator<String> lit = list.listIterator();
while (lit.hasNext()) {
    System.out.println(lit.next()); // 앞에서부터 탐색
}
while (lit.hasPrevious()) {
    System.out.println(lit.previous()); // 뒤에서부터 탐색
}
```
