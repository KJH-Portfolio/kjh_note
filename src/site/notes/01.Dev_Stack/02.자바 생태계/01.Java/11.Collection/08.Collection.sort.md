# 📘 정렬 (Collections.sort)

## Summary 📝

> [!summary] Summary 📝
> 
> - **Collections.sort**는 리스트를 정렬하기 위한 유틸리티 메서드로, 오름차순·내림차순·무작위 섞기 기능을 제공함
> - **Comparable**은 클래스 내부에서 정렬 기준을 정의하고, **Comparator**는 외부에서 별도의 정렬 기준을 정의함

---

## 목적 🎯

- `Comparable`: 객체 자체에 **기본 정렬 기준** 부여
- `Comparator`: 필요에 따라 **추가적·다양한 정렬 기준** 제공

---

## 핵심 개념 💡

| 구분    | Comparable                                     | Comparator                                                      |
| ----- | ---------------------------------------------- | --------------------------------------------------------------- |
| 패키지   | `java.lang`                                    | `java.util`                                                     |
| 메서드   | `int compareTo(T o)`                           | `int compare(T o1, T o2)`                                       |
| 정의 위치 | 클래스 내부                                         | 클래스 외부                                                          |
| 정렬 기준 | 1개만 정의 가능 (자연 순서)                              | 여러 개 정의 가능 (상황별 적용)                                             |
| 적용 방식 | `Collections.sort(list)` → 내부 `compareTo()` 사용 | `Collections.sort(list, comparator)` 또는 `list.sort(comparator)` |
| 활용 예  | 숫자 오름차순, 문자열 사전순                               | 내림차순, 다중 조건 정렬, 사용자 지정 규칙                                       |

---

## 특징 ✨

- `Comparable`: **객체 자체가 정렬 기준을 알고 있음**
- `Comparator`: **외부에서 정렬 기준을 주입**
- 두 인터페이스를 함께 사용 가능 (`Comparable` 기본 + `Comparator` 추가 기준)


---

## 팁과 주의점 ⚠️

- `Comparable`은 **한 가지 기준만** 정의 가능 → 다양한 기준 필요 시 `Comparator` 사용
- `comparator` **반환값 의미 (`compareTo`, `compare`)**
    - 양수: 현재 객체 > 비교 객체
    - 0: 동일
    - 음수: 현재 객체 < 비교 객체
- `Collections.shuffle(List)` : 무작위 섞기

---

## 예시 📌

```java
import java.util.*;

class Music implements Comparable<Music> {

    // Comparable 구현: 제목 기준 오름차순
    @Override
    public int compareTo(Music o) {
        return this.title.compareTo(o.title);
    }

}

// 외부 Comparator 클래스 정의 (가수명 기준 오름차순)

class SingerComparator implements Comparator<Music> {
    @Override
    public int compare(Music o1, Music o2) {
        return o1.getSinger().compareTo(o2.getSinger());
    }
}

// 메인 클래스
public class SortExample {
    public static void main(String[] args) {
        List<Music> list = new ArrayList<>();
        list.add(new Music("하늘", "김가수"));
        list.add(new Music("바다", "이가수"));
        list.add(new Music("구름", "박가수"));

        // **기본 기준 정렬 (Music 클래스에서 정의)**
        Collections.sort(list);
        System.out.println("기본 정렬: " + list);

        // **Comparator 기준 정렬 (해당 클래스에서 정의)**
        Collections.sort(list, new Comparator<Music>() {
            @Override
            public int compare(Music o1, Music o2) {
                return o1.getSinger().compareTo(o2.getSinger());
            }
        });
        
        //**Comparator 기준 정렬 (전용 Comparator에서 정의)**
        Collection.sort(list,new SingerComparator<Music>()) //Music은 생략가능
        
        // 무작위 섞기
        Collections.shuffle(list);

    }
}
```
