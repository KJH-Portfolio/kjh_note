# 📘 자료구조

## Summary 📝

> [!summary] Summary 📝
> 
>- 컬렉션(Collection)은 자바에서 제공하는 자료구조를 담당하는 프레임워크 
>- 추가, 삭제, 정렬 등의 기능처리가 간단하게 해결 되어 자료구조적 알고리즘을 구현할 필요 없음
>- 자료구조의 분류(단순·선형·비선형)와 자바 컬렉션 프레임워크(`List`, `Set`,` Map`, `Properties`, `Iterator`) 정리
>- `java.util` 패키지

# 📘 자료구조

## 목적 🎯

- 데이터를 메모리에서 구조적으로 처리하기 위한 방법

## 핵심 개념 💡
###  자료구조

| 대분류           | 중분류        | 세부 항목                        |
| ------------- | ---------- | ---------------------------- |
| **단순구조** (변수) | -          | 정수, 실수, 문자, 문자열              |
| **선형구조**      | 리스트        | -                            |
|               | 연결리스트      | 단순 연결리스트, 이중 연결리스트, 원형 연결리스트 |
|               | 스택         | -                            |
|               | 큐          | -                            |
|               | 덱(Deque)   | -                            |
| **비선형구조**     | 트리(Tree)   | 일반트리, 이진트리                   |
|               | 그래프(Graph) | 방향그래프, 무방향그래프                |
| **파일구조**      | 순차파일       | -                            |
|               | 색인파일       | -                            |
|               | 직접파일       | -                            |
|               |            |                              |
## 특징 ✨

- 단순구조: 기본 데이터 타입 중심
- 선형구조: 순차적 접근, 삽입·삭제 위치에 따라 성능 차이 발생(ArrayList)
- 비선형구조: 계층적·복잡한 관계 표현 가능(,Hashset,HashMap)

## 팁과 주의점 ⚠️

- 선형구조는 자바 컬렉션 프레임워크에서 직접 구현
- 비선형구조는 알고리즘 구현 시 직접 정의하거나 라이브러리 활용
