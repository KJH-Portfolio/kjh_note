# Eclipse IDE 생산성 극대화 단축키 가이드
> [!summary] 핵심 요약
> 
> - 주제: Eclipse IDE 환경에서의 코드 편집, 내비게이션, 리팩토링 필수 단축키 정리
>     
> - 목적: 마우스 조작 최소화 및 개발 피드백 루프 가속화
>     
> - 핵심: ==Ctrl + Space==(자동완성), ==Ctrl + 1==(퀵 픽스), ==Ctrl + Shift + F==(포맷팅)
>     
## 코드 편집 및 자동화 (Editing)
- **코드 품질 자동 관리**
    - 설정된 템플릿에 따른 일괄 정렬 및 들여쓰기 교정
    - 불필요한 의존성 제거 및 누락된 패키지 자동 추가
- **작성 효율화**
    - 에러 발생 시 즉각적인 해결책 제시 기능 활용
    - 클래스, 메서드, 변수명 자동 완성을 통한 오타 방지

|**기능**|**단축키**|**설명**|
|---|---|---|
|**코드 포맷팅**|==Ctrl + Shift + F==|코드 정렬 및 스타일 자동 교정|
|**임포트 정리**|==Ctrl + Shift + O==|사용하지 않는 import 제거 및 추가|
|**빠른 수정**|==Ctrl + 1==|에러 지점 해결 방법(Quick Fix) 제안|
|**자동 완성**|==Ctrl + Space==|코드 추천 및 단어 완성|
|**주석 토글**|`Ctrl + /`|현재 행 또는 선택 영역 단일 주석 처리|
|**블록 주석**|`Ctrl + Shift + /`|선택 영역 블록 주석(/* */) 처리|
|**주석 해제**|`Ctrl + Shift + \`|설정된 블록 주석 제거|

## 내비게이션 및 검색 (Navigation)
- **심볼 추적 및 탐색**
    - 메서드 선언부로 즉시 이동하여 논리 흐름 파악
    - 특정 메서드를 호출하는 모든 지점을 역추적하여 영향 범위 확인
- **자원 검색 및 배치**
    - 파일명 검색을 통한 리소스 즉시 접근
    - 행 단위 이동 및 복사 기능을 활용한 코드 구조 재배치

| **대상**    | **단축키**              | **설명**                |
| --------- | -------------------- | --------------------- |
| **선언 이동** | ==F3==               | 변수/메서드 정의 위치로 즉시 이동   |
| **파일 찾기** | ==Ctrl + Shift + R== | 파일명 입력을 통한 리소스 즉시 열기  |
| **호출 계층** | ==Ctrl + Alt + H==   | 특정 메서드 호출 지점 역추적      |
| **전체 검색** | `Ctrl + H`           | 파일 내용 및 타입 상세 검색 창 호출 |
| **행 이동**  | `Alt + ↑ / ↓`        | 현재 행을 상하로 한 칸씩 이동     |
| **행 복사**  | `Ctrl + Alt + ↑ / ↓` | 현재 행을 상하 방향으로 즉시 복제   |

## 실행 및 리팩토링 (Run & Refactor)
- **구조적 리팩토링**
    - 참조된 모든 위치를 동시에 수정하여 무결성 유지
    - 중복 코드를 메서드로 추출하여 재사용성 향상
- **런타임 제어**
    - 최근 실행 구성을 기반으로 한 즉각적인 서비스 가동
    - 브레이크 포인트 기반의 단계별 로직 검증
```plaintext
Refactor (코드 구조 개선)
 ├─ Rename (이름 변경): Shift + Alt + R
 │   └─ 변수, 메서드, 클래스명 일괄 변경
 ├─ Move (위치 이동): Shift + Alt + V
 │   └─ 패키지 또는 클래스 간 이동
 └─ Extract Method (메서드 추출): Shift + Alt + M
     └─ 선택 영역을 새로운 메서드로 분리
```
- **실행 및 디버그 제어**
    - **일반 실행**: ==Ctrl + F11== (최근 실행 구성 가동)
    - **디버깅 시작**: ==F11== (중단점 기반 단계별 실행)
