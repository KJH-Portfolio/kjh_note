# Eclipse 효율 극대화 단축키 가이드
> [!summary] 핵심 요약
> 
> - 내용: Eclipse IDE 환경에서의 코드 편집, 내비게이션, 리팩토링 필수 단축키 정리
>     
> - 목적: 반복적인 마우스 조작 최소화 및 코드 작성 속도 비약적 향상
>     
> - 핵심: ==Ctrl + Space==(자동완성), ==Ctrl + 1==(퀵 픽스), ==Ctrl + Shift + F==(포맷팅)
>     
## 코드 편집 및 자동화 (Editing)
- **코드 품질 관리**
    - 소스 코드의 가독성 확보 및 표준 규격 준수를 위한 자동 정렬 기능 활용
    - 불필요한 임포트 제거 및 누락된 패키지의 일괄 추가 프로세스 최적화

| **기능**     | **단축키**              | **설명**                         |
| ---------- | -------------------- | ------------------------------ |
| **코드 포맷팅** | ==Ctrl + Shift + F== | 설정된 템플릿에 맞게 코드 자동 정렬           |
| **임포트 정리** | ==Ctrl + Shift + O== | 사용하지 않는 import 제거 및 필요한 항목 추가  |
| **빠른 수정**  | ==Ctrl + 1==         | 에러 발생 지점에서 해결 방법(Quick Fix) 제안 |
| **자동 완성**  | ==Ctrl + Space==     | 클래스, 메서드, 변수명 등 코드 추천 및 완성     |
| **주석 토글**  | Ctrl + /             | 선택 영역 또는 현재 행의 단일 주석 처리        |
| **블록 주석**  | Ctrl + Shift + /     | 선택 영역 전체를 블록 주석(/* */) 처리      |
| **주석 해제**  | Ctrl + Shift + \     | 설정된 블록 주석 제거 (역슬래시 앞 공백 주의)    |

## 내비게이션 및 검색 (Navigation)
- **심볼 추적 및 이동**
    - 선언부와 구현부 사이의 빠른 전환으로 코드 흐름 파악 시간 단축
    - 프로젝트 규모 확장 시 파일 및 텍스트 검색 효율성 극대화

|**대상**|**단축키**|**설명**|
|---|---|---|
|**선언 이동**|F3|변수나 메서드가 정의된 위치로 즉시 이동|
|**파일 찾기**|==Ctrl + Shift + R==|파일명 입력을 통한 리소스 즉시 열기|
|**전체 검색**|Ctrl + H|파일 내용, 타입 등 상세 검색 창 호출|
|**호출 계층**|Ctrl + Alt + H|특정 메서드를 호출하는 지점 역추적|
|**행 이동**|Alt + ↑ / ↓|현재 행을 위 또는 아래로 한 칸씩 이동|
|**행 복사**|Ctrl + Alt + ↑ / ↓|현재 행을 위 또는 아래에 즉시 복제|

## 실행 및 리팩토링 (Run & Refactor)
- **안정적 구조 변경**
    - 변수나 클래스명 변경 시 참조된 모든 위치의 일괄 수정을 통한 무결성 유지
    - 빌드 및 실행 프로세스의 단축키 통합으로 피드백 루프 가속화
- **Refactoring 트리 구조**
```plaintext
Refactor
 └─ Rename (Shift + Alt + R)
 └─ Move (Shift + Alt + V)
 └─ Extract Method (Shift + Alt + M)
```
- **실행 제어**
    - **디버깅 시작**: F11 (브레이크 포인트 기반 단계별 실행)
    - **일반 실행**: ==Ctrl + F11== (최근 실행 구성으로 즉시 런타임 가동)
