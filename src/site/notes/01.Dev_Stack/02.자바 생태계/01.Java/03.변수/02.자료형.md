# Java - 자료형(Data Type)의 체계와 메모리 구조
> [!summary] 핵심 요약
> 
> - Java의 정적 타입 시스템을 구성하는 **데이터 분류 체계**
>     
> - 목적: 메모리 사용 최적화 및 컴파일 시점의 ==타입 안정성(Type Safety)== 확보
>     
> - 핵심: 8가지 기본형(Primitive)과 객체의 주소값을 다루는 참조형(Reference)의 구분
>     
## 1. 자료형의 대분류 및 메모리 배치
- **기본형(Primitive Type)**
    - 실제 데이터 값을 메모리(Stack)에 직접 저장
    - 사전에 정의된 8가지 고정 크기 타입만 존재
- **참조형(Reference Type)**
    - 실제 객체는 ==Heap 영역==에 생성
    - Stack 영역에는 객체의 메모리 주소(Reference)를 저장
```plaintext
Java Data Type Hierarchy
 ├─ 1. Primitive Type (기본형)
 │   ├─ 정수형: byte(1), short(2), int(4), long(8)
 │   ├─ 실수형: float(4), double(8)
 │   ├─ 문자형: char(2)
 │   └─ 논리형: boolean(1)
 │
 └─ 2. Reference Type (참조형)
     ├─ Class (String 및 사용자 정의 클래스)
     ├─ Interface
     ├─ Array (배열)
     └─ Enumeration (열거형)
```
## 2. 기본형 자료형(Primitive Type) 상세 스펙

|**분류**|**자료형**|**크기**|**주요 특징 및 주의사항**|
|---|---|---|---|
|**논리형**|`boolean`|1 byte|`true`, `false` 표현 (조건문/반복문 핵심)|
|**문자형**|`char`|2 byte|유니코드 기반 단일 문자 저장 (홑따옴표 `' '` 사용)|
|**정수형**|`byte`|1 byte|이진 데이터 처리 시 활용|
||`short`|2 byte|C언어와의 호환성 유지용|
||`int`|4 byte|==정수 연산의 기본 단위== 및 기본 리터럴 타입|
||`long`|8 byte|큰 범위 정수, ==숫자 뒤 `L` 접미사 필수==|
|**실수형**|`float`|4 byte|정밀도가 낮음, ==숫자 뒤 `f` 접미사 필수==|
||`double`|8 byte|==실수 연산의 기본 단위== 및 기본 리터럴 타입|

## 3. 참조형(Reference Type)과 리터럴 활용
- **String 클래스**
    - 문자열 저장을 위한 참조형 타입 (쌍따옴표 `" "` 사용)
    - 불변성(Immutability) 특징을 가지며 별도의 리터럴 풀(Pool) 관리
- **참조형의 특수성**
    - 실제 데이터가 없는 상태인 `null` 할당 가능
    - 초기화되지 않은 참조 변수 접근 시 ==NPE(NullPointerException)== 발생 위험

| **구분**     | **기본형 (Primitive)**   | **참조형 (Reference)** |
| ---------- | --------------------- | ------------------- |
| **저장 값**   | 실제 데이터(Value)         | ==메모리 주소(Address)== |
| **기본값**    | `0`, `0.0`, `false` 등 | `null`              |
| **사용 방식**  | 산술 연산 가능              | 메서드 호출 및 필드 접근 가능   |
| **제네릭 활용** | 불가능 (Wrapper 클래스 필요)  | 가능                  |

## 4. 자료형별 선언 및 표기 예시
- **정수 및 실수 리터럴**
```java
long distance = 2147483648L; // int 범위 초과 시 L 접미사 필수
float pi = 3.14f;            // f 접미사 생략 시 double로 인식되어 컴파일 에러
```
- **문자 및 문자열**
```java
char grade = 'A';            // 단일 문자는 홑따옴표
String message = "Hello";    // 문자열은 쌍따옴표 (객체 타입)
```
