# 📘 Wrapper 클래스 정리

## Summary 📝

> [!summary] Summary 📝
> 
> - 기본형 데이터를 객체로 포장하여 참조형처럼 다룰 수 있게 해주는 클래스
> - Boxing/Unboxing을 통해 기본형 ↔ 참조형 간 변환 지원
> - 컬렉션, 제네릭 등 **객체만 허용되는 구조**에서 활용됨

---

## 목적 🎯

- **컬렉션/제네릭**에서 기본형을 저장할 수 없으므로 Wrapper로 변환해 사용
- **객체 기반 기능 활용**: 메서드 호출, null 처리, 비교 연산 등 가능
- 기본형과 객체형 간 변환을 통해 **유연한 프로그래밍 지원**

---

## 핵심 개념 💡

|개념|설명|
|---|---|
|Wrapper 클래스|기본형을 객체로 포장하여 참조형처럼 다루는 클래스|
|Boxing|기본형 → Wrapper 객체 변환|
|Unboxing|Wrapper 객체 → 기본형 추출|
|Auto Boxing/Unboxing|Java 5부터 자동 변환 지원|

---

## 특징 ✨

### 기본형 ↔ Wrapper 클래스 대응표

| Primitive | Wrapper       |
| --------- | ------------- |
| `boolean` | `Boolean`     |
| `byte`    | `Byte`        |
| `char`    | **Character** |
| `short`   | `Short`       |
| `int`     | **Integer**   |
| `long`    | `Long`        |
| `float`   | `Float`       |
| `double`  | `Double`      |

### 주요 메서드

| 메서드                   | 설명             | 예시                                                                                                                                                                                                     |
| --------------------- | -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `compareTo(T o)`      | 객체 간 값 비교      | `Integer a = 100; a.compareTo(200)` → -1 (Integer)<br>`Double x = 3.14; x.compareTo(2.71)` → 1 (Double)                                                                                                |
| `parseXxx(String s)`  | 기본형으로 변환       | `Integer.parseInt("123")` → 123 (int)<br>`Long.parseLong("456")` → 456L (long)<br>`Double.parseDouble("3.14")` → 3.14 (double)                                                                         |
| `valueOf(String s)`   | Wrapper 객체로 변환 | `Integer.valueOf("123")` → Integer 객체; <br>`Boolean.valueOf("true")` → Boolean 객체; <br>`Long.valueOf("789")` → Long 객체                                                                                 |
| `toString(primitive)` | 기본형을 문자열로 변환   | `Integer.toString(123)` → "123"<br>`Double.toString(3.14)` → "3.14"<br>`Long.toString(456L)` → "456"                                                                                                   |
| `isDigit(char ch)`    | 숫자인지 판별        | `Character.isDigit('5')` → true<br>`Character.isDigit('A')` → false <br>(다른 Wrapper와 결합: 숫자 검증 후 parseXxx 호출)                                                                                          |
| `xxxValue()`          | 기본형으로 추출       | `Integer boxed = 10; int unboxed = boxed.intValue();` → 10<br>`Double dbl = 3.14; double unboxed = dbl.doubleValue();` → 3.14 <br>`Boolean bool = true; boolean unboxed = bool.booleanValue();` → true |




---

## 팁과 주의점 ⚠️

- `==` 비교는 **참조값 비교**, 값 비교는 반드시 `equals()` 사용
- Auto Boxing/Unboxing은 편리하지만 **불필요한 객체 생성**에 주의
- `parseXxx()`는 문자열을 기본형으로, `valueOf()`는 문자열을 Wrapper 객체로 변환
- `null` 상태의 Wrapper를 Unboxing하면 `NullPointerException` 발생 가능

---

## 예시 📌

```java
// Boxing
int num = 10;
Integer boxed = Integer.valueOf(num); // 명시적 Boxing
Integer autoBoxed = num;              // 자동 Boxing

// Unboxing
int unboxed = boxed.intValue();       // 명시적 Unboxing
int autoUnboxed = boxed;              // 자동 Unboxing

// 비교
Integer a = 100;
Integer b = 200;
System.out.println(a.compareTo(b)); // -1
System.out.println(a.equals(100));  // true

// 문자열 변환
String str = Integer.toString(123);   // "123"
int parsed = Integer.parseInt("456"); // 456
Integer wrapped = Integer.valueOf("789"); // Integer 객체

// Double 변환
double pi = Double.parseDouble("3.14159");
System.out.println(pi); // 3.14159

// Character 메서드 활용
char c = '7';
System.out.println(Character.isDigit(c)); // true
System.out.println(Character.isLetter(c)); // false

// NullPointerException 주의
Integer nullable = null;
// int x = nullable; // 실행 시 NullPointerException 발생
```
