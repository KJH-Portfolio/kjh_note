# 📘 java.lang 패키지
> [!summary] 핵심 요약
> 
> - Java 프로그래밍의 가장 필수적인 기본 클래스들을 모아둔 핵심 패키지
>     
> - 목적: 문자열, 수학 연산, 시스템 제어 등 필수 기능의 기본 탑재로 개발 편의성 극대화
>     
> - 핵심: 별도의 `import` 선언 없이 ==자동으로 로드==되는 유일한 패키지
>     
## 주요 클래스 (Core Classes)
- **클래스 계층 구조**
    - 모든 기본 클래스는 `Object`를 상속받으며 애플리케이션의 핵심 뼈대 구성
```plaintext
java.lang Hierarchy
 └─ Object (모든 클래스의 최상위 부모)
     ├─ String (문자열)
     ├─ System (시스템 제어)
     ├─ Math (수학 연산)
     ├─ Thread (스레드)
     ├─ Throwable (예외/오류)
     │   ├─ Error
     │   └─ Exception
     └─ Number (Wrapper 클래스 부모)
         ├─ Integer
         └─ Double
```

|**클래스**|**주요 기능 및 특징**|
|---|---|
|`Object`|자바의 ==최상위 부모 클래스== (객체 비교 및 문자열 변환 기본 형태 제공)|
|`String`|불변(Immutable) 속성을 가진 문자열 데이터 처리|
|`System`|표준 입출력 스트림 제어 및 시스템 환경 정보 접근|
|`Math`|정적(Static) 메서드 기반의 수학적 계산 및 난수 생성|
|`Wrapper`|기본 데이터 타입(Primitive)을 객체 형태로 포장 (`Integer` 등)|
|`Thread`|멀티스레드 환경 구축 및 프로세스 제어|
|`Throwable`|런타임 내 발생하는 예외 및 오류 처리의 최상위 기준|

## 클래스별 활용 및 핵심 주의사항
- **Object 클래스**
    - 기본 메서드는 참조(메모리 주소)값을 기준으로 동작하므로, 논리적 비교를 위한 `equals()`, `hashCode()`, `toString()`의 ==오버라이딩 습관화 필수==
```java
class Person {
    String name;
    Person(String name) { this.name = name; }

    @Override
    public String toString() {
        // 참조값이 아닌 의미 있는 데이터가 출력되도록 재정의
        return "Person: " + name; 
    }
}
```
- **String 클래스**
    - 상태가 변하지 않는 불변 객체이므로 잦은 문자열 결합 연산 시 메모리 누수 발생
    - 동적 문자열 조작이 필요한 경우 `StringBuilder` 또는 `StringBuffer` 사용 권장
```java
String greeting = "Hello";
// 기존 객체 변경이 아닌 새로운 String 객체 메모리 할당
String upper = greeting.toUpperCase(); 
```
- **System 클래스**
    - `System.exit(0)` 호출 시 실행 중인 JVM 자체가 즉각 강제 종료되므로 웹 애플리케이션 등에서 ==실무 사용 엄금==
```java
// 가장 대표적인 System 표준 출력 스트림 활용
System.out.println("출력 예시");
```
- **Math 클래스**
    - `Math.random()`은 `0.0 <= x < 1.0` 범위의 `double` 난수 반환
    - 정수형 난수 추출 시 명시적 캐스팅(Casting) 및 추가 연산 필요
```java
// 2의 3제곱 계산 (반환형은 double)
double result = Math.pow(2, 3); // 8.0
```
- **Wrapper 클래스**
    - 컬렉션 프레임워크(`List`, `Map` 등)는 객체만 취급하므로 기본형을 담기 위한 필수 변환(Boxing) 도구
```java
int num = 10;
// 기본형 int를 객체형 Integer로 래핑 (Java 5+ 부터는 Auto-boxing 지원)
Integer boxed = Integer.valueOf(num); 
```
