## 📘 `java.time` 통합

`java.time` 패키지에서 자주 사용되는 핵심 메서드들을 **매개변수 타입**과 **반환값 타입**을 중심으로 정리함

---

### 1. `LocalDate`, `LocalTime`, `LocalDateTime` (지역 시간 클래스)

이 클래스들은 모두 **불변(Immutable)** 객체입니다. 따라서 시간을 변경하는 연산은 **새로운 객체를 반환**합니다.

| **클래스**             | **자주 쓰는 메서드**      | **매개변수 타입**                             | **반환값 타입**      | **기능**                                         |
| ------------------- | ------------------ | --------------------------------------- | --------------- | ---------------------------------------------- |
| **`LocalDate`**     | `now()`            | 없음                                      | `LocalDate`     | 시스템 기본 시간대의 **현재 날짜**를 반환.                     |
|                     | `of()`             | `(int year, int month, int dayOfMonth)` | `LocalDate`     | 지정된 년, 월, 일로 날짜 객체를 생성.                        |
|                     | `plusDays()`       | `(long daysToAdd)`                      | `LocalDate`     | 현재 날짜에 일(日)을 더한 **새로운 객체**를 반환.                |
|                     | `minusMonths()`    | `(long monthsToSubtract)`               | `LocalDate`     | 현재 날짜에서 월(月)을 뺀 **새로운 객체**를 반환.                |
|                     | `isAfter()`        | `(ChronoLocalDate other)`               | `boolean`       | 인수로 주어진 날짜보다 나중인지 여부를 반환.                      |
| **`LocalTime`**     | `now()`            | 없음                                      | `LocalTime`     | 시스템 기본 시간대의 **현재 시간**을 반환.                     |
|                     | `of()`             | `(int hour, int minute, [int second])`  | `LocalTime`     | 지정된 시, 분, 초로 시간 객체를 생성.                        |
|                     | `plusHours()`      | `(long hoursToAdd)`                     | `LocalTime`     | 현재 시간에 시간(時)을 더한 **새로운 객체**를 반환.               |
| **`LocalDateTime`** | `now()`            | 없음                                      | `LocalDateTime` | 시스템 기본 시간대의 현재 **날짜와 시간**을 반환.                 |
|                     | `withDayOfMonth()` | `(int dayOfMonth)`                      | `LocalDateTime` | 날짜의 일(日)을 변경한 **새로운 객체**를 반환.                  |
|                     | `toLocalDate()`    | 없음                                      | `LocalDate`     | `LocalDateTime`에서 날짜 부분(`LocalDate`)만 추출하여 반환. |

---

### 2. `ZonedDateTime`, `Instant` (시간대 및 시점 클래스)

| **클래스**             | **자주 쓰는 메서드**           | **매개변수 타입**       | **반환값 타입**      | **기능**                                      |
| ------------------- | ----------------------- | ----------------- | --------------- | ------------------------------------------- |
| **`ZonedDateTime`** | `now()`                 | `([ZoneId zone])` | `ZonedDateTime` | 지정된/현재 시간대의 현재 시각을 반환.                      |
|                     | `withZoneSameInstant()` | `(ZoneId zone)`   | `ZonedDateTime` | **시점(Instant)은 유지**한 채 시간대만 변경한 새로운 객체를 반환. |
|                     | `toInstant()`           | 없음                | `Instant`       | 해당 시간대의 시각을 **UTC 기준의 시점(`Instant`)**으로 변환. |
| **`Instant`**       | `now()`                 | 없음                | `Instant`       | UTC 기준으로 현재의 순간(타임스탬프)을 반환.                 |
|                     | `toEpochMilli()`        | 없음                | `long`          | 해당 시점을 **밀리초(ms)** 단위의 유닉스 타임스탬프로 반환.       |

---

### 3. `DateTimeFormatter` (포맷팅 클래스)

|**메서드**|**매개변수 타입**|**반환값 타입**|**기능**|
|---|---|---|---|
|**`ofPattern()`**|`(String pattern, [Locale locale])`|`DateTimeFormatter`|지정된 패턴 문자열(예: `"yyyy년 MM월 dd일"`)로 포맷터를 생성.|
|**`format()`**|`(TemporalAccessor temporal)`|`String`|`LocalDate`, `LocalDateTime` 등의 객체를 포맷터 형식의 **문자열로 변환**.|
|**`parse()`**|`(CharSequence text, DateTimeFormatter formatter)`|`TemporalAccessor` (또는 해당 클래스)|지정된 문자열과 포맷터 정보를 이용해 날짜/시간 객체로 **파싱(변환)**.|
