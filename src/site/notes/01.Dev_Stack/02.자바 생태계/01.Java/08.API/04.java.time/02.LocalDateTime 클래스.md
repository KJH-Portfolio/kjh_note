# 📘 LocalDateTime 클래스 활용 정리

## Summary 📝

> [!summary] Summary 📝
> 
> - `LocalDateTime`은 Java 8부터 도입된 날짜·시간 클래스이며, `Date`/`Calendar`보다 안전하고 직관적임
> - 현재 시간 확인, 특정 날짜 지정, 날짜 연산 기능을 간단히 구현 가능
> - 불변 객체로 설계되어 연산 시 원본이 변경되지 않음

---

## 목적 🎯

- 현재 또는 특정 날짜·시간을 다루기 위한 활용
- 날짜 연산(더하기/빼기) 및 포맷팅을 통한 다양한 시간 기반 로직 구현
- 멀티스레드 환경에서도 안전하게 사용

---

## 핵심 개념 💡

|항목|설명|
|---|---|
|현재 시간|`LocalDateTime.now()`|
|특정 시간 지정|`LocalDateTime.of(년, 월, 일, 시, 분, 초)`|
|날짜 추출|`getYear()`, `getMonthValue()`, `getDayOfMonth()`|
|시간 추출|`getHour()`, `getMinute()`, `getSecond()`|
|날짜 계산|`plusDays()`, `minusMonths()` 등으로 간단히 연산|
|불변성|연산 결과는 새로운 객체 반환 (원본 변경 없음)|

---

## 특징 ✨

- `Date`/`Calendar`의 단점을 보완한 **현대적 API**
- 직관적인 메서드명으로 가독성 향상
- `LocalDate`, `LocalTime`과 결합해 세분화된 시간 관리 가능
- `DateTimeFormatter`와 함께 사용해 다양한 포맷 출력 가능

---

## 팁과 주의점 ⚠️

- `LocalDateTime`은 **타임존 정보를 포함하지 않음** → 타임존 필요 시 `ZonedDateTime` 사용
- 포맷팅 시 `DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss")` 활용
- 불변 객체이므로 연산 결과는 반드시 **새로운 변수에 할당**해야 함

---

## 예시 📌

```java
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

public class LocalDateTimeExample {
    public static void main(String[] args) {
        // 현재 날짜와 시간
        LocalDateTime now = LocalDateTime.now();
        System.out.println("현재: " + now);

        // 특정 날짜 지정
        LocalDateTime endOfYear = LocalDateTime.of(2025, 12, 31, 23, 59, 59);
        System.out.println("지정 날짜: " + endOfYear);

        // 날짜 정보 추출
        int year = now.getYear();
        int month = now.getMonthValue();
        int day = now.getDayOfMonth();
        int hour = now.getHour();
        int minute = now.getMinute();
        int second = now.getSecond();

        System.out.println("년: " + year);
        System.out.println("월: " + month);
        System.out.println("일: " + day);
        System.out.println("시: " + hour);
        System.out.println("분: " + minute);
        System.out.println("초: " + second);

        // 날짜 계산
        System.out.println("5일 후: " + now.plusDays(5));
        System.out.println("1시간 후: " + now.plusHours(1));
        System.out.println("5달 후: " + now.plusMonths(5));
        System.out.println("21일 전: " + now.minusDays(21));
        System.out.println("8시간 전: " + now.minusHours(8));
        System.out.println("한달 전: " + now.minusMonths(1));

        // 포맷팅
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy년 MM월 dd일 HH:mm:ss");
        System.out.println("포맷팅된 현재: " + now.format(formatter));
    }
}
```
