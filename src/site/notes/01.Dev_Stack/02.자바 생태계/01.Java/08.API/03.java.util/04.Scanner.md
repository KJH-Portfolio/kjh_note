# 📘 Scanner

## Summary 📝

> [!summary] Summary 📝
> 
> - `Scanner`는 키보드 입력을 포함한 다양한 입력 소스를 처리하기 위한 자바의 표준 입력 클래스
> - 문자열, 정수, 실수 등 **다양한 데이터 타입 입력 가능**

---

## 목적 🎯

- 콘솔 기반 프로그램에서 사용자 입력을 손쉽게 처리
- 문자열, 숫자, 실수 등 다양한 타입의 데이터를 입력받아 활용
- 파일, 문자열 등 다른 입력 소스와도 연동 가능

---

## 핵심 개념 💡

|구분|설명|
|---|---|
|패키지|`java.util`에 포함 → `import java.util.Scanner;` 필요|
|입력 소스|`System.in`을 통해 키보드 입력 처리|
|다형성|파일, 문자열 등 다양한 입력 소스와 연동 가능 (`new Scanner(new File("경로"))`)|
|자원 관리|사용 후 반드시 `close()`로 입력 스트림 해제|

---

## 특징 ✨

|메서드|설명|예시|
|---|---|---|
|`next()`|공백 전까지 문자열 입력|`sc.next()`|
|`nextLine()`|한 줄 전체 입력|`sc.nextLine()`|
|`nextInt()`|정수 입력|`sc.nextInt()`|
|`nextDouble()`|실수 입력|`sc.nextDouble()`|
|`nextBoolean()`|불리언 입력|`sc.nextBoolean()`|
|`close()`|입력 스트림 자원 해제|`sc.close()`|

---

## 팁과 주의점 ⚠️

- `nextInt()`, `nextDouble()` 등 숫자 입력 후 `nextLine()`을 바로 쓰면 **버퍼에 남은 개행 문자** 때문에 입력이 건너뛰어질 수 있음 → `sc.nextLine()`으로 버퍼 정리 필요
- `Scanner`는 내부적으로 **정규 표현식 기반 파싱**을 사용
- 파일 입력 시 `new Scanner(new File("경로"))` 형태로 활용 가능
- 입력 스트림(`System.in`)은 프로그램 전체에서 하나만 존재하므로, 여러 Scanner 객체를 동시에 열지 않는 것이 안전

---

## 예시 📌

### ✅ 기본 입력 처리

```java
import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        // Scanner 객체 생성
        Scanner sc = new Scanner(System.in);

        // 문자열 입력 처리
        System.out.print("이름 입력: ");
        String name = sc.nextLine();

        // 정수 입력 처리
        System.out.print("나이 입력: ");
        int age = sc.nextInt();

        // 실수 입력 처리
        System.out.print("키 입력(cm): ");
        double height = sc.nextDouble();

        // 출력 처리
        System.out.println("안녕하세요, " + name + "님!");
        System.out.println("나이: " + age + "세, 키: " + height + "cm");

        // 자원 해제 처리
        sc.close();
    }
}
```

### ✅ 파일 입력 처리

```java
import java.io.File;
import java.io.FileNotFoundException;
import java.util.Scanner;

public class FileScannerExample {
    public static void main(String[] args) throws FileNotFoundException {
        Scanner sc = new Scanner(new File("data.txt"));

        while (sc.hasNextLine()) {
            String line = sc.nextLine();
            System.out.println(line);
        }

        sc.close();
    }
}
```
