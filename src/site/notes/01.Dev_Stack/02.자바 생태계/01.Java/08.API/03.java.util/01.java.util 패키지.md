# 📘 java.util 주요 유틸리티 클래스 정리

## Summary 📝

> [!summary] Summary 📝
> 
> - `java.util`은 자바의 핵심 유틸리티 패키지로, **입력, 배열, 컬렉션 등의 기능을 제공
> - `Scanner`, `Arrays` 등은 실무에서 자주 사용되는 대표 클래스
> - 최신 자바에서는 `java.time`, `Stream`, `Optional` 등과 함께 병행 사용하면 더 강력한 기능 구현 가능

---

## 목적 🎯

- 자바 개발 시 반복적으로 필요한 기능들을 **표준화된 클래스 형태로 제공**
- 복잡한 알고리즘 없이도 **간단한 API 호출만으로 기능 구현 가능**
- 코드의 **재사용성과 가독성 향상**, 유지보수 편의성 확보

---

## 핵심 개념 💡

- `java.util` 패키지는 **자바에서 가장 널리 사용되는 범용 유틸리티 클래스 모음**
- **컬렉션, 난수, 문자열 처리, 입력 처리 등** 다양한 기능 제공
- 자바 프로그래밍의 기본 도구들이 포함되어 있어 **실무와 학습에서 필수적**

---

## 주요 클래스 ✨

| 클래스명                     | 용도       | 특징                                |
| ------------------------ | -------- | --------------------------------- |
| `Scanner`                | 입력 처리    | 콘솔/파일/문자열 등 다양한 입력 소스 지원          |
| `Arrays`                 | 배열 유틸리티  | 정렬, 검색, 복사 등 배열 관련 기능 제공          |
| `Collections`            | 컬렉션 유틸리티 | 리스트/셋/맵 등의 정렬, 동기화, 변경 불가능 처리 등   |
| `HashMap`, `ArrayList` 등 | 자료구조     | 다양한 컬렉션 구현체 제공 (Map, List, Set 등) |
| `ThreadLocalRandom`      | 난수 생성    | 멀티스레디에서 난수 생성                     |
| `Formatter`              | 포맷팅      | 문자,숫자 등의 서식을 지정한다.                |

---

## 팁과 주의점 ⚠️

- `Collections` 클래스는 **정적 메서드**로만 구성된 유틸리티 클래스

---

## 예시 📌

```java
// 1. Scanner 클래스: 콘솔 입력 처리
System.out.println("--- 1. Scanner (입력 처리) ---");
try (Scanner scanner = new Scanner(System.in)) {
	System.out.print("이름을 입력하세요: ");
	String name = scanner.nextLine();
	System.out.println("환영합니다, " + name + "님!");
} // try-with-resources 구문으로 scanner 자동 닫기

// ---

// 2. Arrays 클래스: 배열 유틸리티
System.out.println("\n--- 2. Arrays (배열 유틸리티) ---");
int[] numbers = {50, 20, 80, 10, 60};

// 배열 정렬
Arrays.sort(numbers);
System.out.println("정렬된 배열: " + Arrays.toString(numbers)); // [10, 20, 50, 60, 80]

// 특정 값 검색 (정렬된 배열에서만 작동)
int index = Arrays.binarySearch(numbers, 50);
System.out.println("50의 인덱스: " + index); // 2

// ---

// 3. ArrayList (자료구조 - List 구현체)
System.out.println("\n--- 3. ArrayList (자료구조 - List) ---");
List<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Cherry");
System.out.println("원래 리스트: " + fruits);

// 4. Collections 클래스: 컬렉션 유틸리티 (List 정렬)
System.out.println("--- 4. Collections (컬렉션 유틸리티) ---");
Collections.reverse(fruits);
System.out.println("역순 리스트: " + fruits); // [Cherry, Banana, Apple]

// 5. HashMap (자료구조 - Map 구현체)
System.out.println("\n--- 5. HashMap (자료구조 - Map) ---");
Map<String, Integer> scores = new HashMap<>();
scores.put("Alice", 95);
scores.put("Bob", 88);
System.out.println("Bob의 점수: " + scores.get("Bob")); // 88

// 6. ThreadLocalRandom 클래스: 멀티스레드 안전 난수 생성
System.out.println("\n--- 6. ThreadLocalRandom (난수 생성) ---");
// 1부터 100까지의 난수 생성 (경계값 101은 포함 안 함)
int randomNum = ThreadLocalRandom.current().nextInt(1, 101);
System.out.println("생성된 난수 (1~100): " + randomNum); 
```
