# 📘 Formatter

## Summary 📝

> [!summary] Summary 📝
> 
> - `Formatter` 클래스는 출력 형식을 지정할 수 있는 **유틸리티 클래스**
> - `printf`와 유사하게 다양한 변환 문자를 통해 데이터 표현 가능
> - 문자열, 숫자, 불리언, 객체 등 다양한 타입을 지정된 형식으로 출력

---

## 목적 🎯

- 콘솔 및 출력 스트림에서 **형식화된 문자열 출력** 지원
- 다양한 데이터 타입을 **일관된 형식**으로 표현
- 가독성 높은 출력 결과 제공

---

## 핵심 개념 💡

- **Formatter 객체 생성**: `new Formatter(System.out)` → 표준 출력에 연결
- **format() 메서드**: 지정된 포맷 문자열과 인자를 조합해 출력
- **형식 지정자(Format Specifier)**: `%` 기호와 함께 데이터 타입 및 출력 형식 지정

|변환 문자|의미|예시|
|---|---|---|
|`%d`|정수|`10`|
|`%c`|문자|`'A'`|
|`%b`|boolean|`true`|
|`%s`|문자열|`"Hello"`|
|`%f`|실수|`3.14`|
|`%e`|과학적 표기|`1.23e+02`|
|`%x`|16진수|`0x1a`|
|`%h`|해시코드|객체의 해시값|
|` %n");
    }
}
```

### ✅ 특수 문자 활용

```java
public class SpecialChars {
    public static void main(String[] args) {
        System.out.printf("탭\t구분%n");
        System.out.printf("줄바꿈\n다음 줄%n");
        System.out.printf("큰따옴표: \"%s\"%n", "Hello");
        System.out.printf("유니코드: \u2605%n"); // ★ 출력
    }
}
```
