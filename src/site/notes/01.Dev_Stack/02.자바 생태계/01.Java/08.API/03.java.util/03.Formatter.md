# Java Formatter (형식화 출력 유틸리티)
> [!summary] 핵심 요약
> 
> - **개념**: 다양한 데이터 타입(숫자, 문자열 등)을 지정된 형식의 문자열로 변환하고 출력하는 유틸리티 클래스
>     
> - **왜?**: 출력 데이터의 가독성을 높이고, 일관된 정렬 및 소수점 자리수 제어 등을 쉽게 구현하기 위함
>     
> - **언제?**: 콘솔 화면(`System.out`), 텍스트 파일, 네트워크 스트림 등에 정교하게 조립된 텍스트 데이터를 출력해야 할 때
>     
> - **핵심**: `%` 기호를 활용한 형식 지정자(Format Specifier) 숙달 및 데이터 타입과의 정확한 매핑
>     
## Formatter 기본 동작 원리
- **출력 스트림 연결 및 객체 생성**
    - `new Formatter(출력대상)` 형태로 생성하여 목적지(콘솔, 파일 등) 지정
    - `format("포맷문자열", 인자1, 인자2...)` 메서드를 통해 데이터 조립 및 전송
    - **어떻게?**: 포맷 문자열 내부의 `%` 위치에 순서대로 가변 인자가 치환되는 방식
```plaintext
[ Formatter 처리 흐름 (Formatting Flow) ]

  ┌─ 포맷 문자열 (Format String) ──┐
  │ "%s는 %d살이고 %.1f점입니다."  │
  └──────────────┬─────────────────┘
                 │ 매핑 (Mapping)
  ┌──────────────▼───────────────┐          ┌───────────────────┐
  │        java.util.Formatter   │ ───────▶ │ [지정된 출력 대상]│
  └──────────────▲───────────────┘  변환 후 │ (System.out, File)│
                 │                  출력    └───────────────────┘
  ┌──────────────┴───────────────┐
  │ "홍길동", 20, 95.5           │
  └─ 데이터 인자 (Arguments) ────┘
```
```java
// 1. 표준 출력(콘솔)으로 연결된 Formatter
Formatter consoleFormatter = new Formatter(System.out);
consoleFormatter.format("이름: %s, 나이: %d%n", "홍길동", 25);
// 출력: 이름: 홍길동, 나이: 25

// 2. 파일 출력을 위한 Formatter (사용 후 자원 해제 필수)
try (Formatter fileFormatter = new Formatter(new File("output.txt"))) {
	fileFormatter.format("파일에 작성되는 %s 데이터입니다.%n", "형식화된");
} // try-with-resources 구문으로 자동 close() 처리

// 3. 실무 즉시 적용 팁: System.out.printf()는 Formatter를 내부적으로 감싼 래퍼 메서드임
System.out.printf("printf를 사용하면 %s 편리함%n", "더욱");
```
## 핵심 형식 지정자 (Format Specifier)
- **데이터 타입에 따른 변환 문자**
    - `%` 기호 뒤에 데이터의 종류를 나타내는 영문자 표기
    - ==지정자와 실제 데이터 타입이 불일치하면 `IllegalFormatConversionException` 런타임 에러 발생==
    - 객체 전달 시 내부적으로 `toString()`을 자동 호출하여 문자열로 치환
    - **어떻게?**: 출력하려는 값의 특성(정수, 실수, 진법 등)에 맞춰 변환 문자를 선택 적용
```java
// 정수와 실수 (소수점 제어 없이 기본 출력 시 실수는 소수점 6자리까지 표현됨)
System.out.printf("정수: %d, 실수: %f%n", 42, 3.14159);

// 진법 변환 (소문자/대문자 지정자에 따라 출력 형태 변경)
System.out.printf("16진수 소문자: %x, 대문자: %X%n", 255, 255); // ff, FF

// 불리언 및 문자열
System.out.printf("불리언: %b, 문자열: %s%n", true, "Java");

// 퍼센트 기호 자체를 출력하고 싶을 때는 %% 두 번 연속 사용
System.out.printf("진행률: %d%%%n", 100); // 출력: 진행률: 100%
```

|**변환 문자**|**의미 (동작 원리)**|**예시**|
|---|---|---|
|`%d`|10진수 정수 (Decimal)|`10`|
|`%f`|실수 (Floating-point)|`3.140000`|
|`%s`|문자열 (String)|`"Hello"`|
|`%c`|단일 문자 (Character)|`'A'`|
|`%b`|논리값 (Boolean)|`true`|
|`%x` / `%X`|16진수 정수 (Hexadecimal)|`0x1a` / `0x1A`|
|`%%`|리터럴 `%` 기호 자체 출력|`%`|

## 정밀도 및 여백 제어 (Advanced Formatting)
- **출력 형태의 세밀한 조정**
    - `%`와 변환 문자 사이에 숫자를 넣어 **전체 자리수(폭)** 및 **소수점 정밀도** 제어
    - 기본적으로 우측 정렬되며, `-` 기호 추가 시 좌측 정렬 수행
    - 숫자의 경우 빈자리를 `0`으로 채우는 패딩 연산 지원
    - **어떻게?**: 콘솔 기반의 표(Table) 형태 출력이나, 화폐 단위 맞춤 시 적극 활용
```plaintext
[ 형식 지정자 상세 구조 (Format Specifier Structure) ]

  %[플래그][너비][.정밀도]변환문자
  │   │       │      │       │
  │   │       │      │       └─ d, f, s 등 (필수)
  │   │       │      └─────── 소수점 이하 자리수 (예: .2)
  │   │       └───────────── 전체 차지할 칸 수 (예: 10)
  │   └─────────────────── 정렬(-) 또는 빈칸채움(0) 등
  └─────────────────────── 시작 기호 (필수)
```
```java
// 1. 소수점 정밀도 제어 (반올림 처리됨)
System.out.printf("소수점 둘째자리: %.2f%n", 3.14159); // 3.14
System.out.printf("소수점 넷째자리: %.4f%n", 3.14159); // 3.1416

// 2. 너비 지정 및 여백 정렬 (5칸 확보)
System.out.printf("우측 정렬: [%5d]%n", 99);  // [   99]
System.out.printf("좌측 정렬: [%-5d]%n", 99); // [99   ]

// 3. 0 채우기 (Zero Padding)
System.out.printf("0 채우기 : [%05d]%n", 99); // [00099]
```
## 이스케이프 시퀀스와 줄바꿈
- **제어 문자 (Control Character) 활용**
    - 백슬래시(`\`)와 조합하여 눈에 보이지 않는 공백, 탭, 따옴표 등을 표현
    - ==플랫폼 독립적인 줄바꿈을 위해 `\n` 대신 `%n` 사용을 강력 권장== (OS마다 줄바꿈 기호가 다르기 때문)
    - **어떻게?**: 문자열 내에서 레이아웃을 강제로 띄우거나 기호 충돌을 방지할 때 사용
```java
// 탭(Tab)으로 간격 맞추기
System.out.printf("ID\tName\tAge%n");
System.out.printf("01\tTom\t20%n");

// 큰따옴표, 역슬래시 자체를 문자열 안에 포함시키기
System.out.printf("경로: C:\\Workspace\\Java%n");
System.out.printf("명언: \"포기하지 마라\"%n");
```

|**특수 문자 (제어 문자)**|**동작 원리 및 역할**|
|---|---|
|`%n`|현재 OS에 맞는 안전한 줄바꿈 (Newline)|
|`\t`|탭 (Tab) 공백 추가|
|`\\`|역슬래시(`\`) 자체 출력|
|`\"`|큰따옴표(`"`) 자체 출력|
|`\'`|작은따옴표(`'`) 자체 출력|
|`\uXXXX`|유니코드(Unicode) 문자 출력 (예: `\u2605` = ★)|
