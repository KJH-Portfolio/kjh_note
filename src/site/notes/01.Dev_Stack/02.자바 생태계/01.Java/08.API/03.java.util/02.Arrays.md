# java.util.Arrays 핵심 가이드
> [!summary] 핵심 요약
> 
> - **개념**: 자바 배열(`Array`) 조작을 위한 정적(Static) 유틸리티 클래스
>     
> - **왜?**: 정렬, 복사, 탐색 등 반복적이고 구현하기 까다로운 배열 제어 로직을 표준 API로 안전하게 처리하기 위함
>     
> - **언제?**: 크기가 고정된 정적 배열(`int[]`, `String[]` 등)의 데이터를 가공, 검증, 출력해야 할 때
>     
> - **핵심**: 모든 메서드는 인스턴스 생성 없이 `Arrays.메서드명()`으로 호출하며, ==다차원 배열 제어 시 반드시 `deep~` 접두사가 붙은 메서드를 사용==함
>     
## 배열 초기화 및 문자열 변환
- **값 채우기 (`fill`)**
    - 배열의 모든 요소를 지정된 단일 값으로 일괄 초기화
- **문자열 출력 (`toString` / `deepToString`)**
    - 배열 내부의 실제 데이터를 콘솔에 출력하기 위해 문자열(`String`) 형태로 변환
    - 1차원 배열은 `toString()`, **2차원 이상의 다차원 배열은 `deepToString()`** 사용
    - **어떻게?**: 반복문(`for`) 없이 단 한 줄의 코드로 배열 전체 요소를 확인
```plaintext
[ 다차원 배열 출력 원리 ]

matrix (Stack)
  │
  ├──▶ [0] (Heap: 주소 0x100) ──▶ [ 1, 2 ]
  └──▶ [1] (Heap: 주소 0x200) ──▶ [ 3, 4 ]

※ Arrays.toString(matrix)     ──▶ "[0x100, 0x200]" (하위 배열의 메모리 주소값 출력)
※ Arrays.deepToString(matrix) ──▶ "[[1, 2], [3, 4]]" (재귀적으로 실제 값 추적 출력)
```
```java
// 1. fill: 배열 전체 초기화
int[] zeroes = new int[5];
Arrays.fill(zeroes, 7);
System.out.println("fill 적용: " + Arrays.toString(zeroes)); // [7, 7, 7, 7, 7]

// 2. deepToString: 다차원 배열 출력
int[][] matrix = {
	{1, 2}, 
	{3, 4}
};
// System.out.println(Arrays.toString(matrix)); // [[I@15db9742, [I@6d06d69c] (메모리 주소)
System.out.println("2차원 배열: " + Arrays.deepToString(matrix)); // [[1, 2], [3, 4]]
```
## 배열 복사 및 범위 추출
- **전체/부분 복사 (`copyOf`)**
    - 원본 배열의 0번 인덱스부터 지정한 '길이'만큼 복사하여 새로운 배열 반환
    - 지정한 길이가 원본보다 크면, 기본값(`0` 또는 `null`)으로 남은 공간 채움
- **특정 범위 복사 (`copyOfRange`)**
    - 원본 배열의 '시작 인덱스'부터 '종료 인덱스 직전'까지 잘라내어 새로운 배열 반환
    - **어떻게?**: 배열 크기를 늘리거나 일부 데이터만 슬라이싱(Slicing)할 때 활용
```plaintext
[ copyOfRange(arr, 1, 4) 슬라이싱 구조 ]

인덱스 :   0       1       2       3       4
원본   : [10]    [20]    [30]    [40]    [50]
                   ▲                       ▲
              시작(포함)               끝(미포함)
                   │                       │
복사본 :         [20]    [30]    [40] ◀────┘
```
```java
int[] original = {10, 20, 30, 40, 50};

// 1. copyOf: 앞에서부터 3개 요소만 복사
int[] copy1 = Arrays.copyOf(original, 3); // [10, 20, 30]

// 2. copyOf: 원본보다 길게 복사 (빈 자리는 0으로 채워짐)
int[] copy2 = Arrays.copyOf(original, 7); // [10, 20, 30, 40, 50, 0, 0]

// 3. copyOfRange: 1번 인덱스부터 4번 인덱스 '전'까지 복사
int[] rangeCopy = Arrays.copyOfRange(original, 1, 4); // [20, 30, 40]
```
## 배열 정렬 및 이진 탐색
- **배열 정렬 (`sort`)**
    - 내부 요소를 오름차순(숫자는 작은 수부터, 문자는 알파벳/가나다 순)으로 정렬
    - 원본 배열 자체의 순서를 직접 변경(In-place 정렬)
- **이진 탐색 (`binarySearch`)**
    - 배열 내에서 특정 값의 인덱스(위치)를 고속으로 반환
    - ==탐색 전 반드시 `Arrays.sort()`를 통해 배열이 정렬되어 있어야만 정상 동작함==
```plaintext
[ 이진 탐색 (Binary Search) 실행 조건과 흐름 ]

입력 배열: {30, 10, 50, 20, 40}

 1. 정렬 전 (탐색 불가)
    [30] [10] [50] [20] [40] ──▶ binarySearch 실행 시 엉뚱한 값(음수) 반환

 2. Arrays.sort() 실행 (필수 전제 조건)
    [10] [20] [30] [40] [50]
          ▲
 3. Arrays.binarySearch(arr, 20) 실행 ──▶ 인덱스 1 반환 (정확도 100%)
```
```java
int[] nums = {50, 20, 10, 40, 30};

// 1. 이진 탐색의 잘못된 사용 (정렬 전)
int wrongIndex = Arrays.binarySearch(nums, 20); // 예측 불가능한 음수값 출력

// 2. 정렬 수행 (오름차순)
Arrays.sort(nums); // [10, 20, 30, 40, 50]

// 3. 올바른 이진 탐색
int correctIndex = Arrays.binarySearch(nums, 20);  // 1
```
## 배열 동등성 비교
- **배열 비교 (`equals` / `deepEquals`)**
    - 두 배열의 길이와 내부 요소의 값이 모두 동일한지 `boolean` 타입으로 반환
    - 자바의 `==` 연산자는 객체의 메모리 주소만 비교하므로, 내용물 검증 시 반드시 `Arrays` 유틸리티 사용
    - ==다차원 배열의 값 비교는 반드시 `Arrays.deepEquals()`를 사용해야 함==
```java
// 1. 1차원 배열 비교
int[] arr1 = {1, 2, 3};
int[] arr2 = {1, 2, 3};

arr1 == arr2; // false (서로 다른 객체)
Arrays.equals(arr1, arr2); // true

// 2. 다차원 배열 비교
int[][] m1 = {{1, 2}, {3, 4}};
int[][] m2 = {{1, 2}, {3, 4}};

Arrays.equals(m1, m2); // false (내부 1차원 배열의 주소 비교함)
Arrays.deepEquals(m1, m2); // true
```
## Arrays 주요 메서드 레퍼런스

|**메서드 시그니처**|**반환 타입**|**핵심 기능 설명**|
|---|---|---|
|`sort(arr)`|`void`|배열을 오름차순으로 정렬 (원본 변형)|
|`binarySearch(arr, key)`|`int`|요소의 인덱스 반환 (반드시 정렬된 상태에서 사용)|
|`copyOf(arr, length)`|`배열`|배열의 0번 인덱스부터 지정한 길이만큼 복사하여 새 배열 생성|
|`copyOfRange(arr, start, end)`|`배열`|지정한 시작 인덱스부터 끝 인덱스(미포함)까지 복사|
|`toString(arr)`|`String`|1차원 배열 데이터를 `[1, 2, 3]` 형태의 문자열로 반환|
|`deepToString(arr)`|`String`|2차원 이상 다차원 배열의 데이터를 재귀적으로 추적하여 문자열 반환|
|`equals(arr1, arr2)`|`boolean`|두 1차원 배열의 내부 값이 완벽히 일치하는지 비교|
|`deepEquals(arr1, arr2)`|`boolean`|두 다차원 배열의 내부 값이 완벽히 일치하는지 재귀적으로 비교|
|`fill(arr, value)`|`void`|배열의 모든 요소를 전달한 단일 값으로 덮어씀|
