# 📘 Arrays 클래스 🧩

## Summary 📝

> [!summary] Summary 📝
> 
> - `Arrays` 클래스는 배열을 다루기 위한 **유틸리티 메서드 모음**
> - 정렬, 복사, 비교, 출력 등 배열 관련 기능을 간단히 제공
> - `import java.util.Arrays;` 선언 필요

---

## 목적 🎯

- 배열을 효율적으로 다루기 위한 표준 메서드 제공
- 반복적인 배열 처리 로직을 간단히 구현
- 코드 가독성과 유지보수성 향상

---

## 핵심 개념 💡

- **정적 메서드 제공**: 객체 생성 없이 클래스명으로 직접 호출
- **배열 조작 기능**: 복사, 정렬, 비교, 출력 등 지원

|메서드|설명|예시|
|---|---|---|
|`copyOf(arr, len)`|배열 복사|`Arrays.copyOf(arr, 5)`|
|`copyOfRange(arr, s, e)`|범위 복사|`Arrays.copyOfRange(arr, 0, 3)`|
|`toString(arr)`|배열 문자열 출력|`Arrays.toString(new int[]{1,2,3})`|
|`sort(arr)`|배열 정렬|`Arrays.sort(scores)`|
|`equals(a, b)`|배열 비교|`Arrays.equals(arr1, arr2)`|
|`deepToString(arr)`|다차원 배열 문자열 출력|`Arrays.deepToString(matrix)`|
|`deepEquals(a, b)`|다차원 배열 비교|`Arrays.deepEquals(m1, m2)`|
|`fill(arr, val)`|배열 전체 값 채우기|`Arrays.fill(arr, 0)`|
|`binarySearch(arr, key)`|이진 탐색 (정렬 필요)|`Arrays.binarySearch(arr, 3)`|

---

## 특징 ✨

- **정렬 기능 내장**: 오름차순 정렬 기본 제공
- **배열 비교**: `equals()`는 요소 값 비교, 참조 비교 아님
- **출력 편의성**: `toString()`으로 배열 내용을 문자열로 변환
- **복사 기능**: 원하는 길이 또는 범위만큼 손쉽게 복사 가능
- **다차원 배열 지원**: `deepToString()`, `deepEquals()` 제공

---

## 팁과 주의점 ⚠️

- `sort()`는 기본적으로 **오름차순 정렬** 수행
- 다차원 배열 출력 시 `toString()` 대신 `deepToString()` 사용 필요
- `equals()`는 1차원 배열만 값 비교, 다차원 배열은 `deepEquals()` 사용
- `binarySearch()` 사용 전 반드시 배열을 정렬해야 함

---

## 예시 📌

### ✅ 배열 정렬

```java
import java.util.Arrays;

public class ArraysSortExample {
    public static void main(String[] args) {
        int[] nums = {3, 1, 2};
        Arrays.sort(nums); // 오름차순 정렬
        System.out.println(Arrays.toString(nums)); // [1, 2, 3]
    }
}
```

### ✅ 배열 복사

```java
import java.util.Arrays;

public class ArraysCopyExample {
    public static void main(String[] args) {
        int[] original = {1, 2, 3, 4, 5};
        int[] copy = Arrays.copyOf(original, 3); // 앞에서 3개 복사
        System.out.println(Arrays.toString(copy)); // [1, 2, 3]
    }
}
```

### ✅ 배열 비교

```java
import java.util.Arrays;

public class ArraysEqualsExample {
    public static void main(String[] args) {
        int[] arr1 = {1, 2, 3};
        int[] arr2 = {1, 2, 3};
        System.out.println(Arrays.equals(arr1, arr2)); // true
    }
}
```

### ✅ 다차원 배열 출력

```java
import java.util.Arrays;

public class ArraysDeepExample {
    public static void main(String[] args) {
        int[][] matrix = {{1, 2}, {3, 4}};
        System.out.println(Arrays.deepToString(matrix)); // [[1, 2], [3, 4]]
    }
}
```

### ✅ 이진 탐색

```java
import java.util.Arrays;

public class ArraysBinarySearchExample {
    public static void main(String[] args) {
        int[] nums = {1, 3, 5, 7, 9};
        int index = Arrays.binarySearch(nums, 5);
        System.out.println("5의 인덱스: " + index); // 2
    }
}
```
