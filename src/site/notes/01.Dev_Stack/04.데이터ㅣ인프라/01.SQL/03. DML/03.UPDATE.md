# 📘 DML - UPDATE (데이터 수정)

## Summary 📝

> [!summary] 핵심 요약
>
>   * **UPDATE**: 기존 데이터의 내용을 변경.
>   * **SET**: 변경할 값 지정 (`=` 은 대입 연산자).
>   * **WHERE**: 변경할 대상 행 지정 (생략 시 **전체 수정**되므로 주의).
>   * **제약조건**: PK, FK, NOT NULL 등을 위배하는 값으로 수정 불가.

## 1\. 기본 및 서브쿼리 문법 🛠️

```sql
-- 1. 기본 수정
UPDATE 테이블명
   SET 컬럼1 = 값1,
       컬럼2 = 값2
 [WHERE 조건];

-- 2. 서브쿼리 활용 수정
UPDATE 테이블명
   SET 컬럼1 = (SELECT 값 FROM ... WHERE ...)
 [WHERE 조건];
```

## 2\. 주의사항 ⚠️

  * **WHERE 절 누락**: 테이블의 모든 행이 같은 값으로 변경되는 대형 사고 발생 가능.
  * **무결성 제약조건**:
      * `UPDATE EMP SET DEPT_ID = 99 ...` (부모 키에 없는 99번 부서로 수정 시 FK 오류)
      * `UPDATE EMP SET NAME = NULL ...` (NOT NULL 컬럼일 경우 오류)
