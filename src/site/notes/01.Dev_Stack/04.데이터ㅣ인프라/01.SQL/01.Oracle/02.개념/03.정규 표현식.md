# 오라클 정규 표현식 (Oracle Regular Expressions)

> [!summary]
> **REGEXP** 계열 함수를 통한 데이터 검색, 치환, 추출 기법 및 주요 **메타 문자**별 SQL 구현 사례 요약.

## 정규 표현식 핵심 함수

* **REGEXP_LIKE** : 패턴 일치 기반 데이터 필터링 및 `WHERE` 조건절 **적용**.
* **REGEXP_REPLACE** : 매칭 패턴 탐색 및 지정 문자열 **치환**.
* **REGEXP_SUBSTR** : 패턴 일치 부분 문자열 분리 및 **추출**.
* **REGEXP_INSTR** : 패턴 시작 위치 인덱스 **반환**.
* **REGEXP_COUNT** : 특정 패턴 발생 빈도 **산출**.

```sql
-- 'A' 또는 'S'로 시작하는 직원명 조회
SELECT ename FROM emp WHERE REGEXP_LIKE(ename, '^[AS]');

-- 전화번호 내 하이픈(-) 제거 및 공백 치환
SELECT REGEXP_REPLACE('010-1234-5678', '-', ' ') FROM dual;

-- 쉼표로 구분된 문자열 중 첫 번째 요소 추출
SELECT REGEXP_SUBSTR('Apple,Orange,Banana', '[^,]+', 1, 1) FROM dual;

-- 문자열 내 숫자 개수 계산
SELECT REGEXP_COUNT('SQL2024 Oracle21c', '\d') FROM dual;

```

## 메타 문자별 매칭 사례

* **임의 문자(.)** : 특정 위치의 모든 단일 문자 **일치**. (예: `a.c` -> abc, adc 등)
* **위치 고정(^, $)** : 시작 및 종료 시점 **명시**. (예: `^Apple`, `End$\`)
* **반복 지정(*, +, ?)** : 패턴 출현 횟수 **제어**. (0회 이상, 1회 이상, 0 또는 1회)
* **문자 집합([], [^])** : 대괄호 내 포함 문자 선택 또는 제외 문자 **지정**. (예: `[abc]`, `[^0-9]`)
* **수량 제한({n})** : 정확한 반복 횟수 **고정**. (예: `\d{3}` -> 숫자 3자리)
* **숫자 기호(\d)** : 단일 숫자 패턴 **표현**. (예: `\d+` -> 연속된 숫자)
* **선택 연산자(|)** : 여러 패턴 중 하나와 일치하는 **OR 연산**.

```sql
-- 세 번째 글자가 'A'인 4글자 문자열 검색
SELECT * FROM tab1 WHERE REGEXP_LIKE(col1, '^..A.
);

-- 숫자로만 구성된 데이터 필터링
SELECT * FROM tab1 WHERE REGEXP_LIKE(col1, '^[0-9]+
);

-- 연속된 숫자 2~3자리 포함 여부 확인
SELECT * FROM tab1 WHERE REGEXP_LIKE(col1, '\d{2,3}');

```
