## ✅ Oracle SYSTEM 계정 접속 방법 정리

#### 1. SQL Plus 실행

- 명령 프롬프트(cmd) 열기 → `sqlplus` 입력(시스템 Id/Pwd 찾기)
```BASH
sqlplus 시스템 계정명/비밀번호@서비스명
```
- **시스템 계정명**
- **비밀번호** → 설치 시 설정한 SYSTEM 계정 비밀번호
- **서비스명** → DB 서비스 이름 (예: `XE`, `ORCL`)
## ✅공유시 설정 파일 위치

```
C:\Users\<사용자명>\AppData\Roaming\SQL Developer
```
- **환경설정 (Preferences)**
- **UI 설정 (테마, 폰트, 색상 등)**
- **최근 열었던 파일 목록**
- **접속 정보 (connections.xml)**
- 비밀번호는 공유 되지 않기에 로그인 해줘야함
### ✅ 프로그램 환경 설정
- 도구 - 환경설정 - 인코딩 설정 : UTF-8
- 코드 편집기 - 행 여백 : 행 번호 표시 세팅
- 코드편집기 - 표시 : 오른쪽 여백 표시 해제

### ✅프로그램 개요
- 접속에 + 버튼으로 접속 계정 만들기
	- name은 그냥 이름
	- 색을 변경하면 구분하기 좋음
	- 사용자명 , 비밀번호력
	- 비밀번호 저장 세팅
	- 테스트 눌러서 성공 확인
	- 접속
- 상단 탭 sql워크시트로 선택 가능
### ✅ 단축키
ctrl + enter : 코드 한줄 실행
ctrl + x : 한줄 지우기(잘라내기)
ctrl + d : 전체 지우기
