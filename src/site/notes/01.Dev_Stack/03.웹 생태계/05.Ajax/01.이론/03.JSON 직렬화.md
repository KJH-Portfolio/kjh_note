| **코드 / 개념**                   | **변환 방향**                             | **역할**                       |
| ----------------------------- | ------------------------------------- | ---------------------------- |
| **`Gson().toJson(객체)`**       | 객체 $\rightarrow$ JSONString           | **✅ 직렬화 (Serialization)**    |
| **`JSON.stringify(객체)`**      | 객체 $\rightarrow$ JSONString           | **✅ 직렬화 (Serialization)**    |
| **`JSON.parse(문자열)`**         | JSONString $\rightarrow$ 객체           | **✅ 역직렬화 (Deserialization)** |
| `new JSONParser().parse(문자열)` | JSONString $\rightarrow$ `JSONObject` | **✅ 역직렬화 (Deserialization)** |
- 객체,JSON,JSON문자열을 변환하는 방법
