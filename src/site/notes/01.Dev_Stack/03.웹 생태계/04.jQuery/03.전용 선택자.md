### jq 전용 선택자


| 카테고리 | 선택자 (Code) | 설명 |
|---|---|---|
| 위치/순서 | $('li:first') | 선택된 요소 집합 중 첫 번째 `<li>` 요소를 선택 |
|  | $('li:last') | 선택된 요소 집합 중 마지막 `<li>` 요소를 선택 |
|  | $('tr:even') | 선택된 행 `<tr>` 집합 중 짝수 번째(인덱스 0, 2, 4...) 행들을 선택 |
|  | $('tr:odd') | 선택된 행 `<tr>` 집합 중 홀수 번째(인덱스 1, 3, 5...) 행들을 선택 |
|  | $('p:eq(2)') | 선택된 단락 `<p>` 집합 중 세 번째 (인덱스 2) 요소를 선택 |
|  | $('li:gt(3)') | 선택된 `<li>` 집합 중 네 번째보다 큰 인덱스(5번째 요소부터)의 요소들을 선택 |
|  | $('li:lt(4)') | 선택된 `<li>` 집합 중 네 번째보다 작은 인덱스(0~3번째 요소)의 요소들을 선택 |
| 콘텐츠/상태 | $(':header') | 모든 헤딩 요소(`<h1>`부터 `<h6>`까지)를 선택 |
|  | $('div:animated') | 현재 애니메이션이 실행 중인 `<div>` 요소를 선택 |
|  | $('p:contains("Hello")') | 텍스트 내용에 "Hello"가 포함된 `<p>` 요소를 선택 |
|  | $('div:parent') | 자식 요소나 텍스트 노드를 가지고 있는 `<div>` 요소를 선택 |
|  | $('a:hidden') | `display: none`이거나 `type="hidden"` 등 보이지 않는 `<a>` 요소를 선택 |
|  | $('img:visible') | 화면에 보이는 `<img>` 요소를 선택 |
| 폼 요소 | $(':input') | 모든 폼 요소(`<input>`, `<textarea>`, `<select>`, `<button>`)를 선택 |
|  | $(':text') | `type="text"`인 `<input>` 요소를 선택 |
|  | $(':password') | `type="password"`인 `<input>` 요소를 선택 |
|  | $(':file') | `type="file"`인 `<input>` 요소를 선택 |
|  | $(':button') | `type="button"`인 `<input>` 및 `<button>` 요소를 선택 |
|  | $(':radio') | `type="radio"`인 `<input>` 요소를 선택 |
|  | $(':checkbox') | `type="checkbox"`인 `<input>` 요소를 선택 |
|  | $(':submit') | `type="submit"`인 `<input>` 요소를 선택 |
|  | $(':reset') | `type="reset"`인 `<input>` 요소를 선택 |
|  | $('option:selected') | 선택된 `<option>` 요소를 선택 (드롭다운 목록) |
