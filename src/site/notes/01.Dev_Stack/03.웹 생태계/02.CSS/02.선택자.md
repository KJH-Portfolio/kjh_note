# 📘 CSS 선택자 및 가상 클래스/요소

## Summary 📝

> [!summary] Summary 📝
> 
> - CSS 선택자는 **기본**, **조합**, **가상 클래스**, **가상 요소**로 나뉘며 정밀한 스타일링을 지원합니다.
>     
> - **최신 기능**: `:has()`(부모 선택자), `:is()`/`:where()`(그룹화), `:root`(변수) 등 모던 CSS 기능 포함.
>     
> - **주의**: jQuery의 `:first`와 CSS의 `:first-child` 구분에 유의해야 합니다.
>     

---

## 1. 기본 선택자 및 조합 선택자

HTML 요소를 직접적으로 선택하거나 요소 간의 관계를 통해 선택합니다.

### 기본 및 조합 선택자 목록

| **구분**    | **선택자**  | **설명**                 | **예시**                             |
| --------- | -------- | ---------------------- | ---------------------------------- |
| **전체**    | `*`      | 모든 HTML 요소 선택          | `* { margin: 0; }`                 |
| **태그**    | `tag`    | 특정 태그 선택 (`h1`, `p` 등) | `p { font-size: 16px; }`           |
| **ID**    | `#id`    | 고유 ID 요소 선택            | `#header { background: #eee; }`    |
| **클래스**   | `.class` | 클래스 속성 요소 선택           | `.active { color: blue; }`         |
| **후손**    | `A B`    | A 하위의 **모든** B 선택 (공백) | `nav a { color: #333; }`           |
| **자식**    | `A > B`  | A의 **직계 자식** B 선택      | `ul > li { list-style: none; }`    |
| **인접 형제** | `A + B`  | A **바로 다음** 형제 B 하나    | `h1 + p { margin-top: 20px; }`     |
| **일반 형제** | `A ~ B`  | A **뒤에 오는 모든** 형제 B    | `h1 ~ p { color: gray; }`          |
| **속성**    | `[attr]` | 특정 속성을 가진 요소 (아래 상세)   | `input[required] { border: red; }` |

### 속성 선택자 상세 (Attribute)

|**구분**|**선택자**|**설명**|**예시**|
|---|---|---|---|
|**일치**|`[attr=val]`|속성 값이 **정확히 일치**|`[type="text"]`|
|**단어**|`[attr~=val]`|공백으로 구분된 **단어 포함**|`[class~="btn"]`|
|**시작**|`[attr^=val]`|값으로 **시작**하는 요소|`[href^="https"]` (보안 링크)|
|**끝**|`[attr$=val]`|값으로 **끝나는** 요소|`[src$=".png"]` (이미지 파일)|
|**포함**|`[attr*=val]`|값의 **일부라도 포함**|`[href*="google"]`|

## 2. 가상 클래스 (Pseudo-classes)

요소의 **상태(State)**나 **구조(Structure)**를 기반으로 선택합니다.

> [!TIP] **[비서 Note] 'First' 선택자 헷갈리지 않기** 🚨
> 
> - **CSS에는 단독 `:first`가 없습니다!**
>     
> - `p:first-child`: 형제들 중 **첫 번째 자식**이 `p`여야 함. (구조 중심)
>     
> - `p:first-of-type`: 형제들 중 `p` 태그끼리 모았을 때 **첫 번째**. (태그 중심)
>     
> - _참고: jQuery의 `$(':first')`와 혼동하지 마세요._
>     

### 구조 및 상태 선택자 목록

| **구분**         | **선택자**               | **설명**                          | **예시/팁**                             |
| -------------- | --------------------- | ------------------------------- | ------------------------------------ |
| **구조 (First)** | `:first-child`        | 부모의 **가장 첫 번째** 자식              | `li:first-child`                     |
|                | `:first-of-type`      | 해당 **태그 타입 중 첫 번째**             | `p:first-of-type` (다른 태그가 앞에 있어도 OK) |
| **구조 (Nth)**   | `:last-child`         | 부모의 마지막 자식                      | `li:last-child`                      |
|                | `:nth-child(n)`       | **n번째** 자식 (수식 가능: `2n`, `odd`) | `tr:nth-child(odd)` (홀수 행 스타일)       |
|                | `:nth-of-type(n)`     | 해당 태그 중 **n번째**                 | `p:nth-of-type(2)`                   |
| **반응/상태**      | `:hover`              | 마우스가 올라갔을 때                     | `btn:hover`                          |
|                | `:active`             | 클릭하고 있는 순간                      | `btn:active`                         |
|                | `:focus`              | 포커스(입력 상태)가 됐을 때                | `input:focus`                        |
|                | `:checked`            | 체크박스/라디오가 체크됨                   | `input:checked`                      |
| **폼(Form)**    | `:disabled`           | 비활성화된 요소                        | `input:disabled`                     |
|                | `:valid` / `:invalid` | 입력값이 유효함 / 유효하지 않음              | 이메일 형식 검사 시 유용                       |
| **부정/기타**      | `:not(S)`             | S가 **아닌** 요소                    | `li:not(.active)`                    |
|                | `:root`               | **[추천] 전역 변수 선언** (html보다 상위)   | `:root { --main-color: blue; }`      |
| **최신(Modern)** | `:has(S)`             | **[강력] 부모 선택자** (S를 가진 요소)      | `div:has(img)` (이미지가 있는 div 선택)      |
|                | `:is(S1, S2)`         | 선택자 그룹화 (간소화)                   | `:is(h1, h2, h3)`                    |

## 3. 가상 요소 (Pseudo-elements)

요소의 **특정 부분**을 스타일링하거나, 가상의 콘텐츠를 생성합니다. (`::` 사용 권장)

|**구분**|**가상 요소**|**설명**|**실무 활용 예시**|
|---|---|---|---|
|**텍스트**|`::first-letter`|블록의 **첫 글자**만 선택|잡지 기사 첫 글자 강조 효과|
||`::first-line`|블록의 **첫 번째 줄**만 선택|문단 첫 줄 스타일링|
||`::selection`|마우스로 **드래그한 영역**|`::selection { bg: yellow; }`|
|**콘텐츠 생성**|`::before`|요소 내용의 **맨 앞**에 가상 콘텐츠 삽입|메뉴 아이콘, 불릿 기호 추가|
||`::after`|요소 내용의 **맨 뒤**에 가상 콘텐츠 삽입|`clearfix`, 말풍선 꼬리 구현|
|**폼(Form)**|`::placeholder`|input의 **안내 문구** 스타일링|흐릿한 회색 글자색 변경 등|
||`::file-selector-button`|**파일 선택 버튼** 스타일링|`<input type="file">` 버튼 디자인|
|**기타**|`::marker`|리스트(`li`)의 **불릿/번호** 스타일링|`li::marker { color: red; }`|

## 4. CSS 우선순위 (Specificity)

스타일이 겹칠 때 어떤 스타일을 적용할지 결정하는 점수입니다.

1. `!important` (최우선, 남용 금지)
    
2. **인라인 스타일** (`style="..."`) : 1000점
    
3. **ID 선택자** (`#id`) : 100점
    
4. **클래스/가상클래스/속성** (`.class`, `:hover`, `[type]`) : 10점
    
5. **태그/가상요소** (`div`, `::before`) : 1점
    
6. 전체 선택자 (`*`) : 0점
    

> [!example] **우선순위 계산 예시**
> 
> - `div p` (1 + 1 = 2점)
>     
> - `.active p` (10 + 1 = 11점) -> **승리**
>     
> - `#nav .active` (100 + 10 = 110점) -> **승리**
