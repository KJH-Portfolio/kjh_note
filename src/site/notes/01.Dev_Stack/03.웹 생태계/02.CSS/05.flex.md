# 📌 Flexbox 정리 노트
#### 핵심

| 효과       | 속성                | 주요 값                                      |
| -------- | ----------------- | ----------------------------------------- |
| 배치 방향    | `flex-direction`  | `row / column / row-reverse`              |
| 가로 정렬    | `justify-content` | `start / center / end / between / around` |
| 세로 정렬    | `align-items`     | `start / center / end / stretch`          |
| 줄바꿈 여부   | `flex-wrap`       | `nowrap / wrap`                           |
| 아이템 간격   | `gap`             | `px 값`                                    |
| 공간 비율 분배 | `flex`            | `숫자`                                      |
| 개별 세로 정렬 | `align-self`      | `start / center / end`                    |
| 순서 변경    | `order`           | `숫자`                                      |
## 1. flex-direction (방향)

- `row` : 기본값, 가로로 배치 → 
- `column` : 세로로 배치 ↓
- `row-reverse` : 가로 역순 <-

```css
.container {
  display: flex;
  flex-direction: row;      /* 기본 */
  flex-direction: column;   /* 세로 */
  flex-direction: row-reverse; /* 역순 */
}
```

---

## 2. justify-content (가로 정렬)

- `flex-start` : 왼쪽 정렬
- `center` : 가운데 정렬
- `flex-end` : 오른쪽 정렬
- `space-between` : 양 끝 붙이고, 사이 균등
- `space-around` : 양쪽 포함 균등 간격

```css
.container {
  display: flex;
  justify-content: flex-start;   /* 왼쪽 */
  justify-content: center;       /* 가운데 */
  justify-content: flex-end;     /* 오른쪽 */
  justify-content: space-between;/* 양끝 + 균등 */
  justify-content: space-around; /* 양쪽 포함 균등 */
}
```

---

## 3. align-items (세로 정렬)

- `flex-start` : 위쪽
- `center` : 가운데
- `flex-end` : 아래쪽
- `stretch` : 높이를 컨테이너에 맞춤

```css
.container {
  display: flex;
  align-items: flex-start; /* 위 */
  align-items: center;     /* 가운데 */
  align-items: flex-end;   /* 아래 */
  align-items: stretch;    /* 늘림 */
}
```

---

## 4. flex-wrap (줄바꿈)

- `nowrap` : 한 줄에 다 넣음 (넘치면 줄어듦)
- `wrap` : 공간 부족하면 줄바꿈

```css
.container {
  display: flex;
  flex-wrap: nowrap; /* 기본 */
  flex-wrap: wrap;   /* 줄바꿈 */
}
```

---

## 5. gap (간격)

- 아이템 사이 간격을 간단히 지정 가능

```css
.container {
  display: flex;
  gap: 20px; /* 아이템 사이 20px */
}
```

---

## 6. flex (공간 분배)

- `flex:1` → 남은 공간을 균등 분배
- `flex:2` → 2배 비율로 차지

```css
.item1 { flex: 1; }
.item2 { flex: 2; }
.item3 { flex: 1; }
```

👉 결과: 1 : 2 : 1 비율로 공간 차지

---

## 7. align-self (개별 정렬)

- 부모의 `align-items` 무시하고 개별 적용

```css
.item1 { align-self: flex-start; }
.item2 { align-self: center; }
.item3 { align-self: flex-end; }
```

---

## 8. order (순서 변경)

- HTML 작성 순서와 상관없이 화면 표시 순서 변경

```css
.item1 { order: 2; } /* 두 번째 */
.item2 { order: 3; } /* 세 번째 */
.item3 { order: 1; } /* 첫 번째 */
```

---

## 9. 중앙 정렬 (가로+세로)

```css
.container {
  display: flex;
  justify-content: center; /* 가로 중앙 */
  align-items: center;     /* 세로 중앙 */
}
```

---

## 10. 로고-메뉴 배치 (space-between)

```css
.container {
  display: flex;
  justify-content: space-between; /* 양 끝 배치 */
  align-items: center;            /* 세로 중앙 */
}
```
