# 📘 순수 자바스크립트 모달(Modal) 구현하기

> [!summary] **핵심 요약**
> * **원리**: 기본적으로 HTML 요소를 숨겨두었다가(`display: none`), 특정 사건(클릭 등)이 발생하면 자바스크립트로 보이게(`display: block` 또는 `flex`) 만드는 것입니다.
> * **구성 요소**:
> 1. **Overlay(배경)**: 뒷배경을 어둡게 만들어 모달에 집중하게 함.
> 2. **Modal Window(창)**: 실제 내용이 들어가는 박스.
> 3. **Trigger(트리거)**: 모달을 열고 닫는 버튼.
> 
> 
> 
>

## 1. HTML 구조 (정적 설계)

모달은 페이지 어디에서든 띄워야 하므로 보통 `<body>` 태그의 직계 자식으로 배치하는 것이 가장 좋습니다.

```html
<button id="openModalBtn">상세 정보 보기</button>

<div id="modalContainer" class="modal-overlay">
  <div class="modal-window">
    <div class="modal-header">
      <h2>알림</h2>
      <span id="closeModalBtn" class="close-icon">&times;</span>
    </div>
    <div class="modal-content">
      <p>이것은 부트스트랩 없이 만든 순수 JS 모달입니다.</p>
    </div>
  </div>
</div>

```

---

## 2. CSS 스타일링 (레이아웃 및 숨김 처리)
- $ z-index/background-color로 배경처리
가장 중요한 것은 처음에는 모달이 보이지 않아야 한다는 점입니다.

```css
/* 1. 배경 레이어: 화면 전체를 덮고 반투명하게 설정 */
.modal-overlay {
  display: none; /* 처음에 숨김 */
  position: fixed;
  top: 0; left: 0;
  width: 100%; height: 100%;
  background-color: rgba(0, 0, 0, 0.5); /* 어두운 반투명 배경 */
  z-index: 999; /* 다른 요소보다 위에 표시 */
  justify-content: center; /* 가로 중앙 정렬 (flex 사용 시) */
  align-items: center;    /* 세로 중앙 정렬 (flex 사용 시) */
}

/* 2. 모달 창: 실제 흰색 박스 */
.modal-window {
  background-color: white;
  padding: 20px;
  border-radius: 8px;
  width: 400px;
  box-shadow: 0 5px 15px rgba(0,0,0,0.3);
}

/* 3. 닫기 버튼 스타일 */
.close-icon {
  float: right;
  cursor: pointer;
  font-size: 24px;
}

```

## 3. JavaScript 로직 (제어)

아까 배운 **콜백 함수**와 **이벤트 리스너**를 사용하여 동작을 부여합니다.

```javascript
const openBtn = document.querySelector("#openModalBtn");
const closeBtn = document.querySelector("#closeModalBtn");
const modal = document.querySelector("#modalContainer");

// 1. 모달 열기 콜백
openBtn.addEventListener("click", () => {
  modal.style.display = "flex"; // flex로 설정하여 중앙 정렬 활성화
});

// 2. 모달 닫기 콜백
const closeModal = () => {
  modal.style.display = "none";
};

closeBtn.addEventListener("click", closeModal);

// 3. [보너스] 배경 클릭 시 닫기 기능
window.addEventListener("click", (event) => {
  if (event.target === modal) {
    closeModal();
  }
});

```

## 4. 부트스트랩 방식과의 차이점 비교

| 구분 | 부트스트랩 모달 | 순수 JS 모달 |
| --- | --- | --- |
| **의존성** | 부트스트랩 CSS/JS 파일 필수 | 아무것도 필요 없음 (가벼움) |
| **제어 방식** | `data-bs-*` 속성으로 자동 제어 | JS로 직접 `style`을 조절해야 함 |
| **커스터마이징** | 정해진 디자인을 수정하기 번거로움 | 디자인과 애니메이션이 100% 자유로움 |
| **학습 난이도** | 사용법 숙지가 필요함 | 기본 JS/CSS 지식만 있으면 됨 |
