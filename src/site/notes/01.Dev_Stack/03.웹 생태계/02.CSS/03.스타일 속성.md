# 📘 CSS 스타일 속성

> [!summary] Summary 📝
> * **필수 레이아웃 (90% 사용)**:
> `display: flex` (가로/세로 정렬, `justify-content`, `align-items`), `gap` (요소 간격).
> * **공간 및 크기**:
> `width` / `height`, `margin` (바깥 여백), `padding` (안쪽 여백), **`box-sizing: border-box`** (크기 계산 고정).
> * **위치 및 겹침**:
> `position: absolute` (자유 배치), **`relative` (본래 위치 기준 이동 및 자식의 기준점)**, `fixed` (화면 고정), `z-index` (앞뒤 순서).
> * **UI 디자인**:
> `border-radius` (둥근 모서리), `box-shadow` (그림자/입체감), `cursor: pointer` (버튼 클릭 표시), `color` / `font-size`.
> 
> 

---

## 핵심 개념 💡

* **Typography**: 가독성을 위한 폰트 설정, 줄바꿈 및 말줄임 처리.
* **Layout**: `float`를 대체하는 **Flexbox**와 **Grid** 시스템.
* **Box Model**: `border-box`를 기준으로 한 직관적인 크기 계산.
* **Positioning**: 요소의 흐름을 깨지 않고 배치하는 **`relative`**와 이를 기준으로 자유롭게 움직이는 `absolute`의 관계 이해.
* **Interaction**: 마우스 커서, 클릭 방지 등 사용자 경험 제어.
* **Visuals**: 이미지 비율 유지(`object-fit`)와 입체감(`box-shadow`).

---

## 팁과 주의점 💡

* **초기화 필수**: 모든 프로젝트 시작 시 `* { box-sizing: border-box; }` 설정 권장.
* **레이아웃 주의**: 단순 배치는 `Flex`, 2차원 격자는 `Grid` 사용. (`float`는 텍스트 흐름 제어 외 사용 지양)
* **기준점 설정**: 자식에게 `absolute`를 부여할 계획이라면, 반드시 부모에게 **`relative`**를 설정하여 좌표의 기준을 잡아야 함.
* **공간 유지**: **`relative`**로 요소를 이동시켜도 원래 해당 요소가 차지하던 물리적 공간은 그대로 유지됨(흐름을 깨지 않음).
* **이미지 처리**: 배경 이미지는 `background-size: cover`, 태그 이미지는 `object-fit: cover`로 비율 유지.
* **단위 사용**: 반응형을 고려해 고정 단위(`px`)보다 상대 단위(`rem`, `%`, `vw/vh`) 활용 권장.

## 예시 📌

### 1. 폰트 & 텍스트 (Typography & Text)

| **카테고리** | **속성** | **설명** | **실무 주요 값/예시** |
| --- | --- | --- | --- |
| **글꼴** | `font-family` | 글꼴 지정 (시스템 폰트 스택 선호). | `'Pretendard'`, `-apple-system` |
|  | `font-size` | 글꼴 크기 (`rem` 권장). | **`1rem`**(16px), `0.875rem` |
|  | `font-weight` | 굵기. | `400`, `500`, `700` |
|  | `line-height` | 줄 간격 (단위 없음 권장). | **`1.5`** |
| **텍스트** | `color` | 글자 색상. | `#333`, `rgba(0,0,0,0.6)` |
|  | `text-align` | 정렬. | `left`, `center` |
|  | `text-decoration` | 장식 (링크 밑줄 제거). | **`none`**, `underline` |
| **줄바꿈** | `white-space` | 줄바꿈 제어. | **`nowrap`**(줄바꿈 금지) |
|  | `text-overflow` | 넘침 시 '...' 처리. | **`ellipsis`** |

### 2. 영역 & 레이아웃 (Box Model & Layout)

| **카테고리** | **속성** | **설명** | **실무 주요 값/예시** |
| --- | --- | --- | --- |
| **크기** | `width`/`height` | 너비/높이. | `100%`, `100vh`, `auto` |
|  | `max-width` | 최대 너비 (반응형). | `1200px` |
| **여백** | `margin` | 바깥 여백. | `0 auto`, `1rem` |
|  | `padding` | 안쪽 여백. | `1rem 2rem` |
| **계산** | **`box-sizing`** | **박스 크기 계산(필수).** | **`border-box`** |
| **Flex** | **`display: flex`** | 1차원 레이아웃. | **`flex`** |
|  | `flex-direction` | 방향. | `row`, **`column`** |
|  | `justify-content` | 주 축 정렬. | `center`, `space-between` |
|  | `align-items` | 교차 축 정렬. | `center`, `flex-start` |
|  | **`gap`** | **아이템 간 간격.** | **`10px`**, `1rem` |
| **Grid** | **`display: grid`** | 2차원 레이아웃. | **`grid`** |
|  | `grid-template-columns` | 열 너비 정의. | `1fr 1fr`, `repeat(3, 1fr)` |

### 3. 시각 효과 & 인터랙션 (Visuals & Interaction)

| **카테고리** | **속성** | **설명** | **실무 주요 값/예시** |
| --- | --- | --- | --- |
| **이미지** | **`object-fit`** | `img` 태그 비율 유지. | **`cover`**, `contain` |
| **배경** | `background-size` | 배경 이미지 크기. | **`cover`** |
| **그림자** | `box-shadow` | 입체감 부여. | `0 4px 6px rgba(0,0,0,0.1)` |
| **배치** | **`position`** | **위치 기준.** | **`relative`(기준점/자기기준)**, `absolute`, `fixed`, `sticky` |
|  | `z-index` | 쌓임 순서. | `10`, `100` |
| **마우스** | `cursor` | 커서 모양. | **`pointer`**, `not-allowed` |
| **UI** | `appearance` | 기본 스타일 제거. | **`none`** |
