## `float` 속성

### 적용 위치

- 주로 **박스(block 요소)에 적용 (`div`, `li`, `img` 등)
- 개별 요소에 적용해서 그 요소를 왼쪽/오른쪽으로 띄움

### 영향 범위

- 해당 요소 **자신의 위치**를 바꾸고
- **뒤따르는 형제 요소들의 흐름**에 영향을 줌
- 부모는 float된 자식의 높이를 인식하지 못함 → `clearfix` 필요

### 주요 값

- `float: left;` → 왼쪽으로 띄움
- `float: right;` → 오른쪽으로 띄움
- `float: none;` → 기본 흐름으로 되돌림

👉 **예시**

```css
img {
  float: left; /* 이미지 왼쪽에 붙고 텍스트가 오른쪽으로 흘러 들어감 */
}
```

---

## ✅ 차이점 요약

|속성|적용 대상|영향 범위|주 용도|
|---|---|---|---|
|**display**|모든 요소|자기 자신 + 자식 배치|레이아웃 방식 결정 (block, inline, flex, grid 등)|
|**float**|주로 block 요소|자기 위치 + 형제 요소 흐름|이미지 옆 텍스트, 단순 가로 배치|

---

## ✨ 시각적 이해

- **display:flex** → 부모에 적용 → 자식들이 가로/세로로 정렬됨
- **float:left** → 개별 요소에 적용 → 그 요소만 왼쪽으로 띄워지고, 뒤따르는 형제들이 옆으로 흘러 들어감

---

## 🚀 결론

- **display는 부모 컨테이너 중심**: 자식들의 배치 방식을 결정
- **float는 개별 요소 중심**: 그 요소를 띄워서 형제들의 흐름을 바꿈
- 요즘 레이아웃은 `float` 대신 `flex`나 `grid`를 쓰는 게 훨씬 직관적이고 유지보수에 유리합니다.
