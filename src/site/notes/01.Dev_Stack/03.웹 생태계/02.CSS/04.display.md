# Css display 속성

## None

- **특징:** 요소를 레이아웃에서 완전히 제거. 공간도 차지하지 않음. 접근성 상 숨김이 필요하면 대체 수단 고려.
- **언제 사용:** 조건부 표시/숨김.

```html
<style>
.hidden { display: none; }
</style>
<div>이건 보임</div>
<div class="hidden">이건 완전히 사라져요</div>
```

---

## Inline

- **특징:** 줄 흐름 내에 배치. 콘텐츠 크기에 맞춰짐. `width/height` 직접 지정 불가. 좌우 `padding/margin`은 적용되지만 줄바꿈을 만들지 않음.
- **대표 요소:** `span`, `a`, `em`, `strong`.
- **언제 사용:** 문장 중 일부 스타일링, 링크 등.

```html
<style>
.tag { display: inline; background: #ffd54f; padding: 0 6px; }
</style>
문장 중 <span class="tag">inline 스팬</span>은 줄을 차지하지 않아요.
```

---

## Block

- **특징:** 줄 전체를 차지하고 새 줄에서 시작. `width/height`, `margin/padding` 모두 정상 적용.
- **대표 요소:** `div`, `h1`–`h6`, `p`, `section`.
- **언제 사용:** 문단, 섹션처럼 가로 폭을 채우는 레이아웃.

```html
<style>
.box { display: block; width: 300px; background: #eee; }
</style>
<div class="box">나는 block 요소라 줄을 단독으로 차지해요.</div>
<p class="box">또 다른 block 요소.</p>
```

---

## Inline-block

- **특징:** 줄 흐름 내 배치(인라인처럼) + `width/height` 지정 가능(블록처럼). 요소 사이 공백 문자에 따른 간격이 생길 수 있음.
- **언제 사용:** 버튼, 배지처럼 가로로 나열하면서 크기 제어가 필요한 경우.

```html
<style>
.btn { display: inline-block; padding: 8px 12px; background: #90caf9; border-radius: 6px; }
</style>
<a class="btn">버튼1</a><a class="btn">버튼2</a>
```

---

## Flex

- **특징:** 1차원 레이아웃(가로 또는 세로 한 축 기준). 자식이 “flex 아이템”이 되어 정렬과 공간 분배가 쉬움.
- **핵심 속성:** `flex-direction`, `gap`, `justify-content`, `align-items`, 아이템용 `flex`(grow/shrink/basis).
- **언제 사용:** 가로/세로로 깔끔하게 정렬, 가운데 정렬, 간격 분배.

```html
<style>
.container { display: flex; gap: 8px; justify-content: space-between; align-items: center; }
.item { background: #c5e1a5; padding: 10px; }
</style>
<div class="container">
  <div class="item">좌</div>
  <div class="item">중앙</div>
  <div class="item">우</div>
</div>
```

---

## Inline-flex

- **특징:** 컨테이너 자신은 인라인처럼 흐르고, 내부는 flex 레이아웃. 주변 텍스트와 같은 줄에 배치 가능.
- **언제 사용:** 텍스트 흐름 속에서 작은 그룹을 flex로 정렬할 때.

```html
<style>
.inlineGroup { display: inline-flex; gap: 6px; align-items: center; background: #ffe082; padding: 4px; }
.badge { background: #fff176; padding: 2px 6px; border-radius: 4px; }
</style>
문장 속 <span class="inlineGroup">
  <span class="badge">A</span><span class="badge">B</span><span class="badge">C</span>
</span> 그룹이 인라인으로 흐릅니다.
```

## Table

- **특징:** CSS 테이블 박스 모델 사용. `display: table`, `table-row`, `table-cell` 등과 조합.
- **언제 사용:** 실제 `<table>` 없이 테이블 같은 레이아웃을 만들 때.

```html
<style>
.table { display: table; width: 100%; border: 1px solid #ccc; }
.row   { display: table-row; }
.cell  { display: table-cell; padding: 8px; border: 1px solid #ddd; }
</style>
<div class="table">
  <div class="row">
    <div class="cell">A1</div>
    <div class="cell">B1</div>
  </div>
  <div class="row">
    <div class="cell">A2</div>
    <div class="cell">B2</div>
  </div>
</div>
```

## Grid

- **특징:** 2차원 레이아웃(행·열). 정교한 영역 배치와 템플릿 구성이 쉬움.
- **핵심 속성:** `grid-template-columns`, `grid-template-rows`, `gap`, `grid-template-areas`, 아이템용 `grid-column/row`.
- **언제 사용:** 대시보드, 카드 그리드, 페이지 전체 레이아웃.

```html
<style>
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3열 동일폭 */
  gap: 12px;
}
.card { background: #b39ddb; padding: 12px; border-radius: 8px; }
</style>
<div class="grid">
  <div class="card">카드1</div>
  <div class="card">카드2</div>
  <div class="card">카드3</div>
  <div class="card">카드4</div>
  <div class="card">카드5</div>
  <div class="card">카드6</div>
</div>
```

## 선택 가이드

- **문장 중 일부 스타일링:** inline
- **가로로 나란히 + 크기 제어:** inline-block
- **간단한 정렬/공간 분배:** flex / inline-flex
- **복잡한 행·열 배치:** grid
- **문단/섹션:** block
- **조건부 숨김:** none
- **테이블 구조 필요:** table

원하는 화면을 말해주면 가장 적절한 `display` 조합과 세부 속성까지 맞춰 드릴게요.
