## 1. 뼈대 잡기: 중앙 정렬 컨테이너 (Container)

홈페이지의 콘텐츠가 양옆으로 무한히 퍼지지 않고 일정한 폭을 유지하며 중앙에 오게 만드는 가장 중요한 설정입니다.

```css
.container {
    max-width: 1200px; /* 홈페이지의 최대 너비 설정 */
    margin: 0 auto;    /* 좌우 마진을 auto로 주어 중앙 정렬 */
    padding: 0 20px;   /* 모바일 등 작은 화면에서 양옆 여백 확보 */
}

```

## 2. 네비게이션 바: 가로 나열 목록 (Navigation)

메뉴 바는 `<ul>`과 `<li>` 태그를 사용하여 만들며, **Flexbox**를 사용하여 가로로 나열합니다.

### 핵심 CSS

```css
nav {
    border-top: 1px solid #e1e1e1;    /* 위쪽 구분선 */
    border-bottom: 1px solid #e1e1e1; /* 아래쪽 구분선 */
}

nav ul {
    display: flex;               /* 가로로 나열 */
    justify-content: center;     /* 메뉴들을 중앙에 배치 */
    list-style: none;            /* 리스트의 점(.) 제거 */
}

nav li {
    position: relative;          /* 구분선을 넣기 위한 기준 */
}

nav a {
    display: block;
    padding: 15px 25px;          /* 클릭 영역 및 여백 확보 */
    text-decoration: none;       /* 링크 밑줄 제거 */
    color: #333;                 /* 글자 색상 */
    font-size: 15px;
}

/* 메뉴 사이의 세로 구분선 넣기 */
nav li:not(:last-child)::after {
    content: "";
    position: absolute;
    right: 0;
    top: 50%;
    transform: translateY(-50%);
    width: 1px;
    height: 12px;
    background-color: #e1e1e1;
}

/* 마우스 올렸을 때 효과 */
nav a:hover {
    color: #ff3b3b; /* 강조색으로 변경 */
}

```

## 3. 상단 헤더: 3구역 분할 (Top Header)

전화번호, 로고, 지점 위치가 좌-중-우로 배치된 형태입니다.

```css
header {
    display: flex;
    justify-content: space-between; /* 좌, 중, 우로 간격을 벌림 */
    align-items: center;            /* 세로축 중앙 정렬 */
    padding: 20px 0;
}

.logo img {
    height: 40px; /* 로고 크기 조절 */
}

```

## 4. 게시판 형태 목록 (Notice Board)

이미지 하단의 공지사항 리스트와 같은 테이블 형태는 깔끔한 구분선 처리가 핵심입니다.

```css
.board-list {
    width: 100%;
    border-collapse: collapse; /* 테두리 겹침 방지 */
    margin-top: 30px;
    border-top: 2px solid #333; /* 게시판 상단은 두꺼운 선 */
}

.board-list th, .board-list td {
    padding: 15px;
    border-bottom: 1px solid #eee; /* 각 행 사이의 얇은 구분선 */
    text-align: center;
}

.board-list .title {
    text-align: left; /* 제목은 왼쪽 정렬 */
    color: #222;
    cursor: pointer;
}

.board-list tr:hover {
    background-color: #f9f9f9; /* 마우스 올렸을 때 행 강조 */
}

```

## 5. 페이지 타이틀 (Content Title)

"학사공지"와 같은 중앙 텍스트 디자인입니다.

```css
.content-header {
    text-align: center;
    padding: 60px 0 40px;
}

.content-header h2 {
    font-size: 40px;
    font-weight: 700;
    margin-bottom: 10px;
}

.content-header p {
    color: #666;
    font-size: 18px;
}

```

### 💡 작성 팁

1. **색상 코드**: 이미지와 비슷하게 가려면 배경은 `#ffffff`(흰색), 선은 `#e1e1e1`(연한 회색), 텍스트는 `#333333`(짙은 회색)을 추천합니다.
2. **폰트**: 고딕 계열의 폰트(`noto-sans-kr` 등)를 사용하면 훨씬 기업 홈페이지 같은 정갈한 느낌이 납니다.
