# 📅 JavaScript Date 객체

> [!summary]
> 
> - **Date 객체:** 날짜와 시간을 위한 내장 객체. `new Date()`로 생성.
>     
> - **주의사항:** **월(Month)**은 `0`부터 시작(0=1월, 11=12월). 조회 시 `+1`, 설정 시 `-1` 필수.
>     
> - **날짜 연산:** `getTime()`을 사용하여 밀리초(ms) 단위로 변환 후 계산.
>     

## 1. 객체 생성 및 초기화

- **`new Date()`**: 현재 시스템의 날짜와 시간.
    
- **`new Date("2025-12-25")`**: 문자열에 해당하는 날짜 객체 생성.
    
- **`new Date(년, 월, 일, ...)`**: 숫자로 지정 (월은 -1 해서 입력).
    
```js
const now = new Date(); // 현재 시간
const xmas = new Date(2025, 11, 25); // 2025년 12월 25일 (11 입력 주의!)
```

## 2. 주요 메서드 (Getter)

날짜 정보를 가져오는 메서드들입니다.

| **메서드**         | **설명**         | **비고**              |
| --------------- | -------------- | ------------------- |
| `getFullYear()` | 4자리 연도         |                     |
| `getMonth()`    | **월 (0 ~ 11)** | **출력 시 +1 필요**      |
| `getDate()`     | 일 (1 ~ 31)     |                     |
| `getDay()`      | 요일 (0 ~ 6)     | 0: 일, 1: 월 ... 6: 토 |
| `getHours()`    | 시 (0 ~ 23)     |                     |
| `getMinutes()`  | 분 (0 ~ 59)     |                     |
| `getSeconds()`  | 초 (0 ~ 59)     |                     |
| `getTime()`     | 타임스탬프 (ms)     | 1970.1.1부터 경과한 시간   |

## 3. 📝 예시: D-Day 계산기

`getTime()`을 이용하여 두 날짜 사이의 간격을 구하는 공식입니다.

> [!EXAMPLE] 개강일로부터 며칠 지났는지 계산
> 
> 공식: (오늘ms - 과거ms) / (1000 x 60 x 60 x 24)

```js
function calculateDday() {
    const startDateInput = document.querySelector("#startDate").value;
    
    // 1. 날짜 객체 생성
    const today = new Date();
    const startDate = new Date(startDateInput);

    // 2. 유효성 검사
    if (isNaN(startDate.getTime())) {
        alert("날짜를 올바르게 선택해주세요.");
        return;
    }

    // 3. 시간 차이 계산 (밀리초)
    const diffMs = today.getTime() - startDate.getTime();

    // 4. 일(Day) 단위 변환 (밀리초 -> 초 -> 분 -> 시 -> 일)
    // Math.floor로 소수점 버림
    const diffDay = Math.floor(diffMs / (1000 * 60 * 60 * 24));

    alert(`개강일로부터 ${diffDay}일 지났습니다.`);
}
```
