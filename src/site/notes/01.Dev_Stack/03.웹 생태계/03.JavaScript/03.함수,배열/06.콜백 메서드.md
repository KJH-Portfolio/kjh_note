# JS 콜백 메서드 매개변수 및 이벤트 속성 데이터 구조

> [!summary]
> 배열 고차 함수(Array Methods)와 이벤트 리스너(Event Listener) 실행 시 콜백 함수로 자동 전달되는 **인자(Parameter)** 및 **이벤트 객체 속성(Property)** 명세 정리.

---

## 1. 배열 콜백 메서드 매개변수 (Array Callback Parameters)

`forEach`, `map`, `filter`, `reduce` 등 배열 메서드 실행 시 각 요소마다 콜백 함수에 주입되는 데이터.

### 1.1 표준 인자 구성

* **`item` (또는 `element`)**: 현재 순회 중인 배열의 **개별 데이터**. (가장 많이 사용)
* **`index`**: 현재 요소의 **순번 (0부터 시작하는 정수)**.
* **`array`**: 메서드를 호출한 **원본 배열 자체**.

### 1.2 활용 예시 코드

```javascript
const colors = ['red', 'blue', 'green'];

colors.forEach((item, index, array) => {
    // item: 'red', 'blue', 'green'
    // index: 0, 1, 2
    // array: ['red', 'blue', 'green']
    console.log(`${array.length}개 중 ${index + 1}번째 색상: ${item}`);
});

```

## 2. 이벤트 콜백 객체 속성 (Event Object Properties)

- 이벤트 발생 시 전달되는 **이벤트 객체(`e` 또는 `event`)**의 내부 속성.
- 콘솔을 통해 알아보는 방법도 있다.
### 2.1 대상 및 상태 관련 속성

* **`e.target`**: 이벤트가 **실제 발생한** 가장 안쪽의 요소 (클릭된 실제 타겟).
* **`e.currentTarget`**: 이벤트 리스너가 **직접 등록된** 요소.
* **`e.type`**: 발생한 이벤트의 **종류** (`'click'`, `'submit'`, `'keydown'` 등).

### 2.2 키보드 이벤트 전용 속성 (`keydown`, `keyup`)

* **`e.key`**: 입력된 **키의 값** (`'Enter'`, `'a'`, `'Escape'`, `' '` 등).
* **`e.code`**: 키보드의 **물리적 위치** (`'KeyA'`, `'Digit1'`, `'NumpadEnter'` 등).
* **`e.shiftKey` / `e.ctrlKey` / `e.altKey**`: 특수키 조합 여부 (**Boolean**).

### 2.3 주요 제어 메서드

* **`e.preventDefault()`**: 브라우저 기본 동작(링크 이동, 폼 제출 후 새로고침 등) **강제 중단**.
* **`e.stopPropagation()`**: 상위 요소로의 **이벤트 버블링(전파) 방지**.

## 3. 콜백 속성값 종합 요약 표

| 구분         | 주요 속성/인자                 | 설명              | 핵심 용도             |
| ---------- | ------------------------ | --------------- | ----------------- |
| **배열 콜백**  | **`item`**               | 현재 순회 데이터       | 데이터 처리 및 출력       |
|            | **`index`**              | 현재 데이터 순서       | 순번 표시, 특정 위치 접근   |
|            | **`array`**              | 전체 원본 데이터       | 전체 개수 파악, 참조 활용   |
| **이벤트 콜백** | **`e.target`**           | 클릭/입력된 요소       | DOM 제어, 데이터 추출    |
|            | `e.currentTarget`        | 이벤트 리스너가 부착된 요소 | 이벤트 위임 시 부모 요소 제어 |
|            | **`e.key`**              | 입력된 키 값         | 단축키 처리, 입력값 분기    |
|            | **`e.type`**             | 이벤트 이름          | 통합 핸들러 내 로직 구분    |
|            | **`e.preventDefault()`** | 동작 차단 함수        | 페이지 새로고침 방지 등     |

## 4. 실전 활용 예시

> [!EXAMPLE] 속성값을 활용한 동적 제어

```javascript
// 1. 배열 인덱스(index) 활용
const tags = ['JS', 'React', 'HTML'];
const upperTags = tags.map((item, index) => `${index}. ${item.toUpperCase()}`);

// 2. 이벤트 객체(e.target, e.key) 활용
document.querySelector('#userInput').addEventListener('keydown', (e) => {
    if (e.key === 'Enter') {
        const value = e.target.value; // 입력된 값 추출
        console.log(`전송 데이터: ${value}`);
        e.preventDefault(); // 엔터 시 줄바꿈 등 방지
    }
});

```
