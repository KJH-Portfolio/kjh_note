# ⚛️ JSX 배열 : 4대 핵심 메서드
> [!summary] **핵심 요약**
> * **`map`**: 데이터 → **UI** (화면 출력)
> * **`filter`**: 데이터 → **정제** (삭제/검색)
> * **`find`**: 데이터 → **단일 요소** (상세 정보)
> * **`reduce`**: 데이터 → **단일 값** (합계/통계)
> 
> 
---
## 1. map(): UI 렌더링
JSX 엘리먼트로 변환하는 **가장 빈번한** 메서드(React에서 자동으로 렌더링)
* 배열을 순회하며 return 에 있는 객체들을 합쳐 배열로 반환한다.
* **역할**: `[Data]` → `[<li>, <li>, ...]`
* **필수 사항**: 고유한 `key` 속성 부여 (성능 최적화)
* **특징**: 배열의 길이는 변하지 않음
```jsx
{items.map((item) => (
  <li key={item.id}>
    {item.name}
  </li>
))}
```
---
## 2. filter(): 데이터 삭제 및 검색
조건에 맞는 항목만 남겨서 **새로운 배열**을 반환
* 배열을 순회하며 콜백함수의 boolean값에 해당하는 것들을 배열로 감싸고 반환
* **삭제 패턴**: `id`가 일치하지 않는 것만 남기기
* **검색 패턴**: 입력값이 포함된 항목만 남기기
* **특징**: 결과값이 항상 **배열 형태**임
```jsx
// 삭제 예시: 선택한 ID만 제외
const onDelete = (id) => {
  setItems(items.filter(item => item.id !== id));
};
```
---
## 3. find(): 특정 항목 상세 조회
조건을 만족하는 **첫 번째 데이터 하나**만 가져옵니다.
* 배열을 순회하며 콜백함수의 boolean값에 해당하는 첫번째 것을 반환
* **활용**: 리스트 클릭 시 상세 모달/페이지 호출
* **반환**: 배열이 아닌 **객체(또는 값) 1개**
* **특징**: 조건을 만족하는 즉시 순회 중단
```jsx
// ID로 상품 하나만 추출
const selected = items.find(item => item.id === 5);
console.log(selected.name);
```
---
## 4. reduce(): 데이터 집계
여러 개의 데이터를 계산하여 **하나의 수치**로 응축합니다.
* **활용**: 장바구니 총액, 평균 점수, 투표 합계
* **구조**: `(누적값, 현재값) => 결과, 초기값`
```jsx
const total = cart.reduce((acc, cur) => acc + cur.price, 0);

return <div>총 금액: {total}원</div>;
```
---
## 📊 메서드 핵심 비교

| 메서드          | 반환 형태         | 주요 리액트 활용 |
| ------------ | ------------- | --------- |
| **`map`**    | **배열 (UI)**   | 목록 출력     |
| **`filter`** | **배열 (Data)** | 삭제, 검색    |
| **`find`**   | **단일 요소**     | 상세 보기     |
| **`reduce`** | **단일 값**      | 총액 계산     |

---
## 📝 종합 예시: 장바구니 패턴
> [!EXAMPLE] **실무 활용 예시**
```jsx
function Cart({ products }) {
  // 1. 특정 상품 찾기 (find)
  const promo = products.find(p => p.id === 1);

  // 2. 품절 제외 (filter)
  const list = products.filter(p => p.stock > 0);

  // 3. 총 합계 (reduce)
  const total = list.reduce((acc, cur) => acc + cur.price, 0);

  return (
    <div>
      {/* 4. 리스트 출력 (map) */}
      {list.map(p => (
        <div key={p.id}>{p.name} ({p.price}원)</div>
      ))}
      <p>합계: {total}원</p>
    </div>
  );
}
```
