# 구조 분해 할당 (Destructuring Assignment)

> [!summary]
> 배열이나 객체의 속성을 해체하여 그 값을 개별 변수에 담을 수 있게 하는 JavaScript 표현식. 변수명 변경, 중첩 구조 접근, 함수 매개변수 적용 등을 통해 데이터 추출 로직의 간결화 및 가독성 확보.

## 1. 객체 구조 분해 할당

### 1) 기본 추출 및 변수명 변경 (Aliasing)

* 객체의 키를 기준으로 값을 추출하여 변수에 할당.
* `기존키: 새변수명` 패턴을 활용하여 기존 키와 다른 이름의 변수 선언 가능.

```javascript
const user = { name: "김철수", age: 25 };

// 기본 할당
const { name, age } = user;

// 변수명 변경 할당
const { name: nickname, age: userAge } = user;
// nickname = "김철수", userAge = 25

```

### 2) 중첩 객체 구조 분해

* 객체 내부의 객체 구조에 직접 접근하여 하위 속성 추출.
* 복잡한 데이터 구조에서 필요한 데이터만 즉시 확보.

```javascript
const student = {
  id: 1,
  info: {
    email: "kim@example.com",
    score: 95
  }
};

// 중첩 구조 접근
const { info: { email, score } } = student;

```

## 2. 배열 구조 분해 할당

* 배열의 요소를 순서(인덱스)대로 변수에 할당.
* 쉼표(`,`)를 활용한 특정 인덱스 건너뛰기 가능.

```javascript
const colors = ["빨강", "노랑", "파랑"];

// 인덱스 기반 할당 및 건너뛰기
const [red, , blue] = colors; 
// red = "빨강", blue = "파랑"

```

---

## 3. 함수 매개변수 활용

* 함수 인자 전달 시 객체나 배열을 즉시 분해하여 수신.
* 필요한 속성만 명시적으로 선언하여 함수 내부 코드의 명확성 증대.

```javascript
// 매개변수 단계에서의 분해 및 기본값 설정
const printUser = ({ name, age, nation = "Korea" }) => {
  console.log(name, age, nation);
};

const person = { name: "이영희", age: 30 };
printUser(person); // 이영희 30 Korea

```

## 4. 종합 예시 (Example)

> [!EXAMPLE] 개념 비유
> * **기본 분해**: 세트 메뉴에서 버거와 감자튀김을 각각의 접시에 나누어 담는 행위.
> * **변수명 변경**: 수령한 '택배물'을 내 방에서는 '선물'이라는 이름으로 부르는 것.
> * **중첩 분해**: 커다란 상자 안의 작은 보석 상자를 열어 반지 속성만 꺼내는 과정.
> 
> 

```javascript
// 실전 응용: API 응답 데이터 처리
const response = {
  status: 200,
  data: {
    items: ["A", "B"],
    count: 2
  }
};

const { data: { items: itemList, count } } = response;
// itemList = ["A", "B"], count = 2

```
