# 자바스크립트 배열(Array)

> [!summary]
> 자바스크립트 배열은 **동적 타입**과 **동적 크기**를 지원하는 강력한 지식 저장소입니다. 데이터의 타입을 가리지 않고 담을 수 있으며, 다양한 내장 메서드를 통해 데이터를 효율적으로 조작할 수 있는 것이 핵심입니다.

## 1. 배열의 기본 특징 및 선언

### 1.1 주요 특징

* **동적 타입**: 숫자, 문자, 객체, 심지어 다른 배열까지 하나의 배열에 혼합하여 저장할 수 있습니다.
* **가변 크기**: 데이터 양에 따라 메모리 크기가 자동으로 조절됩니다.
* **희소 배열**: 인덱스를 건너뛰고 값을 할당하면 중간은 `undefined`로 채워지며 크기가 확장됩니다.
  (의도적으로 사용하지는 않는다.)

### 1.2 선언 방식

* **리터럴 방식 (권장)**: 가장 직관적이고 가독성이 좋습니다.
* **생성자 방식**: 특정 크기의 빈 배열을 미리 만들 때 유용합니다.

```javascript
// 리터럴 방식
const fruits = ['사과', '바나나'];

// 생성자 방식
const emptyArr = new Array(); 
const fixedSizeArr = new Array(5); // 크기가 5인 빈 배열

```

## 2. 배열 메서드 (Array Methods)

배열 조작 시 가장 주의할 점은 
- **"원본 배열이 변경되는가(Mutable)"** 아니면 
- **"새로운 배열을 반환하는가(Immutable)"**입니다.

### 2.1 원본을 변경하는 메서드 (Mutable)

주로 배열 자체의 데이터를 추가하거나 삭제할 때 사용합니다.

| 메서드 | 설명 | 반환값 |
| --- | --- | --- |
| **`push()`** | 배열 **맨 뒤**에 요소 추가 | 변경된 배열의 **길이** |
| **`pop()`** | 배열 **맨 뒤** 요소 제거 | **제거된 요소** |
| **`unshift()`** | 배열 **맨 앞**에 요소 추가 | 변경된 배열의 **길이** |
| **`shift()`** | 배열 **맨 앞** 요소 제거 | **제거된 요소** |
| **`splice()`** | 특정 위치 요소 삭제/추가 | **삭제된 요소들의 배열** |
| **`sort()`** | 요소를 정렬 (기본은 문자열 오름차순) | 정렬된 배열 |

### 2.2 원본을 보존하는 메서드 (Immutable)

데이터를 추출하거나 가공하여 새로운 결과물을 만들 때 사용합니다.

| 메서드 | 설명 | 반환값 |
| --- | --- | --- |
| **`indexOf()`** | 특정 값의 위치(인덱스) 찾기 | 해당 인덱스 (없으면 **-1**) |
| **`slice()`** | 배열의 일부분을 복사 추출 | **새로운 배열** |
| **`concat()`** | 두 개 이상의 배열을 합침 | **합쳐진 새로운 배열** |
| **`join()`** | 요소를 특정 구분자로 연결 | **문자열** |

---

## 3. 값의 비교: 동등(`==`) vs 일치(`===`)

배열 내부의 데이터를 비교하거나 조건문을 작성할 때 필수적으로 구분해야 합니다.

* **`==` (동등 연산자)**: **값**만 같으면 참입니다. (자료형이 다르면 자동으로 형변환을 시도합니다.)
* **`===` (일치 연산자)**: **값과 자료형**이 모두 일치해야 참입니다. (JS에서는 실수를 방지하기 위해 이 방식을 권장합니다.)

## 4. 실전 활용 예시

> [!EXAMPLE] 할 일 목록(To-Do List) 관리 로직
> 배열의 추가, 삭제, 순회 기능을 조합하여 간단한 할 일 목록 관리 기능을 구현한 예시입니다.

```javascript
let todoList = ['공부하기', '청소하기'];

// 1. 새로운 할 일 추가 (맨 뒤)
const addNewTodo = (task) => todoList.push(task);

// 2. 가장 오래된 할 일 완료 처리 (맨 앞 삭제)
const completeOldest = () => {
    const done = todoList.shift();
    console.log(`${done} 완료!`);
};

// 3. 목록 출력 (for...of 사용)
const printTodos = () => {
    console.log("--- 오늘의 할 일 ---");
    for (const task of todoList) {
        console.log(`- ${task}`);
    }
};

addNewTodo('운동하기');
completeOldest();
printTodos();

```
