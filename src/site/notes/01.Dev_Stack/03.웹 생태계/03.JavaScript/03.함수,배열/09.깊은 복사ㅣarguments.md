# 깊은 복사 및 나머지 매개변수 심화

> [!summary]
> 전개 연산자의 한계인 얕은 복사를 극복하기 위한 **깊은 복사(Deep Copy)** 기법과, JavaScript의 가변 인자 처리 방식인 **나머지 매개변수** 및 과거의 **arguments 객체** 간의 차이점 정리. 데이터 무결성 유지와 현대적 코드 작성의 핵심 개념.

---

## 1. 얕은 복사 vs 깊은 복사 (Shallow vs Deep Copy)

전개 연산자(`...`)를 통한 복사는 1단계 계층만 복제되는 **얕은 복사**의 특성 보유.

### 1) 얕은 복사의 한계

* 객체 내부의 중첩된 객체는 복사가 아닌 **참조값(주소)**만 복제.
* 복사본의 중첩 객체 수정 시 원본 객체도 함께 변경되는 부작용 발생.

### 2) 깊은 복사 구현 방법

* **JSON 방식**: `JSON.stringify`로 문자열화 후 `JSON.parse`로 재구성. (함수, Symbol 등 누락 위험 존재)
* **Lodash 라이브러리**: `_.cloneDeep()` 메서드 활용. 가장 안정적이고 보편적인 해결책.
* **StructuredClone**: 최신 브라우저 내장 API를 이용한 깊은 복사 수행.

```javascript
// 1. JSON 방식
const deepCopy = JSON.parse(JSON.stringify(original));

// 2. Lodash 활용 (사전 설치 필요)
import _ from 'lodash';
const deepCopyLodash = _.cloneDeep(original);

```

## 2. arguments 객체(구식 문법)

가변적인 인자를 다루는 두 방식(Rest,arguments)의 기술적 차이
### 주요 차이점 비교 테이블

| 구분         | 나머지 매개변수 (`...args`)                 | arguments 객체              |
| ---------- | ------------------------------------ | ------------------------- |
| **형태**     | **표준 배열**                            | **유사 배열 객체** (Array-like) |
| **배열 메서드** | `map`, `filter`, `reduce` 등 즉시 사용 가능 | 사용 불가 (배열 변환 필요)          |
| **화살표 함수** | 사용 가능                                | **사용 불가**                 |
| **범위**     | 명시된 매개변수 제외 나머지 추출 가능                | 모든 인자를 포함                 |

```javascript
// 나머지 매개변수 (권장)
const modernFunc = (...args) => {
  return args.sort(); // 배열 메서드 직접 호출
};

// arguments 객체 (과거 방식)
function legacyFunc() {
  const argsArray = Array.from(arguments); // 배열 변환 필수
  return argsArray.sort();
}

```

## 3. 종합 예시 (Example)

> [!EXAMPLE] 개념 비유
> * **얕은 복사**: 서류 뭉치를 복사했으나, 그 안의 비밀번호가 적힌 메모지는 원본과 복사본이 동일한 메모지를 공유하는 상태.
> * **깊은 복사**: 서류 뭉치뿐만 아니라 그 안의 메모지 내용까지 새 종이에 똑같이 옮겨 적어 완벽히 분리된 상태.
> * **나머지 매개변수**: 들어오는 물건들을 처음부터 규격화된 바구니(배열)에 담아 정리하는 현대적 방식.
> 
> 

### 실전 응용 코드

```javascript
// 1. 깊은 복사 실습 (Lodash cloneDeep 효과)
const user = {
  name: "이몽룡",
  details: { age: 20, city: "남원" }
};

const shallowUser = { ...user };
const deepUser = JSON.parse(JSON.stringify(user));

shallowUser.details.age = 30; 
// user.details.age 도 30으로 변경 (얕은 복사의 문제)

deepUser.details.age = 40;
// user.details.age 는 유지 (깊은 복사의 성공)


// 2. 나머지 매개변수 활용 가변 인자 처리
const logMessages = (level, ...messages) => {
  console.log(`[${level}]`, messages.join(" | "));
};

logMessages("Error", "파일 없음", "연결 실패", "메모리 부족");
// [Error] 파일 없음 | 연결 실패 | 메모리 부족

```
