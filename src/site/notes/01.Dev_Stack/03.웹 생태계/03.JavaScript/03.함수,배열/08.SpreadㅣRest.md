# 전개 연산자 및 나머지 매개변수 (Spread & Rest)

> [!summary]
> 마침표 세 개(`...`)를 사용하여 데이터를 낱개로 펼치는 **전개 연산자**와, 여러 인자를 하나의 배열로 모으는 **나머지 매개변수** 문법 정리. 데이터 복사, 병합 및 가변 인자 처리의 효율성 극대화.

## 1. 전개 연산자 (Spread Operator)

반복 가능한 객체(배열, 객체 등)를 개별 요소로 분리하여 복사하거나 병합할 때 사용.

### 활용 유형 및 특징

| 구분 | 내용 | 주요 특징 |
| --- | --- | --- |
| **복사(Copy)** | 기존 배열/객체의 복사본 생성 | 얕은 복사(Shallow Copy) 수행 |
| **병합(Merge)** | 여러 데이터를 하나로 결합 | 중복 키 존재 시 뒤의 값이 우선 |
| **수정(Update)** | 원본 유지 및 특정 값 변경 | 리액트 상태 관리 등 불변성 유지에 필수 |

```javascript
// 배열 및 객체 전개
const arr = [1, 2];
const newArr = [...arr, 3]; // [1, 2, 3]

const obj = { x: 1, y: 2 };
const newObj = { ...obj, z: 3 }; // { x: 1, y: 2, z: 3 }

```

## 2. 나머지 매개변수 (Rest Parameters)

함수 호출 시 전달된 여러 개의 인자를 하나의 배열로 집합.

### 주요 규칙

* **배열화**: 가변적인 인자들을 표준 배열로 변환하여 배열 메서드(`map`, `filter` 등) 사용 가능.
* **위치 제한**: 함수의 매개변수 명단 중 반드시 가장 **마지막**에 위치.
* **구조 분해와의 결합**: 배열이나 객체 분해 시 남은 요소들을 한곳에 수집 가능.

```javascript
// 함수 인자 수집
function multiply(factor, ...numbers) {
  return numbers.map(n => n * factor);
}

// 구조 분해에서의 활용 (Rest Element)
const [first, ...others] = [10, 20, 30, 40];
// first = 10, others = [20, 30, 40]

```

## 3. 종합 예시 (Example)

> [!EXAMPLE] 개념 비유
> * **전개 연산자**: 가방 안의 물건들을 바닥에 전부 쏟아 놓아 다른 가방에 합치기 편하게 만드는 상태.
> * **나머지 매개변수**: 입구에 들어오는 사람들에게 번호표를 주고 남은 사람들은 모두 '대기 그룹'이라는 하나의 명단(배열)에 묶어버리는 행위.
> 
> 

```javascript
// 1. 객체 업데이트와 전개 연산자
const originalConfig = { theme: "dark", fontSize: 14, showSidebar: true };
const userConfig = { ...originalConfig, fontSize: 16 };
// { theme: "dark", fontSize: 16, showSidebar: true }

// 2. 가변 인자 합계 구하기 (Rest Parameter)
const calculateSum = (...args) => args.reduce((sum, val) => sum + val, 0);

console.log(calculateSum(1, 2, 3)); // 6
console.log(calculateSum(1, 2, 3, 4, 5)); // 15

```
