# 📜 JavaScript 기본 문법 노트

> [!summary]
> 
> - **핵심 문법:** `var`, `let`, `const` 키워드를 사용한 변수 선언 및 범위(Scope) 이해.
>     
> - **자료형:** 값에 따라 자동 결정되는 동적 자료형 (String, Number, Boolean 등).
>     
> - **형 변환:** `+` 연산 시 문자열 우선, 그 외 연산 시 숫자 우선 자동 변환. `Number()`, `parseInt()` 등 명시적 변환.
>     

---

## 1. 🌐 자바스크립트의 개요와 환경

- **역할:** 웹 문서의 **동적인 효과(행동)** 담당. HTML(뼈대), CSS(스타일) 보완.
- **실행 방식:** 인터프리터(Interpreter) 방식. 코드를 **한 줄씩 읽고 처리**하는 스크립트 언어.
- **코드 작성 위치:**
    - **Inline:** 태그 내 `onclick` 등 이벤트 속성에 직접 작성.
    - **Internal:** HTML 문서 내 `<script>` 태그 영역에 함수 단위로 작성.
    - **External:** 별도의 `.js` 파일로 작성 후 HTML에 `<script src="...">`로 연결.
        

---

## 2. 🧱 변수, 자료형 및 범위 (Scope)

- **특징:** 변수 선언 시 자료형을 별도로 지정하지 않음. 값 대입 시 자료형이 **자동으로 결정** (동적 타이핑).

### 2.1. 변수 선언 키워드와 특징

|**키워드**|**범위 (Scope)**|**재선언**|**재할당**|**특징**|
|---|---|---|---|---|
|**`var`**|**함수 스코프**|가능|가능|`window` 객체에 속함 (전역 오염 가능성).|
|**`let`**|**블록 스코프**|불가능|가능|ES6 문법. `var`의 단점(중복 선언) 보완.|
|**`const`**|**블록 스코프**|불가능|불가능|**상수**. 선언 시 반드시 초기화 필요.|

- **변수 우선순위:** 지역 변수와 전역 변수명이 동일할 경우, **지역 변수**가 우선권 가짐.
    

### 2.2. 자료형과 내장 객체

- **주요 자료형:** String, Number, Boolean, Object, Function, Undefined, Null.
- **자료형 확인:** `typeof()` 함수 사용.
- **객체 접근:** `객체명.속성명` 또는 `객체명['속성명']` 형식.
- **`window` 객체:** 브라우저 창이 열릴 때 생성되는 **모든 요소의 최상위 객체**. (DOM과 Core JS의 교집합)
- **`Math` 객체:** 숫자 관련 함수(난수, 올림, 버림 등) 제공.

## 3. 🔢 데이터 처리 및 형 변환

### 3.1. 문자열과 숫자 관련 함수

- **문자열 (`String`):**
    - `str.length`: 문자열 길이.
    - `str.charAt(i)`: i번째 인덱스 문자 반환.
    - `str.indexOf("l")`: 대상 문자의 첫 번째 위치(인덱스) 반환.
    - `str.split(",")`: 구분자를 기준으로 문자열을 나눠 **배열(Array)**로 반환.
        
- **숫자 (`Number`) 및 수학 (`Math`):**
    - `Math.random()`: 0 이상 1 미만의 랜덤 실수 반환.
    - `Math.round(12.3)`: 반올림 ( $\rightarrow$ 12).
    - `Math.floor(12.3)`: 버림 ( $\rightarrow$ 12).
    - `Math.ceil(12.3)`: 올림 ( $\rightarrow$ 13).
        

### 3.2. 산술 연산 및 강제 형 변환

- **산술 연산 (자동 형 변환):**
    - **덧셈 (`+`):** 피연산자 중 하나라도 문자열이면 **문자열 연결** 처리 (예: `5 + "5"` $\rightarrow$ `"55"`).
    - **다른 연산 (`-`, `*`, `/`, `%`):** 문자열을 **숫자로 자동 형 변환** 후 연산 (예: `"5" * 5` $\rightarrow$ $25$).
        
- **강제 형 변환 (문자열 $\rightarrow$ 숫자):**
    - **`Number(문자열)`:** 숫자가 아닌 값이 포함되면 **`NaN`** (Not a Number) 반환.
    - **`parseInt(문자열)`:** 숫자가 아닌 문자를 만나기 전까지 **정수**로 변환.
    - **`parseFloat(문자열)`:** 숫자가 아닌 문자를 만나기 전까지 **실수**로 변환.
    - **`NaN` 확인:** `isNaN(값)` 함수 사용 (결과가 `NaN`이면 `true` 반환).
        
### 4. 🛑 JavaScript의 Falsy 값 6가지

| **Falsy 값**     | **타입**    | **의미**                           |
| --------------- | --------- | -------------------------------- |
| **`false`**     | Boolean   | 불리언 타입의 거짓 값 (가장 명확한 Falsy)      |
| **`0`**         | Number    | 숫자 `0` (음수 `-0`도 포함)             |
| **`''`**        | String    | **빈 문자열** (공백이 없는 문자열)           |
| **`null`**      | Object    | 값이 **없음** (의도적으로 비움)             |
| **`undefined`** | Undefined | 값이 **정의되지 않음**                   |
| **`NaN`**       | Number    | **Not a Number** (유효하지 않은 숫자 결과) |


### 5. VS Code 강조색상 정리

| **VS Code 색상** | **해당 JavaScript 요소**         | **역할**                                                            |
| -------------- | ---------------------------- | ----------------------------------------------------------------- |
| **노란색** 🟡     | **메서드/함수** (Method/Function) | 객체에 연결되어 동작을 수행하는 기능 (예: `getTime()`, `map()`, `push()`)          |
| **하늘색/파란색** 💙 | **키워드** (Keyword)            | 언어의 구조를 정의하는 예약어 (예: `let`, `const`, `function`, `new`, `return`) |
| **흰색/밝은 회색** ⚪ | **변수 이름** (Variable Name)    | 값을 담는 저장소 이름 (예: `timeNow`, `area1`)                              |
| **주황색/갈색** 🟠  | **속성** (Property)            | 객체에 저장된 값 (예: `innerHTML`, `name`, `length`)                      |

> [!EXAMPLE]
> 
> 자료형 및 형 변환 개념 예시
> 
> - `let a = 10;` (Number)
>     
> - `let b = "20";` (String)
>     
> - a + b의 결과: "1020"
>     
>     (문자열 + 연산이 우선되어 숫자 10이 문자열 "10"으로 자동 변환된 후 연결)
>     
> - a - b의 결과: -10
>     
>     (숫자 - 연산이 우선되어 문자열 "20"이 숫자 20으로 자동 변환된 후 계산)
>     
> - Number(b) + a의 결과: 30
>     
>     (Number() 함수로 "20"을 숫자 20으로 명시적 강제 변환)
>
