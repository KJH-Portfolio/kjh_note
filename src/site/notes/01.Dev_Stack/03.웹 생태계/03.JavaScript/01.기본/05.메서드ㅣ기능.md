# JavaScript 메서드/기능 모음

> [!summary]
> 데이터 처리, 텍스트/숫자 가공, DOM 조작, 비동기 통신 등 JavaScript 개발 전반에 걸쳐 가장 빈번하게 사용되는 핵심 메서드와 속성들을 체계적으로 구조화한 종합 가이드.

---

## 1. 데이터 처리 (Array & Object)

데이터의 변형, 탐색 및 구조 분해를 포함한 핵심 조작 기법.

| 카테고리 | 메서드 / 속성 | 설명 | 비고 |
| --- | --- | --- | --- |
| **Array** (변형) | **`map()`** | 배열 요소를 일괄 변환하여 새 배열 생성 | 리액트 리스트 렌더링 필수 |
|  | **`filter()`** | 조건에 맞는 요소만 추출하여 새 배열 생성 | 데이터 삭제/검색 로직 |
|  | **`reduce()`** | 누적 계산(합계, 객체 변환 등) 수행 | `accumulator` 활용 |
|  | **`flat()`** | 다차원 배열을 지정된 깊이까지 평탄화 | 중첩 데이터 정리 |
| **Array** (조작) | **`slice()`** | 원본 유지하며 배열의 일부 복사본 추출 | 불변성 유지 권장 |
|  | **`splice()`** | 원본 수정하며 요소 추가, 삭제, 교체 | 원본 직접 변경 주의 |
|  | **`push() / pop()`** | 배열 끝에 요소 추가 / 끝 요소 제거 | 스택(Stack) 구조 활용 |
| **Array** (탐색) | **`find()`** | 조건에 일치하는 첫 번째 요소 자체를 반환 | 단일 객체 탐색 추천 |
|  | **`includes()`** | 특정 값 포함 여부 확인 (T/F) | 단순 존재 유무 확인 |
|  | **`Array.from()`** | 유사 배열(NodeList 등)을 표준 배열로 변환 | 배열 메서드 사용 전 필수 |
| **Object** | **`keys() / values()`** | 객체의 키 또는 값만 모아 배열로 반환 | 객체 반복 처리 시 유용 |
|  | **`entries()`** | `[키, 값]` 형태의 2차원 배열 반환 | 구조 분해 할당과 조합 |
|  | **`{...obj}` / `[...arr]**` | 전개 연산자를 이용한 복사 및 병합 | 얕은 복사 수행 |

## 2. 텍스트 및 숫자 처리 (String & Number)

사용자 입력 데이터 정제 및 수치 데이터 포맷팅.

| 카테고리 | 메서드 / 속성 | 설명 | 비고 |
| --- | --- | --- | --- |
| **String** (추출) | **`slice()`** | 시작~끝 인덱스 기준으로 문자열 추출 | `substring` 대비 범용성 높음 |
|  | **`split()`** | 구분자 기준으로 문자열을 분할하여 배열 생성 | CSV, 경로 데이터 분리 |
| **String** (변형) | **`replaceAll()`** | 일치하는 모든 문자열을 찾아 치환 | 정규식 없이 전체 치환 가능 |
|  | **`trim()`** | 문자열 양 끝의 공백 제거 | 유효성 검사 전 필수 |
| **String** (탐색) | **`includes()`** | 특정 문자열 포함 여부 확인 | 키워드 검색 로직 |
|  | **`startsWith()`** | 특정 문자로 시작하는지 확인 | URL, 프로토콜 판별 |
| **Number** | **`parseInt()`** | 문자열을 정수형 숫자로 변환 | 진수 지정 권장 |
|  | **`toFixed()`** | 소수점 자릿수 고정 및 문자열 반환 | 금융 데이터 처리 |
|  | **`Number.isNaN()`** | 전달된 값이 실제 `NaN`인지 엄격하게 검사 | `isNaN()`보다 안전 |

## 3. DOM 탐색 및 조작 (DOM Traversal & Manipulation)

브라우저의 요소 구조 접근 및 스타일/데이터 제어.

| 카테고리 | 메서드 / 속성 | 설명 | 비고 |
| --- | --- | --- | --- |
| **선택 (Selector)** | **`querySelector()`** | CSS 선택자로 일치하는 첫 요소 선택 | 표준 선택 도구 |
|  | **`closest()`** | 상위 방향으로 가장 가까운 조상 요소 탐색 | 이벤트 위임의 핵심 |
| **탐색 (Up/Down)** | **`parentElement`** | 직계 부모 요소 접근 | 상위 노드 참조 |
|  | **`children`** | 자식 요소들의 목록 반환 (HTMLCollection) | 실질적 요소 노드만 포함 |
|  | **`firstElementChild`** | 첫 번째 자식 요소 접근 | 첫 노드 타겟팅 |
| **탐색 (Side)** | **`nextElementSibling`** | 인접한 다음 형제 요소 접근 | 순차 탐색 |
|  | **`previousElementSibling`** | 인접한 이전 형제 요소 접근 | 역순 탐색 |
| **조작 (Action)** | **`classList`** | `add`, `remove`, `toggle`로 클래스 제어 | 스타일 변경 표준 방식 |
|  | **`append()`** | 자식 노드 끝에 요소나 텍스트 추가 | 다중 추가 가능 |
|  | **`textContent`** | 요소 내부의 순수 텍스트 제어 | XSS 방어(HTML 미해석) |
|  | **`dataset`** | `data-*` 속성 읽기 및 쓰기 | 사용자 정의 데이터 바인딩 |

## 4. 비동기 및 이벤트 (Async & Event)

사용자 인터랙션 및 서버 통신 처리.

| 카테고리 | 메서드 / 속성 | 설명 | 비고 |
| --- | --- | --- | --- |
| **Event** | **`addEventListener()`** | 특정 이벤트 발생 시 실행할 함수 등록 | 다중 리스너 등록 가능 |
|  | **`preventDefault()`** | 브라우저 기본 동작(링크 이동 등) 중단 | 폼 제출 처리 필수 |
|  | **`stopPropagation()`** | 이벤트 버블링(부모 전파) 차단 | 중첩 클릭 이슈 해결 |
| **Async** | **`fetch()`** | 네트워크 요청을 통한 리소스 취득 | Promise 기반 통신 |
|  | **`async / await`** | 비동기 로직을 동기적인 흐름으로 작성 | 현대적 비동기 처리 |
|  | **`JSON.parse()`** | JSON 문자열을 JavaScript 객체로 변환 | 수신 데이터 처리 |
|  | **`JSON.stringify()`** | JavaScript 객체를 JSON 문자열로 변환 | 서버 전송 및 로컬 저장 |

## 5. 기타 유틸리티 (Date, Math, Console)

시간 처리, 수학적 계산 및 디버깅 도구.

| 카테고리 | 메서드 / 속성 | 설명 | 비고 |
| --- | --- | --- | --- |
| **Date** | **`new Date()`** | 현재 시각 기반의 날짜 객체 생성 | 시간 데이터 기준점 |
|  | **`toLocaleString()`** | 국가별 설정에 맞춘 날짜/시간 포맷팅 | 현지화 출력용 |
| **Math** | **`Math.random()`** | 0 이상 1 미만의 부동 소수점 난수 생성 | 무작위성 필요 시 사용 |
|  | **`Math.floor()`** | 소수점 이하 내림 처리하여 정수 반환 | 난수 정수화 결합 필수 |
| **Window** | **`location.href`** | 현재 페이지 URL 정보 및 페이지 이동 | 브라우저 주소 제어 |
|  | **`setTimeout()`** | 지정된 밀리초(ms) 후 콜백 함수 실행 | 지연 로직 처리 |
| **Console** | **`console.log()`** | 콘솔창에 일반 정보 출력 | 기본 디버깅 |
|  | **`console.table()`** | 배열/객체 데이터를 표 형식으로 출력 | 가독성 높은 디버깅 |
