# 🖱️ JavaScript DOM 조작 노트

> [!summary]
>  
>
>   - **DOM (Document Object Model):** HTML 문서를 객체(Object)로 해석하여 JavaScript가 제어할 수 있게 만든 모델.
>          
>   - **핵심 조작:** `document.querySelector()`로 요소를 **선택**하고, `.innerHTML`, `.style`, `.dataset` 등으로 내용을 **조작**함.
>          
>   - **`document` 객체:** 현재 웹 문서(HTML) 정보 및 요소를 제어하는 객체.
>          

-----

## 1\. ⌨️ DOM 데이터 입출력

### 1.1. 데이터 출력 구문

  - `window.alert('알림창')`: 사용자 확인용 알림창 출력.
        
  - `window.console.log('콘솔창')`: 개발자 도구(F12) 콘솔에 로그 출력 (디버깅용).
        
  - `document.write('문서 화면')`: 문서 화면에 HTML 출력 (페이지 로드 완료 후 실행 시 기존 내용 덮어씀).
        
  - `요소.innerHTML` / `요소.innerText`: **선택된 요소(Element)** 내부에 값 출력. (DOM 조작의 핵심)
        

### 1.2. 데이터 입력 구문

  - `window.confirm('질문')`: [확인]/[취소] 버튼 제공. **`true`** (확인) 또는 **`false`** (취소) 반환.
        
  - `window.prompt('질문')`: 텍스트 입력 상자 제공. 입력된 **값(문자열)** 또는 취소 시 **`null`** 반환.
        
  - `input요소.value`: `<input>` 태그 요소에 입력된 값을 **추출**.
        

-----

## 2\. 🎯 DOM 요소 선택 및 조작

### 2.1. 요소 선택 (Element Access)

  - `document.querySelector("CSS선택자")`: CSS 선택자에 해당하는 **첫 번째 요소 하나** 반환 (가장 많이 사용).
        
  - `document.querySelectorAll("CSS선택자")`: CSS 선택자에 해당하는 **모든 요소**를 `NodeList` (유사 배열)로 반환.

```js 
%% 구식방법 (사용X) %%
document.getElementById("id명");
document.getElementsByTagName("태그명");
```

### 2.2. 요소 내용/속성 조작

  - **`요소.innerHTML`:** 요소 내부의 **HTML 태그**를 **인식**하며 내용 접근/변경.
        
  - **`요소.innerText`:** 요소 내부의 **순수한 텍스트**만 접근/변경 (태그를 문자열로 처리).
        
  - **`요소.style.속성명`:** 요소의 CSS 스타일 변경 (예: `area1.style.backgroundColor = "skyblue"`).
        
  - **`요소.value`:** (주로 `<input>` 요소에서) 요소의 '값'을 가져오거나 설정.
        
  - **`요소.classList`:** 클래스 추가/제거 (`.add()`, `.remove()`, `.toggle()`).

### 2.3. 🏷️ 커스텀 데이터 속성 (dataset)

HTML 태그에 숨겨진 데이터를 저장하고, JS로 제어할 때 사용합니다.

  - **HTML:** `data-속성명="값"` (속성명은 소문자와 하이픈 권장)
  - **JS:** `요소.dataset.속성명` (자동으로 **CamelCase**로 변환됨)

```html
<div id="user" data-user-id="12345" data-role="admin"></div>
```

```javascript
const user = document.querySelector("#user");

// 1. 읽기 (HTML의 data-user-id -> JS의 userId)
console.log(user.dataset.userId); // "12345"
console.log(user.dataset.role);   // "admin"

// 2. 쓰기 (HTML 속성도 data-grade="vip"로 자동 생성됨)
user.dataset.grade = "vip"; 
```

## 3\. 💡 종합 예시: DOM 조작 및 데이터 처리

> [!EXAMPLE]
>  
> innerHTML vs innerText 개념 예시
>  
>
>   - **상황:** `<div id="test">안녕하세요! <strong>반갑습니다.</strong></div>` 라는 요소가 있을 때
>          
>   - `document.querySelector("#test").innerText`의 결과 $\rightarrow$ "안녕하세요\! 반갑습니다." (순수 텍스트만 추출)
>          
>   - `document.querySelector("#test").innerHTML`의 결과 $\rightarrow$ "안녕하세요\! `<strong>`반갑습니다.`</strong>`" (내부 HTML 태그까지 모두 포함)
>          

```javascript
/*
  HTML에 <input id="n4"> 와 <button onclick="check2()"> 가 있다고 가정.
  check2() 함수는 입력된 이름이 리스트에 있는지 확인하고, 
  입력창을 비우는 DOM 조작을 수행.
*/
function check2(){
  let list = ["박보검","이효리","김윤아","차태형","박보영","이상순"];
  
	// 1. DOM 요소 선택
	let inputElement = document.querySelector("#n4");
	
	// 2. DOM 요소의 '값' 추출 (데이터 입력)
	let check = inputElement.value.trim();  

	// 3. 데이터 처리 (Array.includes() 사용)
	if (check === "") {
		alert("이름을 입력해주세요.");
	} else if (list.includes(check)) {
		alert(`"${check}" 이름이 있습니다.`);
	} else {
		alert(`"${check}" 이름이 없습니다.`);
	}
  
  // 4. DOM 조작 (입력창 초기화 및 포커스)
  inputElement.value = "";  
  inputElement.focus(); // 사용자 편의성을 위해 커서 이동
}
```
