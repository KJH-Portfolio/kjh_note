# 자바스크립트 괄호 문법 및 활용 가이드
> [!summary]
> 자바스크립트에서 데이터 구조 정의, 함수 제어, 연산 우선순위 결정을 위해 사용하는 세 가지 주요 괄호(**[]**, **()**, **{}**)의 명세 및 활용법. 동적 프로퍼티 접근부터 코드 블록 정의까지의 핵심 기능 포함.
---
## 소괄호 (Parentheses: `()`)
* **함수 호출 및 매개변수 정의**: 함수 실행을 위한 인자 전달 및 선언 시의 파라미터 수신 구획.
* **표현식 그룹화**: 수학적 연산 또는 논리 연산의 우선순위 강제 지정.
* **제어문 조건식**: `if`, `while`, `switch` 등의 키워드 뒤에서 평가될 조건 기술.
* **화살표 함수 선언**: 인수가 없거나 다수인 경우의 매개변수 그룹화.
* **즉시 실행 함수(IIFE)**: 익명 함수의 정의와 동시에 실행을 위한 구문 감싸기.
```javascript
// 함수 호출 및 연산 우선순위
function multiply(a, b) {
  return a * b;
}

const total = multiply(2, (3 + 5)); // 16

if (total > 10) {
  console.log("Result is large");
}

// 화살표 함수
const greet = () => console.log("Hi");
```

## 중괄호 (Curly Braces: `{}`)
* **객체 리터럴 정의**: 키-값(Key-Value) 쌍을 포함하는 객체 생성.
* **코드 블록 획정**: 함수 본문, 조건문, 반복문의 실행 범위 설정 및 지역 스코프 형성.
* **템플릿 리터럴 보간**: `${}` 내부에 변수 또는 표현식을 삽입하여 문자열 생성.
* **객체 구조 분해 할당**: 객체 내부의 특정 프로퍼티를 변수로 즉시 추출.
* **모듈 가져오기 및 내보내기**: `import` 및 `export` 문에서 특정 명칭을 지정하여 모듈 공유.
```javascript
// 객체 정의 및 구조 분해
const member = { id: 1, name: "Admin" };
const { name } = member;

// 코드 블록 및 템플릿 리터럴
function showInfo(user) {
  if (user) {
    console.log(`User name: ${user.name}`);
  }
}

showInfo(member);
```

## 대괄호 (Square Brackets: `[]`)
* **객체 프로퍼티 동적 참조**: 변수 또는 식의 결과값을 키로 활용하여 객체 멤버에 접근.
* **특수 문자 키 처리**: 점 표기법으로 접근 불가능한 공백, 하이픈(`-`), 숫자로 시작하는 키의 핸들링.
* **배열 리터럴 생성**: 요소의 순차적 나열을 통한 배열 데이터 구조 정의.
* **배열 인덱스 접근**: 0부터 시작하는 정수를 통한 특정 위치 요소 검색 및 수정.
* **배열 구조 분해 할당**: 배열 요소를 개별 변수로 추출하여 할당하는 문법.
* **계산된 프로퍼티 이름**: 객체 생성 단계에서 대괄호 내 표현식을 통한 키 이름의 동적 결정.
```javascript
//배열
const arr = [10, 20, 30];
const [first, second] = arr;

//객체 동적 접근1
const key = "status";
const obj = { [key]: "active", "user-id": 101 };

//객체 동적 접근2
console.log(obj[key]);        // "active"
console.log(obj["user-id"]);  // 101
console.log(arr[1]);          // 20
```
