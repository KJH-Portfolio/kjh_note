# 리액트 createPortal 및 이벤트 전파 제어
> [!summary] 요약: 부모 컴포넌트의 DOM 계층 구조와 상관없이 지정된 외부 DOM 노드에 자식 요소를 렌더링하는 **React DOM API** 및 리액트 트리 내 **이벤트 버블링 차단** 기술.
---
## 1. createPortal 정의 및 이론적 메커니즘
* **DOM 트리 분리**
	* 부모 컴포넌트의 물리적 위치에 국한되지 않는 **독립적 위치**에 요소 배치.
	* `index.html` 내 별도 정의된 루트 노드(예: `modal-root`) 활용 가능.
* **이벤트 전파 규칙**
	* 물리적 위치와 무관하게 **리액트 트리 계층**을 통한 상향식 이벤트 전달(**Bubbling**) 유지.
	* 포털 내부에서 발생한 이벤트는 리액트 상위 컴포넌트로 정상 전달.
* **논리적 일관성**
	* **Context API** 접근성 및 **Props** 전달 방식 등 기존 리액트 컴포넌트 특성 동일 적용.
* **구문 구성**
	* `createPortal(렌더링할_내용, 대상_DOM_노드)`.
```javascript
import { createPortal } from 'react-dom';

//기본 문법 구조
createPortal(
  <div>포털 내부 요소</div>,
  document.getElementById('portal-target')
);
```

## 2. 주요 활용 사례 및 해결 과제
* **레이아웃 제약 극복**
	* 부모 요소의 **z-index**, **overflow: hidden**, **position: relative** 속성에 의한 시각적 잘림 현상 해결.
* **전역 UI 컴포넌트**
	* **모달(Modal)** 및 **다이얼로그**: 화면 전체 중앙 배치가 필요한 UI.
	* **툴팁(Tooltip)** 및 **팝오버**: 대상 요소 근처에 위치하되 부모 스타일 영향을 배제해야 하는 경우.
	* **로딩 스피너**: 전체 화면을 덮는 오버레이 구현.
```javascript
import { createPortal } from 'react-dom';

//모달 전용 포털 컴포넌트 예시
const ModalPortal = ({ children }) => {
  const el = document.getElementById('modal-root');
  //실제 DOM 노드 'modal-root'로 children 전송
  return createPortal(children, el);
};

export default ModalPortal;
```

## 3. stopPropagation을 통한 이벤트 제어
* **이벤트 버블링 차단**
	* 하위 요소에서 발생한 이벤트가 상위 요소의 핸들러를 호출하지 않도록 전파 중단.
* **포털 내 필수 적용**
	* 포털 내부 콘텐츠 클릭 시, 리액트 트리상 상위에 위치한 **Overlay 닫기 함수** 등의 의도치 않은 실행 방지.
* **구현 방법**
	* 이벤트 핸들러 내부에서 `e.stopPropagation()` 호출.
```javascript
const ModalContent = ({ children, onClose }) => {
  return (
    <div className="modal-overlay" onClick={onClose}>
      {/* // 콘텐츠 영역 클릭 시 이벤트가 overlay로 퍼지지 않도록 차단 */}
      <div className="modal-inner" onClick={(e) => e.stopPropagation()}>
        {children}
      </div>
    </div>
  );
};
```

## 4. 실전 구현 패턴 및 주의점
* **대상 노드 검증**
* 컴포넌트 렌더링 시점에 대상 DOM 노드(예: `document.body`)의 존재 여부 확인 필수.
* **웹 접근성(A11y) 고려**
* 포털 이동 시 키보드 **포커스 흐름(Tab)** 및 **스크린 리더** 읽기 순서 관리 주의.
* **클린업 작업**
* 동적으로 DOM 노드를 생성하여 포털을 사용할 경우, 언마운트 시 해당 노드 삭제 로직 포함.
```javascript
import { createPortal } from 'react-dom';

const GlobalModal = ({ isOpen, onClose, children }) => {
  if (!isOpen) return null;

  return createPortal(
    <div className="modal-overlay" onClick={onClose}>
      <div className="modal-content" onClick={(e) => e.stopPropagation()}>
        {children}
        <button onClick={onClose}>닫기</button>
      </div>
    </div>,
    document.body // // body 태그 직하단 렌더링
  );
};
```
