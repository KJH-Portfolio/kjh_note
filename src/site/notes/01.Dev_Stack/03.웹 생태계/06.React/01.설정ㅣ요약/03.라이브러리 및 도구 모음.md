# React 개발 필수 라이브러리 및 도구 모음
> [!summary] 요약
> 
> React 생태계의 핵심인 라우팅, 통신, 상태 관리 및 UI 최적화를 위한 필수 패키지 구성 및 설치 자동화 정리.
## 핵심 라이브러리 (Core Packages)
- ! **react-router-dom**
    - 싱글 페이지 애플리케이션(SPA) 내 페이지 이동 및 경로 분기 관리.
- ! **axios**
    - HTTP 클라이언트 라이브러리로 서버와의 데이터 비동기 통신 수행.
- **tanstack/react-query** (추천)
    - 서버 데이터 캐싱, 동기화, 업데이트 및 로딩/에러 상태 관리의 표준.
- **zustand** (추천)
    - Redux 대비 가볍고 직관적인 훅(Hook) 기반 전역 상태 관리 도구.
```bash
// 한 줄 설치 명령어
npm install react-router-dom axios @tanstack/react-query zustand
```
## 스타일링 및 컴포넌트 라이브러리
- **styled-components / emotion**
    - CSS-in-JS 방식으로 컴포넌트 단위의 스타일 캡슐화 및 동적 스타일링.
- **tailwind css**
    - 유틸리티 우선(Utility-first) 프레임워크로 빠른 UI 빌드 지원.
- **react-icons**
    - Font Awesome, Material Design 등 수만 개의 아이콘을 컴포넌트 형태로 제공.
- **shadcn/ui** (추천)
    - Radix UI와 Tailwind 기반의 복사/붙여넣기 방식 고품질 컴포넌트 세트.
```bash
// 스타일 및 아이콘 설치
npm install styled-components react-icons
// tailwind 및 shadcn은 별도 초기화 과정 필요 (npx tailwindcss init)
```
## 실시간 메시징 라이브러리 (Real-time Messaging)
- ! **stompjs / @stomp/stompjs**
    - 웹소켓 위에서 동작하는 단순 텍스트 지향 메시징 프로토콜(STOMP) 클라이언트 구현체.
    - Pub/Sub 구조를 통한 채팅, 실시간 알림, 주식 차트 등 양방향 통신 구현 시 필수.
- ! **sockjs-client**
    - 웹소켓을 지원하지 않는 브라우저를 위한 폴백(Fallback) 기능을 제공하는 웹소켓 에뮬레이션 라이브러리.
```bash
// STOMP 및 웹소켓 관련 설치
npm install @stomp/stompjs sockjs-client
```
## 개발 생산성 및 유틸리티
- **react-hook-form**
    - 복잡한 폼(Form) 유효성 검사 및 데이터 수집 최적화.
- **framer-motion**
    - 선언적 문법을 활용한 고성능 컴포넌트 애니메이션 구현.
- **lucide-react**
    - 현대적이고 깔끔한 벡터 아이콘 라이브러리.
```bash
// 유틸리티 설치
npm install react-hook-form framer-motion lucide-react
```
