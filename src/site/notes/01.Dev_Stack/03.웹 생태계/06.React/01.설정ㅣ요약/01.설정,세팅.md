# 프로젝트 생성하기
## 프로젝트 만들기
- cd + 폴더명 으로 프로젝트 위치로 이동
	- `cd [프로젝트 경로]` 와 같은 방식
- 프로젝트 생성
`npm create vite@latest react-front -- --template react`
- 이후 y -> no -> yes
- cmd창에 Local : url에 실행되어있음(프로젝트 폴더 내부 main.jsx화면)
- ! cmd는 종료하지 말 것
#### 라우터 라이브러리 설치
- 해당 프로젝트 들어간 뒤에(my-app까지 들어가고)
`npm install react-router-dom`
## 프로젝트 실행시키기
- ctrl + 백틱 으로 터미널을 켜서 실행 가능
- index나 jsx등이 있는 프로젝트 내부까지 들어가야함
	- `cd C:\Users\user1\Desktop\WorkSpace\react\프로젝트 폴더명`
- 프로젝트 실행 구문
	- `npm run dev`
