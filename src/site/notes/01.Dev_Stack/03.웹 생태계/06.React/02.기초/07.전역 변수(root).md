# JSX 전역 변수 설정 및 :root 활용
> [!summary] 요약
> 
> CSS 변수를 활용한 애플리케이션 전체 스타일 제어 및 `:root` 의사 클래스를 통한 디자인 시스템 전역화.
## 전역 변수 설정 원리 및 방식
- **CSS 전역 변수 (:root)**
    - 문서의 최상위 루트 요소(HTML)에 변수를 정의하여 하위 모든 컴포넌트에서 접근 가능하도록 설계하는 방식.
    - `--` 접두사를 사용하여 변수명을 선언하고 `var()` 함수를 통해 호출.
- **JSX와의 연동**
    - 외부 CSS 파일(.css) 또는 CSS-in-JS 라이브러리 내 전역 스타일 정의 영역에 작성.
### :root 변수 정의 및 호출
- **변수 선언**
    - `:root` 선택자 내부에 색상, 간격, 폰트 크기 등을 정의.
- **변수 사용**
    - 개별 스타일 시트 혹은 JSX 인라인 스타일에서 해당 변수 참조.
```javascript
/* index.css 또는 App.css */
:root {
  --main-color: #61dafb;
  --bg-dark: #282c34;
  --spacing-unit: 8px;
}

// App.jsx
const style = {
  backgroundColor: 'var(--bg-dark)',
  color: 'var(--main-color)',
  padding: 'var(--spacing-unit)'
};
```
### JSX 동적 변수 제어
- **Style 속성 주입**
    - 자바스크립트 변수를 활용하여 `:root` 또는 특정 요소의 스타일 값을 동적으로 변경.
- **setProperty API**
    - `document.documentElement.style.setProperty`를 통한 런타임 중 전역 변수 값 수정.
```javascript
// 다크모드 전환 예시
const toggleTheme = (isDark) => {
  const root = document.documentElement;
  if (isDark) {
    root.style.setProperty('--main-color', '#ffffff');
    root.style.setProperty('--bg-dark', '#000000');
  } else {
    root.style.setProperty('--main-color', '#000000');
    root.style.setProperty('--bg-dark', '#ffffff');
  }
};
```
