# 리액트 JSX(JavaScript XML) 핵심 문법 정리

> [!summary] 요약
> 자바스크립트 내 마크업 작성을 위한 **확장 문법**인 JSX의 **표현식 주입**, HTML과의 **문법적 차이점**, **스타일링 및 주석 규정**, 그리고 리액트 엔진의 **렌더링 최적화 규칙** 정리

## 1. 기본 작동 원리 및 표현식

### JSX의 본질

* 자바스크립트 함수 내에서 HTML 태그를 직접 반환하는 구조
* 실행 시점: 일반 자바스크립트 객체로 변환되어 **Virtual DOM** 형성

### 중괄호 `{ }` 표현식 주입

* JSX 내부에서 자바스크립트 로직(변수, 함수 호출, 연산) 사용 시 필수
* 사용 예: `<p>{name}님 안녕하세요</p>`


---

## 2. HTML vs JSX 주요 차이점

| 구분         | HTML                | JSX (React)         | 충돌 방지 및 이유              |
| ---------- | ------------------- | ------------------- | ----------------------- |
| **클래스 지정** | `class`             | **`className`**     | JS의 `class` 예약어와 충돌 방지  |
| **라벨 연결**  | `for`               | **`htmlFor`**       | JS의 `for` 반복문과 충돌 방지    |
| **스타일링**   | `style="color:red"` | **`style={{...}}`** | 객체 형태 전달 및 CamelCase 적용 |
| **태그 닫기**  | 일부 생략 가능            | **반드시 폐쇄**          | XML 기반의 엄격한 파싱 규칙 준수    |

---

## 3. 스타일링 및 주석 규정

### 인라인 스타일 (Inline Style)

* **객체 형태**: 문자열이 아닌 **JS 객체**로 전달
* **CamelCase**: 속성명에 카멜 케이스 적용 (예: `fontSize`, `backgroundColor`)
* **이중 중괄호**: `style={{ ... }}` 구조 (외측: 표현식, 내측: 객체 리터럴)

### 주석 (Comments)

* **태그 사이**: `{/* 주석 내용 */}` 사용
* **여는 태그 내부**: `<div /* 주석 */ >` 처럼 일반 JS 주석 활용 가능

## 4. 엄격한 규칙 (최적화 및 데이터 무결성)
>기본 규칙은 [[3. Resource/01.Dev_Stack/03.웹 생태계/06.React/02.기초/01.JSX 기초이론\|01.JSX 기초이론]] 참고
### 리스트 렌더링과 `key` 속성

* `map()` 등을 이용한 반복 출력 시 각 요소에 **고유한 `key**` 부여 필수
* **목적**: 변경된 항목만 식별하여 **DOM 업데이트 성능 최적화**

> [!important] **Key 선택 기준 및 주의사항**
> * **고유 ID 사용**: 데이터베이스의 `id` 등 변하지 않는 고유값 할당 권장
> * **Index 사용 지양**: 배열 인덱스를 `key`로 사용 시 데이터 삽입/삭제 과정에서 성능 저하 및 컴포넌트 상태 오류 유발 가능
> * **안정성 확보**: 리렌더링 시마다 `key`값이 변하지 않도록 관리 (고정된 값 유지)
### 데이터 타입 제한 (불리언, Null)

* `true`, `false`, `null`, `undefined`는 렌더링 시 화면에 나타나지 않음
* **주의**: 숫자 `0`은 화면에 출력됨 (조건부 렌더링 시 유의 필요)

## 5. 실전 예시 코드
>jsx 기본 문법형태

```jsx
function UserProfile() {
  const user = {
    name: "Gemini",
    isVIP: true
  };

  const customStyle = {
    fontSize: "20px",
    color: user.isVIP ? "gold" : "black"
  };

  return (
    <>
      {/* 1. 클래스명 및 주석 */}
      <h1 className="title">사용자 프로필</h1>
      
      {/* 2. 객체 기반 인라인 스타일 및 표현식 */}
      <div style={customStyle}>
        이름: {user.name}
      </div>

      {/* 3. 태그 폐쇄 및 속성명 준수 */}
      <input type="text" placeholder="메모 입력" readOnly={false} />
      <br />
    </>
  );
}

export default UserProfile;
```
