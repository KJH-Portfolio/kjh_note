# 📘 HTML 목록 관련 태그

### 1. 목록 기본 요소📝

| **태그/문법** | **용도**    | **특징**                       |
| --------- | --------- | ---------------------------- |
| `<li>`    | **목록 항목** | 실제 목록의 각 항목을 정의 (`list`의 약자) |

### 2. 순서 없는 목록 (Unordered List)📝

| **태그/문법**               | **용도**       | **특징**                                   |
| ----------------------- | ------------ | ---------------------------------------- |
| `<ul>`                  | **순서 없는 목록** | 항목 앞에 점(•) 등의 기호가 붙는 목록`Un Ordered Line` |
| `<ul><li>...</li></ul>` | **중첩 목록**    | 목록 안에 또 다른 목록을 넣을 수 있음                   |


### 3. 순서 있는 목록 (Ordered List)📝

| **태그/문법**                 | **용도**       | **특징**                                           |
| ------------------------- | ------------ | ------------------------------------------------ |
| `<ol>`                    | **순서 있는 목록** | 항목 앞에 숫자/문자 등의 순서가 붙는 목록 (기본값: 숫자)`Ordered Line` |
| `type="a", "A", "i", "I"` | **순서 기호 변경** | 표기 기호를 소문자/대문자 알파벳, 로마 숫자 등으로 변경                 |
| `reversed`                | **역순 표기**    | 목록의 순서를 역순으로 표시 (예: 4부터 시작)                      |

### 4. 설명 목록 (Description List)📝

| **태그/문법** | **용도**       | **특징**                                               |
| --------- | ------------ | ---------------------------------------------------- |
| `<dl>`    | **설명 목록**    | 제목과 그에 대한 설명으로 구성된                                   |
| `<dt>`    | **정의/제목**    | 목록의 제목이나 용어 입력 (`definition term`)-`용어`라             |
| `<dd>`    | **설명/정의 내용** | 제목(`<dt>`)에 대한 구체적인 설명 작성 (`definition description`) |

#### 팁과 주의점
- `ul>li` : 로 간편하게 만들 수 있음
	- `ol>li * 10` 같은 문법도 가능
