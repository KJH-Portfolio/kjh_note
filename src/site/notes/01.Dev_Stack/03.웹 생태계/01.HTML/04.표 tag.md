##  # 📘 HTML 표(Table) 관련 태그 및 속성 정리

### 1. 표의 기본 구조 태그📝

| **태그/문법** | **용도**         | **특징/설명**                               |
| --------- | -------------- | --------------------------------------- |
| `<table>` | **표 생성**       | HTML 문서에 표(Table)를 생성하는 기본 태그           |
| `<tr>`    | **행 (Row)**    | 행(가로줄)을 나타냄 (`Table Row`)               |
| `<td>`    | **일반 데이터/열**   | 일반 셀 데이터(`Table Data` / Cell)를 나타냄      |
| `<th>`    | **데이터 제목/머리글** | 데이터 제목을 나타냄 (**굵은 글씨, 가운데 정렬**이 기본 스타일) |

### 2. 표의 구역 구분 태그 (시맨틱 구조)📝

| **태그/문법** | **용도**    | **특징/설명**                 |
| --------- | --------- | ------------------------- |
| `<thead>` | **제목 영역** | 제목(`<th>` 포함)을 나타내는 영역    |
| `<tbody>` | **내용 영역** | 본문 내용(`<td>` 포함)을 나타내는 영역 |
| `<tfoot>` | **바닥 영역** | 요약이나 합계 등을 나타내는 영역        |

### 3. 표의 크기 및 병합 속성 (Attributes)📝

| **태그/문법**      | **용도**    | **특징/설명**               |
| -------------- | --------- | ----------------------- |
| `border="1"`   | **경계선**   | 테두리(경계선) 두께를 지정합니다.     |
| `width="..."`  | **가로 길이** | 가로 길이를 지정합니다. (px 또는 %) |
| `height="..."` | **세로 길이** | 세로 길이를 지정합니다. (px 또는 %) |
| `colspan="N"`  | **열 병합**  | **N개의 열**을 하나로 합친다.     |
| `rowspan="N"`  | **행 병합**  | **N개의 행**을 하나로 합친다.     |

---

### 💡 구조화 시 참고 사항

- **`colspan`과 `rowspan`** 속성은 `<td>`와 `<th>` 태그에 사용될 수 있다.
