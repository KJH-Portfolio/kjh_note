# 📘 HTML 멀티미디어 태그

## 🖼️ `<img>` : 이미지 태그

| **속성**       | **설명 / 기능**               |
| ------------ | ------------------------- |
| **`src`**    | 표시할 이미지의 경로 (필수)          |
| **`alt`**    | 이미지가 표시되지 않을 때 나타날 대체 텍스트 |
| **`width`**  | 이미지의 가로 크기                |
| **`height`** | 이미지의 세로 크기                |

**크기 단위**

- `px`: 고정 크기 (화면 크기와 무관)
- `%`: 부모 요소 대비 비율 (화면 크기에 따라 변함)

---

## 🎵 `<audio>` : 오디오 태그

**공통 속성**

- `src`: 파일 경로

| **속성**     | **설명 / 기능**                   |
| ---------- | ----------------------------- |
| `controls` | 재생/정지/볼륨 등 컨트롤러 표시            |
| `autoplay` | 페이지 로드시 자동재생 (브라우저 정책에 따라 제한) |
| `loop`     | 반복재생                          |

**특징**

- `<audio>…</audio>` 사이 텍스트는 지원하지 않는 브라우저에서 대체 메시지로 표시됨

---

## 🎬 `<video>` : 비디오 태그

**공통 속성**

- `src`, `controls`, `autoplay`, `loop`, `width`, `height`

**전용 속성**

| **속성**   | **설명 / 기능**      |
| -------- | ---------------- |
| `poster` | 로딩 중 표시할 썸네일 이미지 |
