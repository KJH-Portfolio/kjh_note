# 📘HTML 하이퍼링크 태그 (`<a>`) 

## 1. `<a>` : 앵커(Anchor) 태그 핵심 속성📝

| **속성**       | **설명 / 기능**            |
| ------------ | ---------------------- |
| **`href`**   | **이동할 대상 URL/경로** (필수) |
| **`#id명`**   | **같은 페이지 내 특정 위치**로 이동 |
| **`title`**  | 마우스 오버 시 **툴팁** 표시     |
| **`target`** | 링크 열기 위치               |

---

## 2. 사용 패턴 및 예시📝

| **패턴**       | **설명**                 | **예시**                                 |
| ------------ | ---------------------- | -------------------------------------- |
| **외부/내부 링크** | 텍스트 클릭 시 페이지 이동        | `<a href="페이지 경로">텍스트</a>`             |
| **새 탭 열기**   | `target="_blank"` 사용   | `<a href="URL" target="_blank">링크</a>` |
| **이미지 링크**   | `<a>` 태그 안에 `<img>` 포함 | `<a href="URL"><img src="..."></a>`    |
| **페이지 내 이동** | `href="#id명"` 사용 (북마크) | `<a href="#top">맨 위로</a>`              |

---

## 3. 참고 및 유의 사항📝

- `<a>` 태그는 기본적으로 **인라인 요소**입니다.
    
- **페이지 내 이동**은 이동할 요소에 **`id` 속성**을 지정하여 `href="#지정된_id"`와 짝지어 사용합니다.
    
    > _Tip_: `<a href="#">맨 위로</a>`는 페이지 최상단으로 이동합니다.
    
- **보안 권장 사항**: `target="_blank"` 사용 시 **`rel="noopener"`**를 함께 추가하는 것을 권장합니다.
    
    - _예:_ `<a href="..." target="_blank" rel="noopener"></a>`
