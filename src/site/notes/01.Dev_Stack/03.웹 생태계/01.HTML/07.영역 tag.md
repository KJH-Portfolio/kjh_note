# 📘 HTML 영역 관련 태그 

> [!summary] Summary 📝
> - div : 블록 나누기(Row)
> - spn : 인라인 나누기(Col)
> - iframe : 인라인 프레임 태그
### 1. `<div>` : 블록 요소 태그📝

| **특성**           | **설명 / 기능**                                                                 |
| ------------------ | ----------------------------------------------------------------------------- |
| **요소 유형**       | **블록 요소 (Block-level Element)**                                             |
| **영역 차지 방식**   | **한 줄 전체**를 차지 → 줄바꿈 자동 적용                                            |
| **사용 목적**       | 레이아웃 구분, 콘텐츠 그룹화, 스타일 적용 영역 지정                                      |
| **시각적 특징**     | 사각형 박스 형태로 **전체 너비**를 감쌈 (기본 `width: 100%`)                          |
- `division`
---

### 2. `<span>` : 인라인 요소 태그📝

| **특성**       | **설명 / 기능**                              |
| ------------ | ---------------------------------------- |
| **요소 유형**    | **인라인 요소 (Inline Element)**              |
| **영역 차지 방식** | **콘텐츠 내용의 크기만큼만** 차지 → 줄바꿈 없음, 옆으로 연속 배치 |
| **사용 목적**    | 문장 내 **특정 부분**에 스타일 적용 (예: 색상, 배경, 강조)   |
| **시각적 특징**   | 텍스트 흐름을 따라 **필요한 만큼만** 영역 지정             |
- `span` 기간, 범위, 폭폭
---

**참고: 블록 vs 인라인 요소 비교**

| 구분           | 블록 요소 (`div`)                     | 인라인 요소 (`span`)                     |
| -------------- | ------------------------------------ | ---------------------------------------- |
| **줄바꿈**      | 자동 적용                              | 없음 (연속 배치)                           |
| **너비 지정**   | `width`, `height` **유효**             | `width`, `height` **기본 무시** (인라인 특성상) |
| **예시 태그**   | `div`, `p`, `h1~h6`, `ul`, `section` | `span`, `a`, `strong`, `em`, `img`       |

---

### 3. `<iframe>` : 인라인 프레임 태그📝

| **속성**            | **설명 / 기능**                                                  |
| ----------------- | ------------------------------------------------------------ |
| **`src`**         | 포함시킬 **외부 HTML 문서의 경로** (필수)                                 |
| **`width`**       | 프레임의 **가로 크기** (px 또는 %)                                     |
| **`height`**      | 프레임의 **세로 크기** (px 또는 %)                                     |
 ```html
<iframe src="01_글자 관련 태그.html" width="300" height="300" style="border:1px solid black;"></iframe>
```

---
### 4. textarea

| 태그/속성                     | 설명                         |
| ------------------------- | -------------------------- |
| `<textarea>`              | **여러 줄** 텍스트 입력 상자         |
| **`cols` , `rows`**       | 가로/세로 크기를 지정 (글자 수/줄 수 기준) |
| **`style="resize:none"`** | 사용자가 크기를 조절할 수 없도록 고정      |

---

**참고 사항**

- `<div>`는 **레이아웃의 기본 단위**로, CSS와 함께 사용 시 강력한 구조화 가능
- `<span>`은 **텍스트 단위 스타일링**에 최적화 → `width`/`height` 적용하려면 `display: inline-block;` 필요  
- `<iframe>`은 **다른 페이지를 현재 문서에 삽입** → 보안 정책상 동일 출처 정책(Same-Origin Policy) 주의 필요  
- HTML5에서는 `frameborder` 대신 CSS `border` 사용 권장
