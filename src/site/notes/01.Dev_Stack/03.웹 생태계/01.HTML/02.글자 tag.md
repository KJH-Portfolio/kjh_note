# 📘 HTML 글자 관련 태그

## 1. HTML 문서 기본 구조📝

| **태그/문법**         | **구분**      | **설명**             |
| ----------------- | ----------- | ------------------ |
| `<!DOCTYPE html>` | **문서 형식**   | HTML5 문서임을 알림      |
| `<html>`          | **HTML 시작** | 문서 시작 (언어 설정)      |
| `<head>`          | **헤더**      | 문서 설정 (제목, 스크립트 등) |
| `<title>`         | **문서 제목**   | 브라우저 탭에 표시되는 제목    |
| `<body>`          | **본문**      | 화면에 출력될 내용 영역      |


## 2. 기본 요소 및 특수문자📝

| **태그/문법**            | **구분**      | **설명**                                  |
| -------------------- | ----------- | --------------------------------------- |
| `<br>`               | **줄 바꿈**    | 강제 줄 바꿈 (단일 태그)`Break Row`              |
| `<hr>`               | **수평선**     | 수평선 표시 (단일 태그)`Horeizontal Line`        |
| `&nbsp;`             | **공백 문자**   | 줄 바꿈 되지 않는 공백 하나`Non-breaking space`    |
| `<h1>` ~ `<h6>`      | **헤딩/제목**   | 제목 크기 (h1이 가장 큼)                        |
| `<a href="...">`     | **하이퍼링크**   | 다른 페이지로 연결 `Anchor Hypertext Reference` |
| `<abbr title="...">` | **툴팁 (약어)** | 마우스 오버 시 설명 표시                          |

## 3. 문단 및 텍스트 서식📝

| **태그/문법**          | **용도**         | **특징/의미**                         |
| ------------------ | -------------- | --------------------------------- |
| `<p>`              | **문단**         | 줄 바꿈 태그 필요, 공백 하나만 인식             |
| `<pre>`            | **문단 (서식 유지)** | 작성된 형태(공백/줄 바꿈) 그대로 표시            |
| `<b>` / `<strong>` | **굵게**         | **`<strong>`은 중요/강조 의미** (스크린 리더) |
| `<i>` / `<em>`     | **기울임**        | **`<em>`은 강조 의미** (스크린 리더)        |
| `<mark>`           | **형광펜**        | 텍스트 하이라이트                         |
| `<u>`              | **밑줄**         | 텍스트에 밑줄 표시                        |
| `<s>`              | **취소선**        | 더 이상 유효하지 않은 내용                   |
| `<sub>`            | **아래첨자**       | 기본 글자 아래 작은 글자                    |
| `<sup>`            | **윗 첨자**       | 기본 글자 위 작은 글자                     |
| `<small>`          | **작은 글자**      | 텍스트를 작게 만듦                        |
| `<big>`            | **큰 글자**       | 텍스트를 크게 만듦<br>(CSS사용을 권장)         |
#### 어원
```
Paragraph
Preformatted text
Emphasis 강조
Strike through
Subscript
Superscript
```

## 예시📌
```HTML
<!DOCTYPE html>
<html lang="ko">
<head>
    <title>HTML 태그 활용 예시</title>
</head>
<body>
    <h1>HTML 기본 태그 사용 예시</h1>
    <hr>
    
    <p>
        이것은 `<p>` 태그로 작성된 일반 **문단**입니다.<br>
        줄 바꿈 태그를 사용해야만 줄이 바뀌며, 공백은 하나만 인식됩니다. &nbsp; &nbsp; &nbsp;</p>

    <h2>텍스트 서식과 강조</h2>
    
    <p>
        <b>단순 굵은 글씨</b>와 
        <strong>강조(스크린 리더 인식)</strong>를 비교해보세요. 
        <br>
        <i>단순 기울임</i>과 
        <em>강조(스크린 리더 인식)</em>도 함께 확인합니다.
    </p>

    <p>
        <u>밑줄</u>을 긋거나 
        <mark>형광펜 효과</mark>를 줄 수 있으며, 
        <small>작은 글자</small>와 
        <big>큰 글자(사용 비권장)</big>도 있습니다.
        <br>
        <s>이 내용은 더 이상 유효하지 않습니다. (취소선)</s>
        <br>
        아래첨자: H<sub>2</sub>O
        윗 첨자: E=mc<sup>2</sup>
    </p>

    <h4>하이퍼링크 및 툴팁</h4>
    <p>
        <addr title="World Wide Web">WWW</addr>에서
        <a href="https://www.google.com">Google로 이동</a>할 수 있습니다.
    </p>

    <pre>
        이 영역은 <pre> 태그입니다. 
        작성된    그대로의 
        줄바꿈과     공백이 
        화면에 표시됩니다.
    </pre>

    <h6>예시 끝</h6>

</body>
</html>
```
