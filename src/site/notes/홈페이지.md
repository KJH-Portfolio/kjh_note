## 🚀 Core Technology Stack
*제가 깊이 있게 다루고 탐구하는 핵심 기술 생태계입니다.*

> [!info] **Backend Ecosystem**
> - **Language**: Java (Modern Syntax, Object-Oriented Design)
> - **Framework**: Spring Framework, **Spring Boot**, Spring Data JPA
> - **Security**: Spring Security, **JWT (Json Web Token)** Authentication
> - **Messaging**: WebSocket, **STOMP** Protocol
> - **Persistence**: JDBC, **Mybatis**, SQL & DB Integration

> [!success] **Frontend Ecosystem**
> - **Library/FW**: **React.js** (Hooks, Router, API Integration)
> - **Language**: JavaScript (ES6+), HTML5, CSS3 (Modern Responsive Layout)
> - **State & Logic**: Context API, Axios, Fetch API, jQuery
> - **Build tool**: **Vite**, Modern Web Standards

> [!warning] **Data & Infra**
> - **Database**: **Oracle SQL**, PL/SQL (Procedure, Function, Trigger)
> - **Modeling**: Database Modeling & ERD Design, Normalization
> - **Version Control**: **Git**, **GitHub** (Collaborative Strategy, Convention)
> - **Automation**: Gemini API & AI Integration
