## 🚀 Core Technology Stack
*정원에 기록되고 실무에 적용되는 핵심 기술 생태계입니다.*

- **Backend**: Java, Spring Boot, Spring Security (JWT), Spring Data JPA, Mybatis, WebSocket (STOMP)
- **Frontend**: React.js, JavaScript (ES6+), HTML5/CSS3, Vite, 비동기 통신(Axios/Fetch)
- **Data & Infra**: Oracle SQL, PL/SQL (프로시저/함수/트리거), DB 설계 및 정규화, **Docker** 컨테이너 환경 구축
- **Tools**: Git, GitHub (협업 전략 및 관리 컨벤션), Gemini API 연동 및 자동화
